　　这一群人，接连几次都是攻打失败，被逼退了出去。

<br/>　　“看来这剑道门的人，倒是有些能耐，那个什么门主，楚风眠，据说还是这九域之中极为有名的一位天才，这剑道门也是此人一手缔造的。”
<br/>　　道寒也是坐下恢复了一些灵力，开口说道。
<br/>　　“哼，什么楚风眠？天才？这九域的天才，在我们九华天，就连给我们提鞋都不配，据说这楚风眠不过只是一个小小的九劫古帝罢了，半步天人境都没有踏入，居然都是这九域之中极为有名的天才，看来九域真的没落了。”

<br/>　　不过对他这样一位半步天人境的武者来说，九劫古帝，的确都是如同蝼蚁的存在，他挥手之间就可以碾死。
<br/>　　他们这些人可以联合在一起，也是因为互相都是半步天人境的武者，如果是一尊九劫古帝，就连站在他们身边的资格都没有。
<br/>　　“从我们九华天，随便找一个天才弟子来，都可以碾压这九域之中所谓的天才，现在九域之中，除了七大宗门之中一些老古董之外，其他的人，都不足为据。”
<br/>　　“如果不是忌惮天九域，就七大宗门那些老古董，也不是我们九华天之中强者的对手，我们九华天早就可以占据这九域了。”

<br/>　　“据说我家族之中的一些老古董，已经是去试探了天九域了。”
<br/>　　“等九华天占据了九域，对我们反倒是会变得束手束脚，我们要敢在这之前，多多掠夺九域的资源才行。”
<br/>　　“先攻打了这一方圣地再说，日后哪怕是九华天的大军到来，我们也有立足之地！”
<br/>　　一阵交谈之后，这一群人便是再度开始攻打剑源界了。

<br/>　　“八品圣地，莫非是这圣地也跟大陆一样，分为九品？九华天，境魔天，离恨天！这是三大圣域的名字，这五个人，居然是来自于圣域九华天的天才？”
<br/>　　楚风眠从一旁，听着这些人的交谈，一下子明白了过来，这些人的来历。
<br/>　　这些人居然是来自九华天，如今无尽星河之中，唯有的三方圣域之一的九华天。
<br/>　　怪不得这些人一个个都是如此妖孽，居然是出身自圣域。
<br/>　　九华天，可是圣域，一级大陆，像是整个大罗天星域，才只有四个三级大陆罢了。
<br/>　　一个九华天的实力，只怕是比起一百个大罗天星域加起来还要强，怪不得可以有着如此之多的妖孽天才。
<br/>　　“九华天，境魔天，离恨天这三大势力，居然都盯上了九域，幸好是有着天九域，让他们忌惮，不然这三方圣域的人，前来九域，剑道门这点实力，根本是无法跟他们抗衡，只能够是灰溜溜的带人逃走了。”
<br/>　　楚风眠心中都是不由的想到。
<br/>　　这从九华天之中出来的五名武者，居然都是半步天人境的强者，这的确是太可怕了。
<br/>　　虽然楚风眠也知道，这五名武者，纵然是在圣域九华天之中，都是绝对的天才妖孽。
<br/>　　可九华天的实力，也可以由此看出，起码远不是现在已经没落的九域，可以抗衡的了。
<br/>　　“这五个人，必须全部斩杀！不然的话，未来麻烦无穷，决不能放走一个。”
<br/>　　这五个人，已经是将主意达到了剑源界上面来了，楚风眠也没有置身事外的可能了。
<br/>　　既然是要杀，楚风眠就要将这五个人全杀了。
<br/>　　这五个人的身份在九华天之中都绝对不低，如果被九华天追查到，可就麻烦了。
<br/>　　楚风眠身形闪烁，通过云羽天宫，一下子回到了剑源界之中去。
<br/>　　只要楚风眠在九域之中，便是可以随时通过云羽天宫，回到剑源界之中去。

<br/>　　“门主回来了。”
