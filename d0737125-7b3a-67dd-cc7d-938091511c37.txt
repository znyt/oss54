叶翎又不是什么都不懂的小姑娘，成年男女，她大晚上去他房间跳fire，还真能跳出火来。
<br/>
<br/>        “我不去，我都说了我不去，顾夜瑾，你是我哥！”
<br/>
<br/>        “我怎么不记得我妈还给我生了一个妹妹，我算你什么哥哥，叶翎，你现在成年了，那层窗户纸也该捅破了，我不要做你哥，我把你养这么大是想做你男人的！”
<br/>
<br/>        他和她之间那层薄如蝉翼的窗户纸现在真的被捅破了，以后他再也不用顾忌什么了。
<br/>
<br/>        叶翎抬手，直接将顾夜瑾的俊脸上扇去。
<br/>
<br/>        但是顾夜瑾扣住了她纤细的皓腕一把按在了墙壁上，薄唇又吻上了她的红唇，嗓音低哑狠戾的警告道，“你不来我的房间也行，我去你的房间，不过不是看你跳fire的，我要搂着你跳脱衣舞。”
<br/>
<br/>        叶翎张嘴，狠狠的咬下了他的唇角。
<br/>
<br/>        鲜血在两个人的口腔里肆意蔓延。
<br/>
<br/>        “顾夜瑾，你是不是爱上我了？”叶翎问。
<br/>
<br/>        顾夜瑾一僵，“什么？”
<br/>
<br/>        “难道不是么，你这么想跟我上床难道不是爱上我了么，但是你不敢爱我，所以拿夏妍妍当借口，对么？”叶翎望着他。
<br/>
<br/>        顾夜瑾缓缓将她松开了，他唇角勾出了一道讥讽而残忍的弧线，大手拍了拍她娇艳的小脸，“想多了，你妈不是喜欢爬我爸的床么，那我就让她的女儿一辈子待我床上下不来，翎翎，你说你妈会不会…死不瞑目？”
<br/>
<br/>        叶翎感觉浑身一凉，像是黑暗里伸来一只大手直接将她推入了万丈深渊里。
<br/>
<br/>        “不许这样说我妈咪！顾夜瑾，我恨你！”
<br/>
<br/>        叶翎拽着拳，眼眶通红的瞪着他，像是被困的小兽，恨不得扑上去跟他同归于尽。
<br/>
<br/>        顾夜瑾看着她通红的眼眶里慢慢的凝结上了一层水珠，蒲扇般的羽捷掀动了两下，那眼里的雾珠却强忍着不落下，真是可怜极了。
<br/>
<br/>        每次他狠下心欺负她的时候，她都没有还手之力，楚楚可怜的。
<br/>
<br/>        顾夜瑾垂在身侧的两只大手蜷起，缓缓收紧，这时一串悠扬的手机铃声响起，他来电话了。
<br/>
<br/>        拿出手机一看，是夏妍妍打来的。
<br/>
<br/>        顾夜瑾看了叶翎一眼，“晚上我等你。”
<br/>
<br/>        说完他拉开了洗手间门走了出去，回廊里可以听到他变得温润的嗓音，“喂，妍妍…好，我现在过去。”
<br/>
<br/>        ……
<br/>
<br/>        女洗手间里，叶翎站在盥洗台前，用冷水拍了一下小脸。
<br/>
<br/>        她看着放在手边的那张房卡，双目有些放空。
<br/>
<br/>        这时门被推开，霍璇走了进来。
<br/>
<br/>        霍璇看见叶翎真是仇人相见分外眼红，她阴阳怪气的开口道，“叶翎，刚才你跟夜瑾哥哥在一起的吧，但是妍妍姐一个电话就把夜瑾哥哥叫走了，生了一张狐媚子的脸又有什么用，夜瑾哥哥又不喜欢你。”
<br/>
<br/>        叶翎关了水龙头，站直了身，她看了霍璇一眼，然后用目光指了一下盥洗台，“看这是什么？”
<br/>
<br/>        霍璇抬头一看，双眼都亮了，是顾夜瑾的房卡！
<br/>
<br/>        “叶翎，夜瑾哥哥的房卡怎么会在你这里？”
<br/>
<br/>        叶翎抽出一张纸巾擦了擦手，神色慵懒淡漠，“哦，这是你的夜瑾哥哥让我转交给你的，今晚你可以进入他的房间了，他在等你。”
