        &lt;h3&gt;第三卷第二百六十四章  郝景文到西城&lt;/h3&gt;
<br/>
<br/>        郝景文下了飞机，直奔西城分部，这几年西城在大发展，而他们的事业也是蒸蒸日上，从哲真是个人才，将西城管理得井井有条，让他省心。
<br/>
<br/>        他每次来都是从哲亲自接机，这一次也不例外。
<br/>
<br/>        “郝总，这一次的会议每个分部的人都来了，哦，不对，就是泉城的周宏生没有来，他好像身体不太好，安排了他们公司的副总傅海生来了。”
<br/>
<br/>        “傅海生，我怎么从来都没有听说过他？”郝景文对全国各地的中层大致都能知道一下，傅海生确实是很陌生。
<br/>
<br/>        “这个傅海生其实也是一个人才，原来是某单位的一把手，只是好像有些事情没有遂了他的意，直接就辞职不干，回家带孩子，然后和周宏生又是同学，俩个人关系很好，所以就到了周宏生的公司上班，周宏生直接给了一个副总的位置。”从哲解释着。
<br/>
<br/>        “哈哈哈，性情中人啊，能不要工作直接回家带孩子，这一份魄力一般人怕是做不到，我倒是很好奇这个傅海生是什么样的人才，你见过吗？”
<br/>
<br/>        “我没有见过，只是看到了名单，调查了一下。”
<br/>
<br/>        “你还调查了些什么？”
<br/>
<br/>        “他老婆倒是很厉害的人物，现在是西城响当当的红人，可以说一步登天，后期前途无量。”
<br/>
<br/>        “什么样的人物？”
<br/>
<br/>        “信息界重量级人物，和陈子昂和江总都有关联。”
<br/>
<br/>        “你这人还真有意思，竟然调查了人家的家人。”
<br/>
<br/>        “西城满共就这么大的东西，风一吹草一动，大家都知道，这个世界上哪里有那么多的秘密可言。”
<br/>
<br/>        “按照你的说法，傅海生的辞职和他老婆有关系？”
<br/>
<br/>        “郝总不愧是郝总！你是怎么想到的？”
<br/>
<br/>        “他的老婆如此强势，如果说，俩个人关系好，傅海生的老婆飞黄腾达的时候，他也会跟着飞黄腾达的，而不是辞职，肯定是他的老婆太强势了，让他无法反抗。”
<br/>
<br/>        “西城背地里流传着一句顺口溜：瞎眼恋老头，放弃小鲜肉，为了向上爬，猪狗都接受。”从哲摇了摇头。
<br/>
<br/>        “嘶，你这样说，我倒是好奇傅海生是什么样的人了。”郝景文八卦心起。
<br/>
<br/>        “反正明天你就会见到，我也没有见过，不知道是不是小鲜肉。哈哈哈。”从哲大笑。
<br/>
<br/>        “你这太恶趣味了，看你笑得那猥琐的样子，我鸡皮疙瘩都出来了。”郝景文配合着他的话打了一个寒颤。
<br/>
<br/>        “确实，我不应该给我们年轻单纯，帅气可爱，温文尔雅的郝总说这样的话，抱歉，我收回，我也不应该在背后议论我们的同事。”从哲立马转变了态度，稳重的道歉。
<br/>
<br/>        “好了，看你那认真的样子，我也就是说说，还当真了。对了，你有没有通知江总明天开会？”郝景文给江俞轩打了电话说要开会，但是没有说时间。
<br/>
<br/>        “还没有呢，我等会将你送到酒店，我再给他打电话。我也好久没有见到江总了，虽然都在西城，大家都是忙忙碌碌的。”从哲解释着，他给江俞轩发邮件了，江俞轩就给他回复了两个字：“知晓”。也不知道江俞轩来不来参加会议。
<br/>
<br/>        “那你就不用打电话了，我等一会打个电话，刚好我和他很久没有见面了。你只管将会议组织好就是，另外这次会议的内容要清晰明了。”郝景文吩咐。
<br/>
<br/>        “行，我知道了，虽然是同一个公司，但是业务线条是分开的，我会抽调一部分人新安排工作。”从哲点点头。
<br/>
<br/>        从哲是个人才，虽然学历不高，但是好在头脑灵活，接触新事物比较快，而且对用人方面也自有他自己的一套认知，整个西省的中层领导被他管理得服服帖帖。
<br/>
<br/>        郝景文到酒店洗漱完毕，给江俞轩打电话。
<br/>
<br/>        江俞轩正在和寒伯安、安男、张函、陈子昂说话。
<br/>
<br/>        张倩楠对江俞轩说：“俞轩，你的电话响了。”
<br/>
<br/>        “嗯，是文文的，估计文文到西城了。”江俞轩看了一下电话说。
<br/>
<br/>        “文文，你到西城了？”
<br/>
<br/>        “我到了，明天开会，记得来啊。”郝家文一手拿着手机，一手打开电脑。
<br/>
<br/>        “嗯，大哥也来了，我看大哥有没有时间和我们一起参加。”江俞轩看了一眼寒伯安，似乎他们几个人又可以聚聚了。
<br/>
<br/>        “啊，大哥什么时候回来的，怎么到西城了，赶快将电话给大哥，我可想他了。”郝景文在电话那头说。
<br/>
<br/>        “文文，你好没良心，有了大哥就不要我了。”江俞轩笑着将电话递给寒伯安。
<br/>
<br/>        寒伯安能与陈家人坐在一起完全是因为安男，按照他高冷的性子和身份地位，基本是不会参加别人家的聚会的，但是他太想念安男了，俩个人在一起一直都是十指紧扣。
<br/>
<br/>        “文文为什么不直接给我打电话，要打到你的手机上，不接。”寒伯安拒绝接电话。
<br/>
<br/>        安男将电话拿过来：“文文。”
<br/>
<br/>        “嫂子，嫂子我好想你啊，不是，是我好想大哥！”文文一改平时那种温文尔雅的样子在电话里大喊。
<br/>
<br/>        “他说他好想你。”安男将电话递给寒伯安。
<br/>
<br/>        “说，什么事情？我不想听到你的声音。”寒伯安傲娇的说。
<br/>
<br/>        “哇，大哥，都好久没有见到你了，你这一走就好多天，回来了都不给我说。”郝景文委屈的声音传过来。
<br/>
<br/>        “这是我的错了？”寒伯安笑了。
<br/>
<br/>        “不，是我的错，大哥，给我一个弥补错误的机会，明天能不能参加公司会议，我向你汇报工作。”郝景文问寒伯安。
<br/>
<br/>        寒伯安看了一眼身边的安男：“不，明天下午我们几个人可以聚一聚。”
<br/>
<br/>        “好，那明天下午我们聚一聚，明天见，大哥。”寒伯安来西城肯定不是因为工作而来，一定是因为安男而来，只是安男为什么会来西城？
<br/>
<br/>        郝景文不在纠结安男为什么来西城，他又将注意力集中到了工作上，明天的西城会发现一种全新的行业，会带动一种全新的生态链，这种改变将由他郝景文，江俞轩和寒伯安来续写，每到一个地方，每开通一个据点，郝景文心里都会产生自豪感和成就感。
<br/>
<br/>        从哲这几年已经是西城的著名人士了，在这个行业里他是西省的龙头老大，明天的会议是内部工作会议，更是西城的发布会，他已经联系了一些领导，明天会参加他们的发布会，还有新闻媒体。
<br/>
<br/>        明天，将会是新的一天！
