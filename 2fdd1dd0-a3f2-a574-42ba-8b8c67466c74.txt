　　洪荒一如既往地发展着，人族碎暗流涌动，但明面上有着神农的存在，一时之间，还掀不起多大风浪，而此时深入混沌的混沌道君，却是突然间遇到了危险。
<br/>　　混沌苍茫无限，无边无际，洪荒宇宙只能算作是无垠的混沌海中的一座小岛。混沌广阔，难以形容。混沌之中，更是危机重重。
<br/>　　不说其他，就是一个洪荒附近这一片混沌，便孕育出了三千魔神，更何况混沌中的其他地方了。
<br/>　　混沌道君在道魔大战之后，便独自一人进入了混沌。他修炼的是混沌本源法则，只有游历混沌，才能更快地进步。
<br/>　　这无数元会之中，混沌道君走过大片混沌，遇过无数的危机。在混沌之中，最怕的就是迷失了方向。一旦迷失了方向，就再也不可能回到原来的地方。
<br/>　　不过，混沌道君因为有着元始这一本尊在洪荒，所以，尽管他也不知道自己身处何处，但冥冥之中他们存在一种感应，他知道大致的方向。
<br/>　　混沌之中，苍茫是主题。这里灰蒙蒙一片，没有天，没有地，没有光明与黑暗，在这里，一切都是昏暗的，一切都是混乱的。很长时间不能遇到一个生命。若是没有坚定的道心，早晚会崩溃。
<br/>　　这些年，混沌道君遇上过无数次爆裂的混沌风暴。天地涌动，混沌之气暴动，狂涌而来，覆灭一切。就算混沌道君身为混元强者，也有好几次遇到生死之危。但，最后终究是躲过了。但就算躲过了，他也受到了巨大的重创，事后更需要花费许多时间去修复伤势。
<br/>　　而且混沌之中，除了混沌风暴，还有混沌深渊，混沌裂缝、混沌爆炸等危机。这些混沌道君或多或少都有经历过。可以说，每一次他都是从生死危机之中挺过来。
<br/>　　更甚者还有混沌之中生存的莫名凶兽。他们在混沌之中不断破坏，创造毁灭，他们就是混沌的毁灭者。他们智慧极低，只知道破坏，但又实力极强。大多都有准圣的实力，更甚者有混元大罗金仙的实力。
<br/>　　混沌道君就遇上过不止一次被混沌凶兽攻击，但最终依靠自己的实力与智慧，终究是挺过来了。
<br/>　　混沌之中有危机，自然也伴随着无数机缘。从根本上来说，游历混沌，本来就是为了这些机缘而来，如若不然，谁还会冒着巨大惊险而进入混沌。
<br/>　　先不说混沌之中那些比洪荒中高级许多的神料器材，神铁、道金，神玉、灵宝等，可以用来炼制无数强大的灵宝，就说那混沌之中的无数道果、灵植、灵果等，也是洪荒无法堪比的。毕竟，能够生存在混沌之中的，又岂能平凡？
<br/>　　当然，还有其他好处，如在混沌之中孕育的世界。能够找到一个混沌之中的世界，并将之占为己有，那本身就是一种无双的气运与机缘。
<br/>　　不同的世界孕育不同的法则，能够有机会参悟其他世界的法则，并将之融会贯通，那就是一种实力道行提升的机会。更别说这世界之中的无数资源了。一个世界的资源，又岂能少得了。
<br/>　　可惜，这种机缘十分难得，以混沌道君混沌树的身具的气运，经过这无数元会，也仅仅只是遇到过一个，而且还是一个快要破灭的中千世界。
<br/>　　生灵有寿命，世界同样有寿命。一方世界经历无数岁月，终将要走向毁灭，天地同归混沌。等待无数年后，再次孕育出来，那样，就是另一个纪元的开始了。
<br/>　　混沌道君在发现这个世界后，这个世界的道则就有些破损了，要不了几个元会，便会毁灭。混沌道君自是不能眼睁睁开着它走向毁灭，既然遇到了，那便是他的机缘。他以无双神通，将此世界炼化，收入自己体内，以自身的混沌本源大道不断滋养，欲让它重复昔日光彩。
<br/>　　同时，他也发现了这个世界的状况，此世界并没有其他种族，只有一个种族，那便是木族。在此世界之中，只有一种生灵，他们生来就具有不同的木系神通法则。他们全是木系生物所化。
<br/>　　这就是一个木系大道法则所演化的世界。同时，它又不是一个简单的木系大道世界，因为只要是世界，它都是由三千大道构成，不同的只不过是它们构成的比例、融合的方式，各种大道演化的多少等。
<br/>　　此木灵世界就比较单一了。构成此世界的三千大道中，占据九成的是木之大道，其他两千九百九十九中大道，只占据了一成。
<br/>　　真可谓是混沌之大，无奇不有。这种世界，对于修炼木之一道的修士，自然是天堂，但对于其他修士，又是一种限制。太过单一，本身就是一种巨大的限制。他们的成就不可能有多大。
<br/>　　情况就是如此，在此世界之中，最强者为一堪比准圣的修士，名为木祖。堪比大罗金仙的有六人，其他的，大多是一些大罗以下者。这还是一个宇宙一个纪元积累出来的强者。近乎耗费了一个世界的机缘遇气运，才孕育出这点强者。
<br/>　　这，就是世界的限制。
<br/>　　而且那唯一的准圣强者，还是此方世界的开辟者，此方混沌的大道之子。他是一种名为通天建木的木灵。
<br/>　　故而，此方世界之中，建木一族虽然人少，但却是第一大族。它又分为建木主族、世界树一族、生命树一族，他们之中，就有两个堪比大罗金仙的修士。
<br/>　　另外四个拥有堪比大罗强者坐镇的势力是：扶桑树所化的太阳一族，月桂树所化的月亮一族，不死天藤所化的不死族以及只有一人的攀天藤。
<br/>　　当然，这些月桂与扶桑，对比洪荒的众多先天灵根，自然是有所不足的。毕竟，此世界只是一个中千世界。
<br/>　　当然，大族依然还有一些，比如虚空藤一族、长生木族、雷霆木族、绿柳一族等，他们的资质并不比六大种族差。只不过，这个世界并没有能够让他们突破的机缘了，所以他们才会被限制在大罗金仙以下境界，不得突破，若是能够给他们足够的机缘，他们一样可以突破。
<br/>　　混沌道君发现此世界后，自然是用强大的实力将世界中的众修士慑服。若有不服者，就打散其灵智，使之重归本体。这样虽然会让他们的资质有所下降，但混沌道君来此是为了这世界中的机缘的，并不是为这些强者服务的，他们重归本体，这样对他更好。
<br/>　　就这样，在灭了世界树一族、扶桑一族、攀天藤一族强者的灵智之后，混沌道君收服了这方世界。
<br/>　　而距离收取了那木灵世界后，混沌道君再次经历过不少危机，他的实力也自然是大涨，由刚开始入混沌时的混元初期，如今都进入混元九重天巅峰境界了，只差一步，便可步入混元圆满。由此可以说明，混沌道君混沌树的资质，比起本体来说，还要强许多。
<br/>　　而此次混沌道君遇到的危机，就是之前他经历过多次的混沌风暴。但此次的风暴，比起之前的可要厉害无数倍，就连混沌道君都没有信心度过它。
<br/>　　漫天的混沌之气化为巨大的风暴侵略而来，风暴如同一个虚幻的滔天巨兽，不断吞噬者混沌，由小变大，渐渐有一种毁天灭地的气势，冲击而来。
<br/>　　混沌道君估计，此刻就算是一方世界挡在它面前，它都会被瞬间吞噬。
<br/>　　轰隆隆！
