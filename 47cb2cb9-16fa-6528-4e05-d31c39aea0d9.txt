　　大朝会上所发生的一切似乎并没有影响什么，所有人依旧是每天该干什么继续干什么，除了魏征发丧的那一天李承乾带着满朝文武去送行之外，再也没有什么大事发生。
<br/>　　老头子的病还是那样，据孙思邈那老头儿说，应该是心气郁结，忧思过度。
<br/>　　这事儿老孙搞不定，李承乾同样也搞不定，只能看老头子啥时候能想开，等想开了之后也就好了。
<br/>　　不过这次老头子倒是真的放权了，所有朝政全都不闻不问，一概交给李承乾来处理，最后处理成什么样子，老李同样也不知道知。
<br/>　　对于这样的老头子，李承乾还是心存愧疚的，毕竟如果让老头子继续干下去，再有六年估计也就退休了，他完全没有必要着急这六年时间。
<br/>　　所以每次批完了奏折之类的东西，没有事情的时候李承乾都要跑到老头子那里看看，陪自家老爹说说话儿，东家长西家短的扯上一会儿。
<br/>　　这一日，处理完了手头儿的事情，李承乾再一次来到老头子的卧房，着人通报了之后，被老头子叫了进去。
<br/>　　“父皇，今日身体可好些了？”看着正在提笔写字的老头子，李承乾笑着上前问道。
<br/>　　“还是老样子，看来朕是真的老了！”老李感慨的叹了口气，手下笔走龙蛇，很快一幅字便写了出来：难得糊涂。
<br/>　　李承乾看着纸上漂亮的飞白体字迹，对着老头子挑起大拇指：“父皇的字越来越漂亮了！”
<br/>　　“少拍马屁！”李二将手中的笔放到一边，白了李承乾一眼：“不过你小子这话说的不错，难得糊涂，果真是难得糊涂。”
<br/>　　李承乾尴尬了笑笑：“父皇，您想的多了，还是保养身体重要，莫要为一些杂事烦心。”
<br/>　　眼下老头子是怎么看自己怎么不顺眼，说话还总是带刺，这都是魏征一番神操作带来的副作用。
<br/>　　但事情已经到了这个地步，后悔什么的明显已经没用了，再说这件事情发生之前李承乾根本就不知道，所以就算是想后悔也没有什么可以后悔的，只能感慨世事无常罢了。
<br/>　　而就在李承乾纠结于老头子态度的时候，李二却开口了：“朕的身体朕心里清楚，你的事情你自己也清楚，过段时间朕会下召书禅位，你回去准备吧！”
<br/>　　前面都是废话，但最后一句回去准备吧，却让李承乾的心跳陡然加快一半拍。
<br/>　　虽然他口口声声说自己不想那么早当皇帝，可是这种事情真的轮到自己头上又有谁不想呢？毕竟那可是大唐皇帝，不是什么后世的那种小班长，小科长什么的。
<br/>　　而且就算是后世单位里的班长又或者科长也不是说当就能当的，如果论到谁的头上，那也是让人高兴万分的事情。
<br/>　　可是现在问题来了，老头子的这句吩咐要怎么回答呢？
<br/>　　说好？是不是有些不要脸了？说不好？太假了吧？
<br/>　　怎么办？要不要说点什么？可是说什么呢？
<br/>　　李承乾只觉得平日里运转十分迅速的脑子似乎卡住了，想了半天也没有想到应该说些什么来表达自己的想法。
<br/>　　而就在此时，老头子又开口了：“你也不用觉得为难，这魏征其实说的并没有错，朕现在的确是搞不懂这个国家了。而且……现在把这个位置交到你的手里也好，至少朕还能动，有些事情还能帮你把把关，到至于让你行差踏错。”
<br/>　　李承乾不知道老头子这话的真实程度到底如何，可是看着老李失意的样子，心里在高兴的同时也有些不是滋味。
<br/>　　本来嘛，你说干了一辈子领导，现在要退体了，放在谁的身上谁也不舒服，那种消积几乎不可用语言来形容。
<br/>　　但李承乾也知道，此时自己是不能在老头子这里再停留下去了，越在这里待着估计老头子的心情就会越差，一个不好揍他一顿也不是没有可能。
<br/>　　想到这里，李承乾默不作声的给老头子施了一礼，慢慢退出了书房。
<br/>　　……
<br/>　　出了含凉殿，回到东宫没多久。
<br/>　　李承乾便收到了消息，老头子亲自下诏，十日之后群臣上朝，他要亲自宣布禅位！
<br/>　　这下没得说了，老头子既然已经公布了，那这件事情便已经成了定局，再也不用继续纠结下去了，一切只等十日后之。
<br/>　　不过随着老头子这道诏收，整个大明宫可就彻底不消停了，无数的宫娥彩女忙个不休，礼部同样忙的脚不沾地。
<br/>　　十天时间太急张了，各种登基的礼仪要安排，新的衣服要准备，而且不光是李承乾的衣服，他那几个大上老婆的吉服同样要准备。
<br/>　　毕竟登基之后他就是皇上了，原来的太子妃什么的也要改成皇妃，品级水涨船高没有新衣服怎么成。
<br/>　　另外一点就是老头子也能折腾，竟然要李承乾给他准备十艘战舰，说是世界那么大，他要去看看，只等李承乾一登基就走。
<br/>　　你说这不是添乱么，好好的太上皇不当，没事儿瞎跑什么呀！这要是在外面被人给绑架了可咋整！
<br/>　　李承乾试着劝了一下，结果发现老头子似乎认准了这一条，怎么也说不通，还说如果没有战舰，他就要坐以前的那个楼船。
<br/>　　无奈，李承乾只能把整个‘海狼’调回来，作为老头子的护卫，顺便再准备一批新式武器给老头子安装到船上。
<br/>　　等到一切都准备的差不多了，时间也到了第九天的夜里，眼看着再有一晚的时间，整个大唐就要换一位新的主人，整个大明宫灯火通明，无数人彻夜难眠。
<br/>　　李承乾自然也是睡不着的人之一，如今的他已经二十六岁，孩子也有了四个，按说应该稳重一些。
<br/>　　可不知为什么，这个夜晚他就是睡不着，于是从炕上坐起来看着窗外的夜空发呆，回忆着这段时间所发生的一切。
<br/>　　从老魏征谏言开始，一直到老头子对他说要禅位给他，这段时间所发生的一切都是那么出人意料，甚至突然到让他有些难以接受，就好像是在做梦，显的那么不真实。
