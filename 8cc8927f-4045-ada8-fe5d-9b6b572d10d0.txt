　　何为气运？
<br/>　　气运是每一个人一生中不可缺少的东西。虽说气运虚无缥缈，看不清，摸不着，但不可否认的是，它的确是真实存在的。一个人从出生的那一刻，便会有或多或少的气运加身。气运深厚者，逢凶化吉，遇难成祥，出门随地捡灵根，坠崖有奇遇，修为突破如喝水，越级作战只是寻常。气运稀薄者，便会身缠重病，麻烦不断，何时死的都不知道。
<br/>　　修士修炼，若是气运深厚，修为便会不断上涨，别人修炼数万年达到的境界，你只需修炼数百年便可，就好似整个天地都围着你转，天地主角便是这样的人。
<br/>　　就好似西游之中的孙悟空，别人修炼到金仙，需要数万年时间，而他却只需要数百年即可。这就是因为他是量劫主角，天地气运加身的缘故。
<br/>　　可见，气运对于修士是必不可少的东西。
<br/>　　故而，也有人说，修炼之路，便是要不断地夺运。夺天地气运，夺他人气运，夺众生之运。聚万千气运于一身，便可不断突破，扶摇直上。
<br/>　　随着修为越高，气运的作用也就越大。就仿佛元始，若是他没有盘古开天留下的遗泽，福运，功德，他也不可能如今就走到这个地步。
<br/>　　又如道祖鸿钧，他若不是天选者，他也不会得天道气运所钟，第一个成圣。
<br/>　　一个人的气运也不是一尘不变的，它会随着时间的流逝而不断变化，也会随着那人所做之事的好坏，或增加，或减少。故而，有一件能够镇压自身气运的灵宝，那是多么不容易啊。可以说，就算是数千万个修士之中，也很难有一个。
<br/>　　就连女娲这个未来的圣人若不是道祖之后赐下灵宝，她恐怕都没有镇压气运的灵宝。还有原著中的通天，截教号称万仙来朝，但没有镇压气运之物，待到了量劫一至，依旧化为飞灰。
<br/>　　可见，有一件能够镇压气运的灵宝，是多么美妙的一件事啊。但元始就有，而且他还有不止一件。十二品白莲不用多说，那是元始在东海三清岛上得到的。先天至宝混沌钟更不用多说，那是元始与东皇太一交换来的。鸿蒙元始印更不用多说。以前元始还不知此印有镇压气运的功能，直至他修炼突破，不断祭炼，才发现它除了之前的那些功能外，他还有遮蔽天机，镇压气运的功能。
<br/>　　这让元始不得不感慨，这鸿蒙元始印，究竟还有多少功能，没有被他发觉？仅仅之前的那些功能，就相当于鸿钧手中的造化玉蝶了，而且它还有一个本源世界，这比什么混沌珠好多了，如今再加上镇压气运、遮蔽天机等功能，又相当于混沌钟等灵宝，这元始印，真不愧是鸿蒙至宝，多重功能集一身，一件可顶多件的灵宝。
<br/>　　元始看着鸿钧，他此刻有些猜测到道祖讲道的目的了。道祖布道洪荒，真正目的恐怕就是为了收集洪荒众生的气运，虽然一人之气运可能很少，但集万千气运为一身，聚少成多，那时可就是一笔无比深厚的气运了。
<br/>　　至于为何要收元始等人为弟子，那就更简单了，因为他们不仅是天道注定的圣人，还是洪荒之中气运最为深厚的修士。可惜，他没有料到元始有至宝镇压气运，他从元始这里并不能得到多少气运。
<br/>　　有了如此多的气运，他不仅修为能够蹭蹭地飞速上涨，而且还能够使他身合天道只是占据更多主动。到时候，究竟谁是天道，就不好说了。
<br/>　　嘶！
<br/>　　一想到此处，元始就深深地倒吸一口气，若是他的想法是真的，那么，道祖的行为恐怕就不是身合天道那般仁爱了，他的目标恐怕一直都是天道，从算计盘古那一刻，就没变过。
<br/>　　只是，元始心中亦冷笑道，徒弟拜师，老师可以从徒弟身上收取气运，徒弟又何尝不是在借助老师身上的气运来修炼呢？
<br/>　　这些想法只是在他心中一闪即逝，他赶紧收敛心神，此处是紫霄宫。紫霄宫是什么地方？这里可是天道气息最为浓重的地方。他不敢再往下多想，因为鸿钧就在他眼前坐着呢。
<br/>　　元始抬起头，看向高台之上的道祖鸿钧，只见道祖此刻眼中更加淡漠，更加无情了。待众人都从悟道之中醒来，他才淡淡开口道：
<br/>　　“贫道早年游历洪荒，收集的一些灵宝，此时留之却是无用，今日便赐予尔等吧。”
<br/>　　鸿钧的话音刚落，在诸多修士心中却掀起了滔天巨浪，我等没有听错吧？道祖说什么了？把他收集的灵宝都分给我们？竟会有这等好事？道祖所留的灵宝，又岂会是平凡之物？若是能够得到一两件，那岂不是发了？
<br/>　　就在众修士心思各异之时，只听那道祖鸿钧的声音淡淡传来：
<br/>　　“元始为盘古元神所化，身居三清之首，众人师兄一职，吾此处有混沌开天斧所化之先天至宝盘古幡一件，理应为你执掌！”
<br/>　　“多谢老师赐宝！”元始脸色不变，悠悠拜谢道。他是真的不在乎，混沌灵宝他都有，先天至宝他更是有，而且还不止一件，现在多了一件，也仅仅只是锦上添花而已。但他没有拒绝，拒绝是傻子，没有人会放着大好的灵宝而不要。
<br/>　　众人只见，光芒一闪时，鸿钧道祖单手一挥，道韵流转，混沌剑气切割空间，一枚锋芒毕露的混沌色小旗出现在元始身前，正是先天至宝盘古幡！
<br/>　　元始无比平淡地接过此番，面色不改，也不激动，也不再次拜谢。他这番模样，让众人看了只觉得心里痒痒的，恨不得上前揍他。这可是先天至宝啊，不是什么平凡之物，要不要如此淡定啊？你好歹也做个样子啊？众人心中嫉妒非常。
<br/>　　“老子为盘古元神所化，三清之一，贫道有盘古斧所化之先天至宝太极图一份，当为老子执掌！”鸿钧开口道。
<br/>　　光芒一闪，一副蕴含着阴阳造化的太极图，出现在老子身前。老子的表现就不如元始那般淡定了，他一接过太极图，感受到其中与自己异常契合的阴阳大道，便激动地向鸿钧拜谢道：
<br/>　　“叩谢老师，愿老师圣寿无疆！”
<br/>　　“嗯。”鸿钧道祖淡淡点了点头，看向满怀期待的通天，淡淡道：
<br/>　　“通天为盘古元神所化，三清之一，吾此处有诛仙四剑及阵图一份，合起来为先天至宝，配合诛仙阵图，设立诛仙剑阵，威力巨大，非四圣不可破，为天地杀伐第一。通天，汝当慎用之，需谨记诛仙剑阵不可轻易布下。”
<br/>　　“弟子通天多谢老师。”一听到“非四圣不可破，天地杀伐第一”这几个字，通天便激动不已，哈哈大笑，得意忘形。
<br/>　　元始脸色淡然，看不出丝毫波动，他知道通天这是真性情，不作伪。老子看着通天手中的四剑以及阵图，眼中则是一丝异色闪过。其他诸多修士更是无比嫉妒地看着三清，特别是通天，恨不得以身待之。
