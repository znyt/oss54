　　叶尘也没有想到，这些人竟然就这么跑了，其实以他如今的状态，自保或许绰绰有余，倘若这些人群而攻之，他还真未必能讨得好。
<br/>　　“小子！你若是敢杀我，我叔父肯定不会放过你的！”
<br/>　　段惊天望着缓缓逼近的叶尘，顿时吓得连连倒退，可惜他身受重伤，一口真元提不上了，想要逃跑都做不到。
<br/>　　叶尘冷冷一笑，
<br/>　　“你放心！他若是敢惹到我头上，我很快就会让你们叔侄在地下相见！”
<br/>　　说完这话，叶尘身体一晃，便瞬间欺身到段惊天跟前，然后朝他腹部虚手一抓，
<br/>　　“吞噬！”
<br/>　　呼呼呼！
<br/>　　段惊天的真元，顿时便开始飞速流逝！
<br/>　　段惊天顿时大惊失色，
<br/>　　“你、你对我做了什么？”
<br/>　　他想要拼命的阻止体内的真元流逝，可是根本无济于事。
<br/>　　“你是魔族！你居然是魔族之人！”
<br/>　　段惊天大吼大叫了起来，叶尘理都不理，继续操控着吞天神功，吞噬着他的真元。
<br/>　　片刻之后，
<br/>　　段惊天体内的真元，已经被叶尘吸收的干干净净。
<br/>　　叶尘吞噬了段惊天的内力之后，直接随手一掌震碎了他的丹田，然后直接转身离去，没有一丝耽搁。
<br/>　　大约十来分钟后，一道黑影自天空降落，落在了段惊天的身旁。
<br/>　　这是一名身穿黑袍的白发老者，气势雄厚无比，不在当初的云千秋之下。
<br/>　　此时段惊天还吊着一口气，并没有死掉，看到这黑袍老者，猛然伸出手臂，
<br/>　　“叔父！你一定要……要帮我……报仇啊！”
<br/>　　原来这黑袍老者，便是段惊天的叔父，天榜排名第三的段九德！
<br/>　　段九德见状，当即冲到段惊天的跟前，
<br/>　　“惊天，你先别说话！叔父为你疗伤！”
<br/>　　段九德说着，便要将自己的真元，输送到段惊天的体内，却发现段惊天的丹田被人毁掉，已经无力回天了。
<br/>　　“这小子也忒狠！竟然毁了你的道基！”
<br/>　　对于修真之人来说，道基被毁比丢了性命还要可怕，简直生不如死！
<br/>　　段惊天一把抓住段九德的手腕，两眼之中透着浓浓的恨意，强行提起一口气，怒吼道：
<br/>　　“叔父！我已经没救了，你快去追上那小子为我报仇！他肯定是回了太初城，准备离开蓬莱，你一定不能让他逃了！一定不能……”
<br/>　　段惊天大吼了一声，也将自己最后一丝生机耗尽，话还没有说完，便头一歪，彻底断了气息。
<br/>　　“惊天！！”
<br/>　　段九德也顿时大吼了一声，一双老眼通红无比。
<br/>　　“惊天，你放心，叔父一定会为你报仇的，一定！”
<br/>　　段九德将侄儿的尸体放回地上，身上猛然升起一股惊人的杀意，继而脚下猛然一顿，便直接朝太初城的方向飞驰而去！
<br/>　　……
<br/>　　太初城，
<br/>　　正如段惊天所料，叶尘吸收了他的内力之后，便第一时间回到了太初城。
<br/>　　其实，
<br/>　　从叶尘将十三名元婴强者引走，到斩杀其中七人，然后又返回太初城，也不过才短短半个多小时而已。
<br/>　　此时传送阵高台之上，依旧空无一人。
<br/>　　叶尘登上高台，便准备返回世俗界，
<br/>　　然而就在这个时候，后方忽然传来一道咆哮之声。
<br/>　　“贼子休走！我段九德来也！”
<br/>　　叶尘回头瞥了一眼来人，不由得冷冷一笑，低声喃喃道：
<br/>　　“等我下次踏入蓬莱之时，你们所有人都会被我踩在脚下！”
<br/>　　说完这话，叶尘直接转身，冲入传送阵之中，而且在他整个人将要消失的一瞬间，猛然一拳砸出！
<br/>　　轰隆隆！
<br/>　　整个传送阵，直接被叶尘一拳打爆，连同整座近百米的高台，也化成了齑粉。
<br/>　　传送阵被叶尘毁掉，不但成功拦住了段九德，没有一两个月的时间，根本不可能修复成功。
<br/>　　而他则可以利用这段时间，修复体内的伤势，并且彻底稳固自己的修为。
<br/>　　段九德追至跟前，看到眼前的这一幕，顿时气得发狂，
<br/>　　“叶狂仙！你给老夫等着！等我踏入世俗界，一定要屠你满门！！”
<br/>　　犹如雷鸣般的声音，在整个太初城上空，悠悠扬扬的传递开来，连同整座城池，都跟着一阵剧烈晃动。
<br/>　　……
<br/>　　叶尘自然已经听不到段九德的声音，在他出手毁掉传送阵的同时，他便已经跨过了结界，从蓬莱仙界返回到了世俗界。
<br/>　　然而，
<br/>　　等叶尘看清楚周围的状况，顿时发现情况似乎有些不太对劲。
<br/>　　周围海水依旧，却似乎并非是他当初，进入蓬莱仙界的那个入口！
<br/>　　“难道是因为我毁坏了传送阵，导致了空间发生扭曲，所以没有传送到最初的那个位置？”
<br/>　　很快，
<br/>　　叶尘在四周方圆数百里逛了一圈，更加证实了自己的判断，
<br/>　　这里的确不是最初的入口！
<br/>　　而且苏蔓三人也不在附近，很显然是传送位置出现了偏差。
<br/>　　搞清楚了情况之后，叶尘也并没有太在意，毕竟只要回到了世俗界，无论是在地球上的哪个角落，他都能找到返回华夏的路。
<br/>　　而眼下的当务之急，是要先确定自己在地球上什么位置。
<br/>　　叶尘隐约记得，有一种叫做六分仪的光学仪器，可以通过光学原理，确定海船的经纬度，可惜他并不知道制作之法。
<br/>　　“看来只有先找到船只，然后问一问船上的人……”
<br/>　　“不过在此之前，我要先将我的修为稳定一下！”
<br/>　　之前他一口气吞噬了七大强者的真元，一直都还没有来得及炼化，丹田竟有些涨疼。
<br/>　　于是，叶尘也顾不得其他，索性盘腿坐在海面之上，开始炼化起来。
<br/>　　滋滋滋~
<br/>　　丹田内吞噬而来的真元，在吞天神功的炼化之下，开始迅速转化成叶尘的真元。
<br/>　　不过，这些真元在叶尘体内缓缓汇聚在一处之后，开始自动形成一个气旋，然后飞速的旋转起来。
<br/>　　随着真元越来越多，那道气旋也旋转的也越来越快，最终竟然凝聚成了珠子状，散发着熠熠光辉，在丹田之中，犹如黑夜中的夜明珠！
<br/>　　轰！
<br/>　　随着这颗金珠成形，叶尘通体上下，猛然爆发出一道璀璨的金芒，直冲云霄！
<br/>　　知道此时此刻，叶尘才算是真正迈入金丹境。
<br/>　　不过，还没等叶尘高兴起来，
<br/>　　原本万里无云的天空，竟然开始雷声四起，轰轰作响，犹如世界末日了一般！
