        傅昼景被童筱颖的这一巴掌打出了神，待他回过神来时，电梯门正在慢慢关上，童筱颖最后一丝容颜在他视野里瞬间消失。
<br/>
<br/>        他什么时候变得如此坏了？
<br/>
<br/>        刚刚上来前，他先行坐的电梯，看到大哥在他之后坐了另一台电梯。
<br/>
<br/>        只是他没想到，会在抵达楼层，电梯门开了后，遇见正在吃糖果的童筱颖。
<br/>
<br/>        那一刻，或许是对筱颍的情难自控，又或许是想报复大哥，他身体不听大脑使唤般，主动走过去强吻了筱颍。
<br/>
<br/>        可结果……
<br/>
<br/>        他吻到的只是一张糖果纸，不但没令大哥生气，反而被筱颍打了一巴掌。
<br/>
<br/>        明明他也想尝尝报复大哥时的爽快滋味，却适得其反。
<br/>
<br/>        大哥似乎完全没有受到他们的任何影响，而他却把自己和筱颍的距离越推越远。
<br/>
<br/>        童筱颖离开酒店后，在路边的休息石凳上坐了下来。
<br/>
<br/>        她眸子里一片黯然，但满脑子里，却都是傅夜沉刚刚那淡漠的身影。
<br/>
<br/>        他仍旧是那般高高在上的帝王，好像她只不过是他生命里不足挂齿的过客而已。
<br/>
<br/>        傅昼景也只是上了个礼数便急匆匆地跑了出来，他还以为童筱颖已经走远，结果发现她就坐在酒店旁这条街边的石凳上时，心里蓦然欣喜地走了过去。
<br/>
<br/>        身边忽然有人坐下，童筱颖惊愣地抬眸偏头，看到是傅昼景后，脸色瞬间又不好看了。
<br/>
<br/>        “我早就说过，大哥不会相信你。”傅昼景心底有一丝得意，不痛不痒地说，顿了顿后，又接着说道，“自从三年前，我窥觊大哥的傅氏集团，私底下背着大哥做了那些背叛傅氏集团的事情后，大哥就不再相信我了。包括我这次回来，大哥明里当我是兄弟，暗里却一直在防着我。所以，他一定也会对你有所防备。”
<br/>
<br/>        他感觉自己很了解大哥，但又其实也并不是那么很了解。
<br/>
<br/>        傅昼景心里也很迷茫。
<br/>
<br/>        童筱颖只知道傅昼景在跟她说话，但是她听不太清楚，正好她也无心去听他说那些话，更不想跟他说话。
<br/>
<br/>        “筱颍，我说过，我可以祝福你和其他男人在一起，唯独大哥不行！”傅昼景忽然疾言厉色地说道。
<br/>
<br/>        童筱颖见他似乎是没完没了，于是从口袋里掏出助听器，重新戴上后，看着傅昼景的眼睛，面无表情地说道：“你刚刚所说的话，我一句也听不清。其次，从现在开始，我真的不想再见到你，也不想再听到你的声音。昼景，太过自私的人，只为自己着想的人，会失去很多珍贵的东西。”
<br/>
<br/>        “你的耳朵怎么呢？”傅昼景惊愕地凝视着童筱颖，抬起手来握住她的肩头，心疼地问道。
<br/>
<br/>        童筱颖挪开他的手，冷笑：“拜你所赐啊！因为你，让我间接地成为了聋子。这回，你高兴了吧？噢！对了！如你所愿，我和大哥离婚了。是大哥提出要跟我离婚的。你现在是不是觉得很开心呢？你是不是觉得，令大哥痛苦，你才会开心？不！大哥一点都不痛苦，他很潇洒啊！你以为他会跟你一样吗？为了一个女人去斤斤计较！我想你猜错了，大哥他对我很无所谓。昼景，面对这样的结果。你满意吗？你开心吗？你应该很满意也很开心吧！虽然没报复到大哥，至少报复到我身上了是不是？”
<br/>
<br/>        “筱颍，我没想过要伤害你……”傅昼景语重心长道，“你还有我，我们还可以……”
<br/>
<br/>        童筱颖嗤之以鼻地打断了傅昼景欲要说的话：“负我的人是你，伤害我的人还是你。你已经有自己的孩子了，也有一个愿意为你付出生命的女人。你为什么就不能放过我？昼景，我不欠你什么。我倒是觉得，你似乎是希望我死了，你才会开心！”
<br/>
<br/>        “筱颍，你是不是爱上大哥了？”傅昼景黯然神伤地问道。
<br/>
<br/>        童筱颖身心一怔，并未否认，但也未承认。
<br/>
<br/>        她从石凳上起身，再次取掉了耳朵上的两个助听器，只身离开。
<br/>
<br/>        这世界安静了真好……
<br/>
<br/>        “筱颍……”傅昼景大声呼喊。
<br/>
<br/>        只可惜，她再也听不到他的声音了。
<br/>
<br/>        童筱颖独自回到自己的小公寓，卸下自己的坚强之后，眼泪如开了闸的洪水决堤而下，将自己伪装了这么多天，原来也有装不下去的时候。
<br/>
<br/>        倒在床上，这段日子与傅夜沉在一起时的画面像播放电影一样，一幕幕闪过眼前，彷佛还是昨天发生的事……
<br/>
<br/>        大哥的心好狠……
<br/>
<br/>        真的好狠，不是吗？
<br/>
<br/>        他怎么可以那么洒脱？
<br/>
<br/>        留下她孤单的一个人在这儿暗自流泪……
