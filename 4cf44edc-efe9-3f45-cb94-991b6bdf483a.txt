　　刹那间，Tim的后背就被冷汗打湿。
<br/>　　布克尔特目光投射过来的时候，Tim感觉到那根本不是人的目光，而是来自南美洲丛林中野兽的凝视！
<br/>　　Tim杀过人，而且杀过很多人，老人、男人、小孩、女人他都杀过，而且杀人的手段也极其残忍，在当地Tim绝对算得上是一个狠角色，可就是Tim这样一个能令当地政府警察都感觉到害怕的狠角色，在布克尔特的注视下感觉到了颤栗，感觉到了恐惧。
<br/>　　Tim避开布克尔特的注视，说道：“对不起Boss，我不该过问的。”
<br/>　　“我知道你还有其他人心里都有疑问，为什么我会这么不遗余力的去帮助西伦科特，这个之前跟我们从来没有任何交集的人。”布克尔特拿起桌子上的银色沙漠之鹰，说道：“你们也许认为西伦科特背后有‘暗月’的关系，但并不是这样，西伦科特这次会被盯上，是因为‘暗月’把他出卖了。我帮助西伦科特……是因为他是我哥哥，同父异母的哥哥！”
<br/>　　布克尔特的回答彻底震惊了Tim，只有Tim自己知道，他是有多努力才保持住了他现在的冷静。
<br/>　　站在房间中的布克尔特仿佛陷入了一段很长很长的回忆，过了差不多有十分钟，也许更长，布克尔特才缓缓道出他跟西伦科特之间这段外人所不知的关系。
<br/>　　“我的妈妈是个中英混血儿，她出生在香港，上个世纪八十年代以偷渡的方式来到美国，原本她去美国是相信那里遍地都是黄金，是为了赚钱的，可是到了那里才发现一切都是谎言。她们一起偷渡的人中，有三分之一在偷渡过程中因为染病死了，被‘蛇头’扔进了大海，我妈妈她们九死一生来到美国，直接就被‘蛇头’送去了当地的妓院。我妈妈在那里待了半年时间，看着身边同伴一个接一个的或是染病、或是遭到虐待而死去，她决定要逃跑。”
<br/>　　“准备了三个月时间，在一个暴雨的夜晚，她从妓院里逃了出来，在打手的追捕下她躲进一家中餐馆集中泔水的塑料桶里。这家中餐馆的老板也是个香港人，在知晓我妈妈的经历后，好心的把她收留了，我妈妈就成了这家中餐馆里的一个洗碗工。虽然工资不高，但是有吃又有住的地方，而且不需要靠出卖自己的肉体来讨好男人。”
<br/>　　布克尔特的语气很平淡，就像是在说一件跟自己毫不相干的事情，“在那家中餐馆，我的妈妈认识了我的父亲，也就是西伦科特的父亲。他是个地地道道的美国人，军人家庭，他有空就会到那家中餐馆吃饭，一次在那里喝酒，他喝醉了，在厨房里把我妈妈强奸了，而那个开中餐馆的老板迫于威胁，就这么在旁边看着，看着老西伦科特把我妈妈强奸。”
<br/>　　“老西伦科特估计自己都没有想到，他因为一时间没管住自己的裤裆，有了我。当然我的出生是根本不会受到他们家认同的，而且我妈妈在被强奸后，就被那个餐馆老板赶了出来，她连去打胎的钱都没有。她辗转几个州，最后在旧金山生下了我，从她怀孕到我出生，老西伦科特都没有出现过。”
<br/>　　“我在一个黑人贫民窟出生，虽然我是个白人，但是却跟黑人为伍，打架、酗酒、抽大麻，在我十五岁的时候，我尝到了女人的滋味，那是一个白人女人，我和几个贫民窟一起长大的同伴强奸了她，然后把她丢在大街上。很快我们就被警察找到，并且抓了起来。我妈妈为了我去找到了老西伦科特，并且苦苦的哀求他，老西伦科特是当时的国会议员，很有权利，我妈妈威胁他，如果他不愿意帮忙就把他当年强奸餐馆洗碗工的事情曝光出来，老西伦科特没办法，答应了我妈妈的请求。”
<br/>　　“被我们强奸的女人是通用公司高管的女儿，影响很恶劣，即便是以老西伦科特的能量也没办法把我从监狱里弄出来，最后在老西伦科特的帮助下，法官把我送进了军队，那里是我唯一能去的地方。而我妈妈跟老西伦科特之间的谈判，被从军校回家的西伦科特听见了，他也知道了我的存在，那个时候的西伦科特虽然年纪还不大，但是却很有心计，他没有去质问老西伦科特，只是暗暗记下了这件事情。”
<br/>　　“我们有相同的父亲，但是却有着截然不同的命运。西伦科特是军校毕业的高材生，而我是一个因为犯了强奸罪而被送进军队服役的强奸犯，在军队，瘦小的我遭到很多人殴打，但是没有人能帮我，我只能靠自己。”
<br/>　　“也许是那里激活了我体内的暴力因子，逐渐的，我喜欢上了那里，喜欢上了那个靠拳头说话，用实力称王的地方。我从一名普通的陆军士兵，变成了‘绿色贝雷帽’的一员，我从一个士兵成为了上尉，我参加了阿富国战争，我拿着M4在街道中穿行，寻找并消灭恐怖分子，我觉得军营才是我应该待的地方。本来我以为，我会在军队里待一辈子，即便我没有西伦科特这样的背景，我依靠自己也能成为将军，可是我错了。从阿富国回来后，我就被强制离开了军队，我成了一名雇佣兵，我为‘黑水’公司服务，我们帮美国政府干脏活湿活黑活。这个时候，西伦科特找到了我，他给了我一笔钱，让我来南美洲发展，他给我送来一批退役军人，让他们跟着我一起干，在西伦科特的帮助下，我建立了我的cocaine王国，有了现在的地位和实力。”
<br/>　　布克尔特看向Tim，问道：“现在你应该明白我为什么会帮助西伦科特了吗？虽然他帮助我，是为了给他自己准备一条后路，但是没有他的帮助，我可能已经死了。在我离开‘黑水’公司后不久，我的小队在非洲执行任务的时候遭到了埋伏，整个小队全都完蛋了，无一幸存，所以如果不是西伦科特让我离开‘黑水’，我也跟着我的小队一起完蛋了，我欠他一条命！”
