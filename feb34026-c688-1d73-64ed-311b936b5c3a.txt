　　不过相比而言，楚风眠受到的影响却是要小不少，一是楚风眠的肉身无比逆天，这压力承受在楚风眠的肉身之上，不需要特意耗费灵力去抵挡。
<br/>　　二是楚风眠拥有始祖天龙的血脉，催动遁光的手段，也要更加玄妙，所以速度仅仅是慢了一点。
<br/>　　而在后面，那碧螺老祖跟五行刀祖，可就没有这样的手段了。
<br/>　　他们二人，也在一路向着独桥深处飞行，只是遁光的速度，已经是削弱了足足九成了。
<br/>　　一来一回，这二人已经是被楚风眠彻底的甩在了身后了。
<br/>　　“这已经是飞行三千公里吧。”
<br/>　　楚风眠张望着四周，还是熟悉的熟悉黑暗，不由的自言自语了一句。
<br/>　　他已经是足足飞行了三千公里，如果在外，三千公里，对古帝来说，却是不值一提。
<br/>　　毕竟在这独桥之上，别说是飞行，没走一步都要耗费不少的灵力。
<br/>　　纵然是楚风眠，如果没有神脉，也坚持不住。
<br/>　　楚风眠身上的一身灵力，比起九劫古帝还要雄厚，可以说没有神脉的话，九劫古帝一口气都飞不到这么远的距离。
<br/>　　不过楚风眠也清楚，这独桥，其实就是一道考验。
<br/>　　这些大型宝藏，许多都是曾经一些强者，故意留下的，以将传承留下去。
<br/>　　就像是九玄仙尊洞府之中一样，想要得到九玄仙尊的宝藏，必须要通过通天塔二十一层才行。
<br/>　　这独桥也是如此，只有通过独桥，达到尽头，才算是通过考验。

<br/>　　这一座宫殿。依然不是尽头，而是跟之前一样，屹立在独桥上方的一座宫殿。
<br/>　　这一座宫殿的大门，依然封闭，宫殿大门之上的禁制，已经是腐朽不堪。
<br/>　　“破！”
<br/>　　楚风眠看了一眼，手心之中凝聚灵力，一掌批了过去。
<br/>　　轰隆一声。
<br/>　　宫殿大门上的禁制，被全部打破，整座大门倒下，露出了其中的模样。
<br/>　　空旷的空点中心，又是一枚紫色的石头，在这紫色石头之上，还露出了金色的斑点，如同一枚枚星辰。
<br/>　　“紫星石！”
<br/>　　楚风眠看了一眼这紫色石头，眼神之中闪过几分喜色。
<br/>　　紫星石，是一种炼器材料，在无数的炼器材料之中，都算是极品了。
<br/>　　甚至据说在炼制极道仙兵的时候，有些都要加入这紫星石。
<br/>　　如果找来一名炼器宗师的话，以这紫星石，在辅以其他的几种炼器材料的话，可能是足以炼制出极道仙兵来。
<br/>　　不过在如今的时代之中，能够炼制出极道仙兵的炼器宗师，根本就不存在了。
<br/>　　想要炼制极道仙兵最为基本的一点，便是天人境之上的强者，才足以做到。
<br/>　　没有成就天人，如何了解天人之力？
<br/>　　在如今的时代之中，就连天人境的强者，都寻找不到了，更何况是天人境的炼器宗师。
<br/>　　这紫星石，给楚风眠来炼，最多也就炼制成祖龙至尊剑那个级别的灵剑。
<br/>　　楚风眠现在自己有着祖龙至尊剑，巨阙神剑，两把灵剑，自然不在欠缺灵剑。
<br/>　　不过紫星石，终究是一件难得一见的至宝，看来这大型宝藏曾经的主人，也是一位身家雄厚的绝代强者。
<br/>　　这两座宫殿，应该都是这大型宝藏的主人，故意留下的，应该是通过这独桥，一部分的考核，也就可以得到这宫殿之中的宝物。
<br/>　　楚风眠心神一动，手心一爪，这紫星石便是落入到楚风眠的手心，被楚风眠放入到了空戒之中。
<br/>　　这紫星石，楚风眠留着，有朝一日如果真的寻找到可以炼制极道仙兵的炼器宗师的话，足以是用这紫星石，锻造出一把新的极道仙兵来。
<br/>　　楚风眠收取了紫星石，身形一闪便是不在停留，向着独桥深处再度飞去。
<br/>　　这样的宫殿，有第一个，有第二个，那必然也会有着第三个。
<br/>　　这宫殿之中的每一件宝物，几乎都是足以让九劫古帝，甚至是半步天人境的强者，争破头的存在。
<br/>　　在古籍之中的记载之中，纵然是踏入天人境的强者，也不是每个人都拥有极道仙兵的。
<br/>　　能够拥有极道仙兵的强者，可都是天人境之中都极为强横的存在。
