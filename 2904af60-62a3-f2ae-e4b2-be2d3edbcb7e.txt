夏夕绾当着所有媒体记者的面，当着整座海城说，我想给陆先生生两个小宝宝，不是一个哦，而是两个。
<br/>
<br/>        这话一落下，陆茵茵一僵，而陆子羡唇角的弧线慢慢的收了回去，他抬眸，看向前方那个角落里。
<br/>
<br/>        所有媒体记者都转过了身，当大家开出一条道目光刷刷的落过去时，才看到了一身笔挺西装的陆寒霆。
<br/>
<br/>        陆寒霆身高腿长的伫立在那里，他将夏夕绾送来后就没有离开，而是站在这里看着被聚光灯环绕的她，今天她是璀璨万丈的公主，而他只是她的观众。
<br/>
<br/>        当听到她要给他生两个宝宝时，他身上那层阴鹜森冷的戾气瞬间消失不见了。
<br/>
<br/>        那几个想逃离现场的记者也察觉到了这一点，刚才阴冷蛰伏像只野兽般随意都有可能扑上去的男人竟然一下子就变得温顺了，夏夕绾那只小手伸了过来直接抚平了他头上的毛，这真的太神奇了。
<br/>
<br/>        这几个记者又get到了夏夕绾另一个技能，那就是驯兽。
<br/>
<br/>        夏夕绾澄亮的翦瞳落在陆寒霆的俊脸上，见他直勾勾的盯着她看，她有些不好意思的眨了眨纤长的羽捷，又大胆的看了过去，那意思是看什么看，就想给你生两个小宝宝！
<br/>
<br/>        陆寒霆的眼里再也看不到任何人，爱慕他的女人多的是，但是这么大胆炙烈的想给他生两个小宝宝的就只有她一个人了。
<br/>
<br/>        原来，他一直在她未来的计划里。
<br/>
<br/>        医学是她热爱的，而他是她最亲爱的。
<br/>
<br/>        媒体记者们抓着这一幕就“咔咔咔”的一阵乱拍，其实他们磕夏夕绾和陆子羡这对才子才女是没有任何恶意的，现在见夏夕绾和陆寒霆这么恩爱，大家纷纷笑道，
<br/>
<br/>        陆总，今天是夏才女的发布会，你果然没有缺席，我们陆总宠妻向来没输过。
<br/>
<br/>        陆总，陆太太都放话要给你生两个小宝宝了，请问你们的生子大计什么时候提上日程啊？
<br/>
<br/>        这场媒体发布会就在陆总和陆太太的造人计划里圆满结束了。
<br/>
<br/>        ……
<br/>
<br/>        媒体记者们都散场了，酒店回廊里，夏夕绾盈盈一握的纤腰被一只大手捞了过去，伸手一推，直接将她推到了墙壁上。
<br/>
<br/>        夏夕绾笑着躲避，两只小手抵着男人精硕的胸膛使劲往外推，“陆先生，你干什么，快点放开我，要是被那些记者们拍到，今天的娱乐头条一定是陆总陆太太迫不及待的在回廊里造人了！”
<br/>
<br/>        陆寒霆掐了一把她的杨柳腰，然后将她软若无骨的身体摁在自己的怀里，嗓音低哑道，“陆太太，别闹了，处.女怎么生宝宝，如果你真的有诚意，那今晚…”
<br/>
<br/>        夏夕绾迅速伸出小手捂住了他的嘴，不许他在这种场合乱说。
<br/>
<br/>        陆寒霆心里软成了水，这个小骗子就喜欢拿甜言蜜语来哄他，偏偏他就是吃她这一套。
<br/>
<br/>        他亲了亲她柔软的手心，然后往她柔白的指尖上亲去。
<br/>
<br/>        他怎么亲她手？
<br/>
<br/>        夏夕绾俏脸一红，她肿么觉得昨晚过后这男人的亲吻又变得色情了起来，她吓得迅速将小手给缩回去了，但小嘴哼哼道，“陆先生，我没有开玩笑，我就是要给你生两个小宝宝，你等着看好了！”
