　　第609章
<br/>　　草原上乱糟糟一片，一群人围在一起乱砍乱杀。呼延力在鹰的护持下，领着手下人拼杀。
<br/>　　虽然敌人的数量超过己方许多，不过能随呼延力出来打猎的，也定是部落中一等一的好手。如今这几十人围成一圈，形成一个堡垒，一时半刻野原火想要攻破也并非易事。
<br/>　　“他们想干什么，他们想干什么！”
<br/>　　呼延力骑在马上狂吼，他仔细搜索自己的记忆。确认自己并没有得罪过野原火，哪怕野原火在草原上搞出偌大风雨，他也没太担心对方。
<br/>　　我过我的小日子，你愿意做什么就做什么。你凭什么来打我的主意。
<br/>　　所以，呼延力此刻除了愤怒，还有困惑。
<br/>　　“大王勿慌，我让燕不归回城报信，等援兵一到，将他们一网打尽。”
<br/>　　“对对对。”呼延力忙不迭的答应：“燕不归，你速去速回，把大军领过来，将他们一个个都杀了。”
<br/>　　燕不归骑了一匹快马，挥舞着重剑往外闯，以他的本事，谁又拦得住，硬生生的趟开一条路，骑着马向外冲去。
<br/>　　他拍马疾行，想着返回王城求援。他心中并没有所谓的忠诚，不过白无双既然吩咐下来，他做到便是。毕竟不是什么太困难的事。
<br/>　　草原上一马平川，行不了多久就可以看见王城的城墙，等把城内的大军领过来，战场上谁杀谁委实说不好。
<br/>　　“好汉留步。”
<br/>　　身后忽然传来一声大喊，听到这声音燕不归就是一激灵，下意识回过头来。果然看到程大雷骑着一柄黑牛，气势汹汹的冲杀过来。
<br/>　　一见程大雷，燕不归完全将呼延力什么的抛在脑后。都过去这么长时间了，没想到这混蛋还留在草原上，这样再好没有，自己正好杀了他宽心。
<br/>　　他拨转马头向程大雷冲杀过来，二人在两马相错的一刹那，短暂交了一招。却也是棋逢对手将遇良才，谁也没占到谁的便宜。
<br/>　　小白狼在一旁提程大雷压阵，她手中提着刀，不过，程大雷与燕不归的对决，她的确没资格插手。
<br/>　　程大雷和燕不归各自坐在坐骑上，相对而立，程大雷的匹夫剑已经出鞘。目光盯着燕不归，脸上浮现一抹冷笑，他一直等待这样一个机会，与燕不归单独交手。
<br/>　　而燕不归何尝不是如此，他也想与程大雷来个了结。
<br/>　　“姓燕的，你我之间的账该算一算了。”
<br/>　　“来。”
<br/>　　二人几乎是同时从马背上起身，在半空中相撞，程大雷一招海底捞月，燕不归一招霸王盖顶。却也是谁也没伤到彼此，各自落在地面上。
<br/>　　程大雷没携带自己的本命武器，马上的功夫施展不开。而燕不归虽然是戎族，骑术自然不用说，可在马上也限制了他的重剑术。所以二人不约而同的选择马下步战，刚刚落地，便向对方攻过去。
<br/>　　二人完全是两种打法，程大雷占一个快字，一剑上刺，下刺，恒刺，斜刺……刺得鬼神莫测。而燕不归的重剑却是势沉力猛，进攻进攻不停进攻……
<br/>　　双方的交手打得难解难分，一时片刻谁也无法取胜，二人都存了要杀掉对方的心，所以打得十分凶险。这很明显将会是一场持久战，就算打上三天三夜也未必能分出胜负。
<br/>　　而燕不归的情况不好，呼延力的情况只能说更加糟糕。他手下的勇士就算再了得，也毕竟是血肉之躯，怎能顶得住对方一轮又一轮的冲锋。
<br/>　　眼看不停有手下被杀死，呼延力急得双目沁血。怎么会这样，怎么会这样……草原上还有那么多美女，自己还有那么多孩子没生，怎么可以死在这里。早上出城时，自己还是威风赫赫的草原王，怎么眨眼就有生死之忧。
<br/>　　“燕不归呢，燕不归呢，他为什么还不回来。”
<br/>　　呼延力心急如焚，目光看着远处，突然一指山坡上的野原火，口中道：“鹰，我要他的性命。”
<br/>　　我不惹你，你也不要惹我，如今你惹我，我只好杀掉你了。
<br/>　　这一刻，呼延力咬着牙想。
<br/>　　鹰亮出长弓，弓开如满月，远远的瞄准了野原火。就在一瞬间，野原火有种被锁定的感觉，目光下意识落在鹰的身上。
<br/>　　鹰，呼延力手下第一神射手，能双手开弓，射十三发连珠箭。
<br/>　　咚。
<br/>　　一根竹杖斜刺里冲过来，砸在鹰的胸口上。鹰一怔，目光呆呆的看着白无双。一瞬间，脑海中所有困惑都想清楚了。蛮雄为何被杀，今天为何会有这场稀里糊涂的狩猎，野原火为何会杀过来……一切都清楚了。
<br/>　　“大王，小心呐……”
<br/>　　他嘴里呐呐道出这句话，身子咚地一声坠马，长弓也随着落地，一代神射手，最后一箭终究是没有射出手。
<br/>　　“是你！”呼延力惊叫一声，目瞪口呆的看着白无双。
<br/>　　白无双轻轻低头，口中道：“大王勿怪。”
<br/>　　他之所以将燕不归支走，是因为燕不归的本事，他自量也胜不了。一个行将就木，一个如日初升，况且燕不归脑子不太正常，真如果自己早早暴露，会发生怎样的事，谁也说不好。
<br/>　　现在，他除掉鹰，已经为野原火扫清大患，也几乎决定这场突袭战的胜负。
<br/>　　草坡上，野原火领着众人冲了下来。一瞬间万马奔腾，如一道洪流。
<br/>　　呼延力呆呆的立在原地，什么都想不明白。只看到身边的人越来越少，曾经的弟兄渐渐消失。
<br/>　　终究是一代戎族王，在生死关头，他骨子里的血勇被激发出来。戎族儿郎，从没有怕死的。
<br/>　　他一刀斩在马身上，吃痛的战马狂奔，冲向野原火。两匹马一黑一红，狠狠的撞在一起，手中的弯刀砰地一声碰撞，冒出火花。
<br/>　　呼延力怒发冲冠，满口银牙都要咬碎，口中不停狂吼着：“为什么，为什么，为什么！”
<br/>　　他或许永远都想不明白原因，乱世这场赌局，有些人自然连上牌桌的机会都没有。可一旦上了牌桌，就绝对没有全身而退的道理。
<br/>　　你只有向前向前向前……正如野原火此刻做的一样。
<br/>(本章完)
