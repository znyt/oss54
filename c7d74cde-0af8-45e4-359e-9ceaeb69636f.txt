　　岁月悠悠，数百年时间眨眼即逝。
<br/>　　在这段时间内，人族的武祖玄天游历洪荒，出入各大名山大川之间，去过四海八荒，到过各大荒古遗迹，觅过九天十地。
<br/>　　一路游览，一路打杀，一路修炼，一路悟道。
<br/>　　遇到好东西，收了，遇到不顺眼的，灭了。
<br/>　　他一边收取各种顶级灵材，一边不断淬炼自己的武道，熬炼真气，打磨肉身，一点点的提升自己的修为。
<br/>　　一路走来，遇到过无数巫妖，他大多时候是不予理会，他又不是嗜杀之人。当然，别人主动招惹自己，他还是会出手的，总不能让人家欺负到自己头顶了，也不出手吧。
<br/>　　人族初生，修为较弱，现在洪荒之中的各大势力，谁也得罪不起，所以，人族如今若是不想被灭，需要找一个队友，或者靠山。
<br/>　　玄天第一时间想到的便是巫族。
<br/>　　巫族与妖族为洪荒最为强大的两方势力，实力自然不用多说。妖族诡异多变，最是不守信用，而且还以人族为食，自然不能与他们结盟。而祖巫就不一样了，虽说最后巫族还是会屠杀人族，但至少现在还不会。
<br/>　　如今人族缺少的就是时间，只有能够争取一段时间，将人族发展起来，玄天相信，人族不惧任何敌人。
<br/>　　巫族没有妖族那么多心思，短时间来说，最适合最盟友。至于以后的事，人族都已经发展起来了，还会怕吗？而巫族之中，又是后土最好说话。
<br/>　　于是，想明白了如今的形势，玄天向着洪荒北荒而去。
<br/>　　“人族玄天，求见后土祖巫！”
<br/>　　玄天来到后土部落，以浩大的声音，笼罩整个后土部落，刹那之间，大地震动！同时，他还稍微释放出一丝武皇的气息，他相信后土若是在部落里，定能感受到。
<br/>　　“什么人？”
<br/>　　这时，一群巫族成员见到玄天，快速走了过来，将他围起来，以防备巫族被突袭。
<br/>　　为首的一人是一个高大英俊的小伙，他此刻手持一张大弓，神情有些戒备地走向玄天，问道：
<br/>　　“你是何人？来我后土部落所谓何事？”
<br/>　　“小哥何人？我乃人族玄天，此番冒昧前来巫族，是为了求见后土祖巫。”
<br/>　　玄天看了他一眼，发现这的巫族小伙竟然是一名大巫，而且还是大巫后期，堪比大罗金仙后期了。他对着这大巫抱手道。
<br/>　　“我叫后羿，你来求见后土祖巫，怕是不行啊，祖巫自从上次大战之后，便一直在闭关，不肯见外人。”那大巫见来人不是妖族，便松了一口气，对着玄天说道。
<br/>　　“后羿？你叫后羿？”
<br/>　　玄天惊异道，他若有所思，眼前这人名叫后羿，又是大巫，恐怕就是日后箭落九日的后羿了。
<br/>　　“是啊，我就叫后羿，兄弟，你听说过我？”后羿惊声问道。不应该呀，我后羿虽然有些名声，但恐怕还没有传到人族吧？
<br/>　　“唔……大巫后羿，巫族中有名的神箭手，实力强大，我自然是听说过。”
<br/>　　玄天虽然之前没有听说过，但此刻说点赞美人的话，还是会说的。
<br/>　　“是吗？”后羿有些兴奋道，想不到我后羿还是挺有名的嘛。
<br/>　　“嗯。”
<br/>　　玄天淡淡点了点头，心道，好听的话，果然谁都爱听。
<br/>　　后羿刚要继续问话，但紧接着，一股厚实磅礴的力量，自大地之中衍生，一道身影，缓缓自后土部落之中走了出来。
<br/>　　玄天一眼看去，只见她身穿黄色衣裙，身材玲珑，娇容柔美，踏着碎步走向自己。女子容貌绝世，气质婉约，行动之间，玄天更是感受到一股厚重、强大至极的大地气息扑面而来。
<br/>　　她，正是土之祖巫后土。
<br/>　　“见过祖巫大人！”后羿等人一见来人，便行礼道。
<br/>　　后土对着后羿点了点头，便将目光投向玄天。
<br/>　　“原来是人族武祖到来，有失远迎了。”后土安然开口出声，语言平和，如传说中那般，她是十二祖巫中最为安静平和的一人。
<br/>　　玄天连忙道：“不敢！”随即，他将自己此行的目的一一说了出来。
<br/>　　闻言，后土不由得为之一阵惊讶：“玄天道友的意思，是想与我巫族联盟，可是，你别忘了，你们人族，可是妖族圣人女娲所创。”
<br/>　　玄天冷笑道：“女娲是妖族，但我人族却不是妖族，相反，妖族还在屠戮人族。人族只拜女娲，不拜妖族。说起来，人族与后土道友还是有些关系的。”
<br/>　　“哦？你说说看，虽然我很想帮你人族，但对于此事，我也无能为力。”后土淡淡道。
<br/>　　玄天一点也不意外，好似早就知道会是如此，后土作为十二祖巫之一，也不会无缘无故与一个种族联盟。他淡淡出声道：
<br/>　　“说起来，人族还是后土道友手中的九天息壤所创造出来的。可能道友还不知道，人族与巫族通婚，所生之人，有一半的几率为人族，一半的几率为巫族，而且，人巫结合生出的巫族，可以修炼元神。”
<br/>　　“什么？！”
<br/>　　后土闻言，顿时大惊失色道：“你不会骗我吧？”
<br/>　　玄天淡淡看了一眼后土，淡淡道：“事关人族之事，我身为人祖，又岂会编出这样的谎话来欺骗道友。更何况，是真是假，道友一试便知。”
<br/>　　见此，后土心中已经相信了七八分，当下准备暗中联系帝江等另外十一个祖巫，巫族本来生育就极为艰难，而且还不能修炼元神，如今听到这么一个消息，对他们来说，若是此事成真，对巫族来说，可是一个极大的机会。
<br/>　　随后，后土对玄天笑道：“妖族猖狂，屠杀无辜生灵，我巫族身为洪荒大地的守护者，自然不能任由妖族继续下去，玄天道友放心，我巫族愿与人族联盟。”
<br/>　　玄天闻之大喜，随即，便与后土祖巫派遣的一对巫族大军，以后羿为代表，随着玄天返回人族。
<br/>　　此时五百年之期也到了，人族外出历练的武者们也纷纷返回人族祖地不周山。当玄天带着巫族大军回到不周山之后，燧人氏等三祖带着人族众人前来迎接。
<br/>　　接着，玄天对人族解释了巫族将与人族联盟之事，他们经过一番思考，觉得此事可行，便同意了人巫联盟。
