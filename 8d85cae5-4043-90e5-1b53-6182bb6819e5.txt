        季思妍觉得，自己是不是应该就趁着现在这个孤男寡女共处一室的机会，主动朝傅昼景走过去，坐在他的大腿上，搂着他的脖子，在他的唇上印上一吻？
<br/>
<br/>        可是以傅昼景现在对她的态度，倘若她这么做的话，一定会糟他嫌弃。
<br/>
<br/>        季思妍脑海里想了想，还是就此罢休，继续操作自己的笔记本电脑弄设计稿。
<br/>
<br/>        童筱颖洗完澡从卫生间里出来后，发现傅昼景还未回家，担心他，便给他打了通电话。
<br/>
<br/>        此刻，这个点，傅昼景的手机振动了起来，引起了季思妍的好奇心。
<br/>
<br/>        因为和傅昼景之间还隔着玻璃隔断，季思妍只隐约间听到傅昼景语气温和地说道：“嗯，我马上就回来！”
<br/>
<br/>        紧接着，傅昼景挂了电话后，就开始收拾手边的文件并关了笔记本电脑。
<br/>
<br/>        当他手里提着公文包出来时，看到秘书办里的灯还是亮着的，才发现，那边办公桌前，季思妍还在。
<br/>
<br/>        季思妍见傅昼景要走，也连忙保存了文件，关了电脑，跟他打了声招呼：“我跟你一起下楼去。”
<br/>
<br/>        她原本以为傅昼景不会理会她，但意外的是，他居然为她驻足了。
<br/>
<br/>        然而，他仅仅只是出于，留她一个女性在办公室里不妥的安全因素考虑而已。
<br/>
<br/>        季思妍收拾完自己的东西，将手提包挂在了肩头，跟着傅昼景进了电梯。
<br/>
<br/>        幽闭的空间里，是她身上飘来的淡淡的香水味道。
<br/>
<br/>        傅昼景赫然发现，季思妍身上所用的香水，是五年前，他俩热恋时，他送给她的那瓶廉价的香水味道。
<br/>
<br/>        这个味道，他终生难忘。
<br/>
<br/>        也未曾想过，居然还有机会能闻到。
<br/>
<br/>        电梯抵达一层后，傅昼景先从电梯里走了出来，在大门口，两个人分道扬镳时，他才开口对季思妍说道：“那个牌子的香水已经不适合你了，你可以换一款更高贵的能配得上你身份的香水。”
<br/>
<br/>        “我……”季思妍欲言又止，只能眼睁睁地看着傅昼景渐行渐远。
<br/>
<br/>        他这话的言外之意，是让她忘了他吗？
<br/>
<br/>        他的意思是说他已经不适合她了，让她另外再找一个比他更好更能配得上她的男人？！
<br/>
<br/>        季思妍一想到这里，心口上就像数万只蚂蚁在撕咬一般，难受、痛苦，热泪盈眶。
<br/>
<br/>        阿景……
<br/>
<br/>        我忘不掉你……
<br/>
<br/>        怎么办？
<br/>
<br/>        她觉得自己真的很矛盾，一边为了家族企业，她又不得不去引诱傅夜沉，一边又放不下自己的旧爱，在这儿多愁善感。
<br/>
<br/>        季思妍时常觉得自己有十足的优越感，并对自己有足够的信心去同时搞定这两个男人，但是又对自己的所作所为表示怀疑。
<br/>
<br/>        因为心情不好，季思妍徒步回家，一路上显得有些跌跌撞撞。
<br/>
<br/>        傅昼景回到家后，就闻到了一股红烧牛肉的香味。
<br/>
<br/>        此时，童筱颖正巧端着一碗香气腾腾的米粉从厨房里走了出来。
<br/>
<br/>        工作到这个点，他还真有点儿肚子饿了。
<br/>
<br/>        童筱颖将米粉端上了一旁的四方小餐桌，并将手里的筷子给傅昼景递了过去。
<br/>
<br/>        “你不吃？”傅昼景接过筷子后，在餐桌前坐了下来。
<br/>
<br/>        童筱颖也跟着坐在他的对面，一脸幸福地看着他，微笑道：“我减肥啊！”
<br/>
<br/>        傅昼景忍俊不禁，反而起身去了厨房重新拿了一副碗筷过来，将桌上的红烧牛肉米粉给分成了两碗。
<br/>
<br/>        “我不怕你胖，就喜欢你和我一起吃！”傅昼景嘴角微扬，温和地说道。
<br/>
<br/>        童筱颖不再拒绝，而是端起傅昼景递过来的碗筷，跟他一起吃了起来。
<br/>
<br/>        “筱颖，对不起，这段日子我会很忙，可能没时间陪你。”傅昼景一脸抱歉地接着说。
<br/>
<br/>        童筱颖连连摇头，微笑着回答道：“没关系的！你忙！我会好好照顾自己的！”
<br/>
<br/>        “这是生活费，你收好。”傅昼景紧接着从裤兜里掏出一个钱包，将里面所有的现金全给了童筱颖。
<br/>
<br/>        童筱颖摆了摆手，笑颜如花地说道：“以后，家里生活上的开销由我全权负责好了！我已经找到工作了！在一家装饰公司做设计，每个月有提成拿！”
<br/>
<br/>        她说完这话时，其实内心是心虚的。
<br/>
<br/>        因为，她撒谎了。
<br/>
<br/>        昼景不喜欢她去会所那种地方，所以，她也是情非得已，不得不向昼景隐瞒事实。
<br/>
<br/>        除了会所那儿肯接收她打工，其他公司，压根就不把她放眼里。
<br/>
<br/>        童筱颖抿嘴尴尬地笑着，试图用笑容掩饰自己的谎言。
<br/>
<br/>        傅昼景却只是皱了下眉头，手里的现金依旧摊放在了餐桌上，不打算收回去。
<br/>
<br/>        童筱颖见他闷闷不乐，立即安慰道：“我知道，你不希望我出去工作。但是，这只是暂时的啊！我想和你一起分担你所有的喜怒哀乐。”
<br/>
<br/>        “这个月的生活费你先拿着。”傅昼景眉心稍稍舒展。
<br/>
<br/>        童筱颖犹豫了一下，还是面带微笑着拿起了桌上的钱，应了声：“好！”
<br/>
<br/>        他的工资卡，她已经还给他了，因为，她知道，他得用自己那每个月的工资去还银行的贷款。
<br/>
<br/>        童筱颖很想用自己的方式安慰一个自尊心很强的男人，但是，她发现，自己的所作所为，似乎有些适得其反。
<br/>
<br/>        傅昼景确实在心底自我嘲讽，他现在连自己的老婆都养不起，算什么男人？
<br/>
<br/>        两人接下来在沉默中各自吃着各自的米粉，吃完后，傅昼景洗漱完又去冲了个澡便直接躺沙发上睡了，白色浴袍加身，他浑身透着一股禁欲系气息。
<br/>
<br/>        童筱颖收拾完厨房，看到傅昼景又睡在沙发上，心里忽然间很不是滋味。
<br/>
<br/>        看到他已经合眼睡了，她只能蹑手蹑脚地爬上床，关了灯，轻轻地躺下。
<br/>
<br/>        之前住顶楼豪华套房的时候，她和他还有一墙之隔，现在两人挤在这单间里，一睁眼就能彼此看到对方，童筱颖心里其实更加踏实了许多，只是心里心疼他在沙发上会不会睡得不好。
<br/>
<br/>        直到手机振动的声音响起，突兀地打扰到两人间这静谧的睡眠时间。
<br/>
<br/>        傅昼景闭着眼睛，摸了半晌，才摸到自己的手机，看也不看，便直接接听了电话。
