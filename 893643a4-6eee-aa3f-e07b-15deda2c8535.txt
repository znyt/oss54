　　“什么？！”
<br/>　　贵妇全身气势大涨，将四周掀得动荡不已，一身实力不下于大罗金仙后期，突然，她眼中射出一道光芒进入敖玉身体之中，顿时大怒道：
<br/>　　“岂有此理！真是岂有此理！想不到还有人看不起我们龙族，看来是以前你父皇太过仁慈了。你先起来吧，等会儿到宝库找神龙果，你很快就能恢复过来。
<br/>　　叫你好好修炼，你就是不听，白瞎了你这一身的资质。”贵妇道。
<br/>　　“看来，还有很多人，不知道我龙族的威名啊！”
<br/>　　“母后，我知道错了，我今后一定会好好修炼的。”敖玉坚定道。
<br/>　　“嗯，希望此次打击能够真的让你认识到自己的不足，好好修炼。”
<br/>　　“儿臣谨记母后教诲。”敖玉躬身道。
<br/>　　“下去吧，我去找你父皇商量此事。”贵妇对着敖玉挥手道。
<br/>　　“是，母后，儿臣告退。”
<br/>　　在敖玉退下后，贵妇眼中闪过一丝怜爱，但转瞬便成为愤怒，接着朝着龙宫深处走去。
<br/>　　……
<br/>　　“嘶！自己取自己的精血还真是疼啊，不过，这一切都值了。
<br/>　　尽管任务没有时期限制，但还是要尽快完成。尊主所交任务，不容怠慢。”回到行宫后，敖玉坐在床上思索道。
<br/>　　“百滴真龙精血呐，要怎么办呢？”
<br/>　　“殿下，三殿下前来看望殿下。”就在此时，侍女前来报道。
<br/>　　“三太子？敖宇？他的消息倒是挺快的！”敖玉心中闪过一丝冷笑，紧接着，仿佛想到了什么，眼前一亮，道：
<br/>　　“真龙精血，你不就有吗？呵呵！”
<br/>　　……
<br/>　　元始不知那敖玉所演的苦肉计，况且，就算知道了也不会在意，他只会惊诧这小子的智商何时提高了。
<br/>　　不管敖玉如何折腾，元始已经在东海深处寻了百年了，他经过百年时间的推算，最后锁定在一片区域。再次经过数年侦查，终于发现了一丝不同寻常。
<br/>　　元始顺着感应急速而行，大海之上，根本不见任何踪影，百万里眨眼而过。
<br/>　　当抵达目的地时，眼中全是一片海洋，除此之外，再无其他。元始能肯定自己并没有算错，难道是有大阵掩护，有可能。但，自己如今又要如何进去呢？
<br/>　　凭立于虚空，元始闭上双眼，再次演算起来。不过一会儿的时间，元始就再次睁开双眼，眼中爆闪出一道金光，金光直入虚空，虚空之中顿时显现出一道道涟漪。
<br/>　　这一现象的出现，让元始就知道此地有秘密，显然是被什么东西遮掩住了，不想让人轻易发现。
<br/>　　“阵法？！”元始略带疑惑的声音响起，声音之中更多的却是肯定。
<br/>　　元始一指点出，虚空破碎出一道巨大的裂痕，一步踏入，元始就出现在阵法之中。
<br/>　　“原来如此，竟然与洪荒不在同一层空间之中，而且还有遮天大阵的掩护！”元始疑虑顿消，此地与洪荒不在同一层空间，怪不得如此难寻。
<br/>　　空间是分层的，每一层空间都各不相同，但，他们都依赖于洪荒这个巨大的空间。虚空穿梭就是进入另一层空间之中。隐藏于虚空，同样是藏在另一层虚空之中，让人发现不了。
<br/>　　虚空之道元始也是刚刚开始研究，并不是太熟，但，不熟悉并不意味着不认识。到了他这一层次，虽然还没有领悟空间法则，但使用一些空间法则皮毛还是可以的，只是没有空间法则那么厉害罢了。
<br/>　　如元始一步踏出，就是成百上千万里，这就是空间手段的简单实用。
<br/>　　元始进入大阵之中，并没有胡乱闯阵，而是就在原地站立，仔细地研究这遮天大阵。这大阵竟然能够遮掩天机，岂会平凡？
<br/>　　时间就在元始研究大阵之中缓缓流逝，眨眼之间，数百年时间过去了。
<br/>　　“呼——”
<br/>　　这天，元始长呼一口气，睁开双眼，眼中闪烁着耀眼金光，在他的视线之中，虚空之中顿时出现了密密麻麻的细线，许多细线都是黑色的，如同一张大网，将周围一切都包裹在内。而在这些细线之中，唯有一条是金色的，一直延伸至大阵深处。
<br/>　　元始沿着金色细线，一步一步往最深处走去。这遮天大阵他虽然没有研究透彻，但找到生路还是可以的，而且，这大阵的主要作用是遮掩天机与掩护阵内的东西，并不是杀敌所用，所以，元始并没有感到什么危机，最多就是被困在其中，不能出来。
<br/>　　不一会儿，元始就达到了大阵内部。大阵之中，充满混沌之气，元始远远就看到一座大大的岛屿，悬浮于虚空之中，岛上仙音袅袅，灵草仙根遍布，好一座仙家洞天福地。
<br/>　　元始并没有立刻进入仙岛，而是取出因果莲台，将此地的混沌之气尽数吸到混沌空间之中，之后再布下阵法，接引洪荒之中的先天灵气，这才走上仙岛。
<br/>　　走上仙岛，入眼所见的是一道高达三丈的竖碑，竖碑之上仅有“蓬莱”二字，除此之外，再无他物，就连碑文，装饰，纹理等都没有，甚至连仙光霞光都没有。一切都是那么的平凡，那么多普通。
<br/>　　但这落在元始眼中却是不平凡，因为“蓬莱”二字与这岛屿的意志连在一起，显得返璞归真，大道自然。
<br/>　　“唔，蓬莱，蓬莱，蓬莱岛么？”元始看着眼前的碑文，轻声呢喃道，接着又说道：“‘蓬莱’二字吾不喜欢，不如换成‘永恒’二字，一朝轮回，万道凋零，一世因果，唯我永恒！”
<br/>　　说完，元始挥手就要将石碑之上的碑文抹去，就在此时，岛屿开始剧烈震动，想要阻止元始的所作所为，不想让他抹去碑文。
<br/>　　“哼！这可由不得你，不周印，给我出！”
<br/>　　元始一声冷哼，不周神印顿时悬于岛屿上空，如同不周神山威压天地，压得蓬莱仙岛不得动弹。
<br/>　　元始挥手便将“蓬莱”二字毁去，再重新刻上“永恒”二字，字里行见可见众生轮回，大道凋零，日月腐朽，天地寂灭，混沌初开，唯我永恒，各种迹象。
<br/>　　只是眨眼间，这些现象就内敛其中，消失不见，各种道光仙气，也尽皆不见，只留下“永恒”二字，仿佛一直如此，亘古存在。
<br/>　　“嗯，很好。”
<br/>　　元始收起不周印，仙岛不再动荡，像是默认了永恒这个新的仙岛名字。
<br/>　　元始沿着仙岛走去，一路上，看到各种仙草灵根，仙果道树，神泉真水。
<br/>　　灵气汇聚，化成液体，奇花异草无数，人参灵芝遍地，青松翠柏长青，瑶草琪花绽放，灵鹤玄禽飞舞，端是令人心旷神怡。
<br/>　　此岛简直好了不能再好了，比起元始此前居住的昆仑山谷还要美妙。
<br/>　　这就是一处顶级的仙家福地！
