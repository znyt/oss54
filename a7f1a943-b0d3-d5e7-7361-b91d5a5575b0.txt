　　梦幻空间，准提圣人证道的空间。
<br/>　　如今，在此空间之中，发生剧烈大战，佛魔争锋，厮杀不断，谁为主宰？
<br/>　　准提与万劫魔尊两人不断交战在一起，瞬间就发生数次剧烈碰撞，风暴迭起，魔气惊天。
<br/>　　准提圣人身为梦幻空间的主人，一举一动，莫不带着惊天的天地威压，他调动空间之中的一切来镇压万劫魔尊。而万劫魔尊却是毫不畏惧，心中凶狠激发出来，魔威滔天，撼动神佛，撕裂天地，大战不断。
<br/>　　“哈哈，诸天劫气，为我魔源，道化魔种，给我聚！”万劫魔尊蓦然狞笑，冰冷的目光之中，不带任何生机，那是一种如天道般的冰冷与死寂。
<br/>　　“不好，心非心，梦非梦，梦幻无极，沉沦诸天！”
<br/>　　准提圣人大惊失色，仿佛是在万劫魔尊身上感受到了什么不可思议的东西，直接用出绝招，想要出手阻止。
<br/>　　“一切都该结束了！”
<br/>　　面对准提圣人的至强一击，万劫魔尊此时此刻居然没有一点慌张的意思，反而淡然的藐视道。
<br/>　　万劫魔尊屹立虚空之中，蓦然一张口，一吞一吐，瞬间便出现一道闪烁着乌黑光芒的魔种，魔种只有米粒般大小，但却周身散发着魔道道韵，大道气息弥漫，看上去便知道这东西十分不凡。
<br/>　　这颗魔种是万劫魔尊集一身魔道大成之作，乃是魔只极致的体现。
<br/>　　魔种是万劫魔尊以无数量劫之气凝聚，再加上自己的魔道本源温养无数年，它并没有其他什么特别厉害的作用，唯有一个作用，便是道心种魔。
<br/>　　所谓道心种魔，便是在别人的道心深处埋下一颗魔种，别人发现不了，或者就算发现了，也消除不了，他能够不断吸收被种魔之人的大道，作为养料，随之时间的发展，能够引发他们成为大魔头。
<br/>　　最重要的是，被种魔之人就相当于一个炉鼎，能够为魔种不断提供大道温养，最终魔种吸收完全部的大道，便会成为一颗全新的道种，也就相当于那人的道果。
<br/>　　道果成熟了，自然就可为采摘了，吞下一颗道果，就相当于得到那人的一切修为。
<br/>　　道心种魔，乃是非常可怕的神通。
<br/>　　下一刻，万劫魔尊的身影变得伟岸，邪意，乌黑的头发随风飘散，深邃的瞳孔包罗诸天，囊括寰宇，如天道一般，俯视苍生。
<br/>　　他的气息，浑厚如海，他的气势，磅礴如山！
<br/>　　他手捏玄妙复杂的印诀，手中的魔种顿时化为一道乌光，迅速没入准提圣人的身体之内。
<br/>　　“不好！”
<br/>　　准提圣人暗叫一声，想要将那道乌光拦住，可是，那乌光无形无相，刚一触碰到他的身体，便化为一道黑气清流，没入他的身体之内，转瞬间便消失不见。
<br/>　　准提圣人圣念连忙向着自己身体各处搜寻而去，一寸寸的血肉，一寸寸的神魂，他都没有放过，可惜，搜寻良久，他什么也没有发现，顿时，他心中一沉，感到不妙。
<br/>　　“你到底在我身上放了什么东西？”
<br/>　　准提圣人阴沉着脸，对着万劫魔尊质问道。他身上虽没有感觉到什么不适，但却本能的感觉到有些不妙。
<br/>　　“好东西，希望你能够喜欢！哈哈！”万劫魔尊肆意大笑道。
<br/>　　“可恶！万劫魔尊，看打！”准提圣人脸色阴沉，心情糟糕，大吼一声，施展出种种神通道术，向着万劫魔尊杀去。
<br/>　　“哼！我若为魔，天下无佛！诸天神佛，烟消云散！”
<br/>　　只听万劫魔尊一声冷哼，震动诸天，无边煞气，覆盖苍穹。这一刻，梦幻空间中所有的生灵，蓦然发现，他们的头顶，多出了一片天！一片对于他们怀揣这无尽恶意的天，充满毁灭杀戮的天！
<br/>　　这一刻，浩瀚失色，苍穹暗淡，宇宙黑暗，诸天沉沦。
<br/>　　“纵天彻地，魔天灭世！”
<br/>　　此刻的万劫魔尊仿佛化身为梦幻空间的天道，体内无穷无尽的磅礴力量，遍布诸天，浩瀚魔威，轰动天地，一念之间，苍穹轰鸣。
<br/>　　“怎么可能？混元三重天，这是什么神通，你怎么一下子提升这么多？”虚空之中，看着万劫魔尊那恐怖的威压，准提圣人惊叫道。
<br/>　　“怎么不可能？此处乃梦幻空间，既是你的梦，亦是我之梦！我心无限，即梦无极！不要说这里只是梦幻中的佛国，就算是真实的，我也定将他覆灭！只因为，吾乃魔尊！”
<br/>　　下一刻，万劫魔尊心念一动，如清风拂过，整个梦幻空间忽然变得剧烈抖动起来，之后瞬间破碎，宛如镜子破碎一般，一块块空间碎片落下，发生剧烈爆炸，整个世界，亿万佛国，瞬间化为灰烬，只余下一道道泡影。
<br/>　　外界。
<br/>　　众人只看到准提圣人什么都没说，来到万劫魔尊面前，眼神一阵闪烁，之后，两人就几乎同时闭上了双眼，一道无形的力量自两人体内射出，彼此相连在一起。
<br/>　　谁也不能靠近此处，一旦有人偷袭攻击，就变遭到他们的共同反击。
<br/>　　“这是什么？”
<br/>　　“发生了什么事？两人怎么都不动了？”
<br/>　　“听说准提圣人有一大神通，名为梦中证道，难道这就是梦中证道，他们在梦中决斗？”
<br/>　　“可惜我们看不到决斗的真正情况？”
<br/>　　……
<br/>　　“梦中证道？好一个梦中证道，连万劫魔尊都避免不了！”女娲圣人脸色微凝，看着两人。
<br/>　　“梦幻空间？师弟，希望你能成功！”准提圣人默默道。
<br/>　　所有人都目不转睛地盯着万劫魔尊与准提圣人，心思各异的等待着结果。
<br/>　　……
<br/>　　梦幻空间与万劫的时间不同，里面一天，外界可能只是一小会儿。
<br/>　　忽然，万劫魔尊身体一震颤动，一道恐怖的威压从他身上升起，魔气纵横，威凌苍生，他缓缓睁开眼睛，眼中精光闪烁，魔光如闪电般射出，一股潮红充满他的脸上，之后，他的脸色变得苍白无比。
<br/>　　显然，这次对决，他也并不是没有受伤。
<br/>　　正当所有人都惊骇异常之时，那原先包裹着万劫魔尊两人的力量消失了，突然，准提圣人身体一阵抖动，他踉跄着倒退数步之后，才停了下来，脸色一阵潮红，正当所有人都一脸莫名之时。
<br/>　　“噗嗤！”
<br/>　　准提圣人一口逆血喷出，一道佛光在他身上不断涌动，他的脸色变得苍白无比。血滴飞溅，大地崩坏，山河破碎，虚空洞穿，空间瞬间燃起了熊熊圣火，这就是圣血的厉害，一滴血可毁灭星辰，一点也不夸张。
<br/>　　近乎所有人都瞪大了眼睛，满脸震惊与不可思议！
<br/>　　洪荒六圣之一的准提圣人，就这么……败了！
<br/>　　万劫魔尊，一个新晋的混元强者，就这么战胜了洪荒老牌圣人，真是……
<br/>　　他们不愿相信，但事实就摆在眼前，由不得他们不信。
<br/>　　……
