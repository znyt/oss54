　　数十万里之外。
<br/>　　一道狼狈的身影，在高空中疾速飞行，忽然，身影一颤，险些没从高空跌落。
<br/>　　身影，自是萧逸。
<br/>　　从武道长河封锁中逃离，他便一直远遁。
<br/>　　北隐宗大长老的手段，确实可怕。
<br/>　　那滔天的武道之力绞杀下，几乎无物能活。
<br/>　　萧逸也不行，顷刻便在里头重伤。
<br/>　　逃出武道长河的一瞬间，已是重伤垂危。
<br/>　　若非他身体素质惊人，恐怕根本撑不到逃出武道长河，更别说远遁。
<br/>　　这时便看出体修武者的过人之处了。
<br/>　　特别萧逸修的还是修罗战体这门逆天体修功法。
<br/>　　曾几何时，萧逸所需的妖兽精血这等修炼之物，甚至比他的庞大小世界所需修炼之物还要多。
<br/>　　可想而知修罗战体修炼之难。
<br/>　　不过，修罗战体达到第七层巅峰后，萧逸并无后面的功法，便也无法修炼。
<br/>　　即便如此，这还是给予了萧逸极其惊人的身体素质。
<br/>　　这是一点。
<br/>　　还有便是云生铠甲这件中品防御圣器的护身。
<br/>　　最后便是萧逸的诸多护身手段，以及吸摄灵脉后，元力爆发下的护身元力。
<br/>　　这层层防御之下，才让他撑到逃出武道长河，最后远遁。
<br/>　　但即使如此，他还是重伤垂危。
<br/>　　此时，体内，金焱圣火正不断修复着破损的五脏六腑，调养着伤势。
<br/>　　若非如此，他即便能逃出生天，怕也是无力逃离，甚至重伤而死。
<br/>　　不过无论如何。
<br/>　　他现今终究是逃脱了，也无生命之虞。
<br/>　　至于这些伤势，虽仍旧极重，但慢慢调养便是。
<br/>　　“说到底，还是修为太低，实力太弱了。”萧逸暗暗摇头。
<br/>　　事实上，他并不知道北隐宗大长老已经停止了对他的追杀。
<br/>　　但沿途势力的截杀已经停下，他却知道。
<br/>　　身上，仍旧有那丝武道力量的烙印，起码需要数天时间才能完全消散。
<br/>　　故他现今仍旧未有半分停留，不断疾驰着。
<br/>　　……
<br/>　　数天后。
<br/>　　萧逸松了口气后，他已能感知到身上那丝武道力量已完全消散。
<br/>　　也就是说，北隐宗大长老，休想再追踪到他。
<br/>　　天大地大，他大可去得。
<br/>　　不过，他现今的目的地，乃是风刹殿总殿。
<br/>　　……
<br/>　　大半个月后。
<br/>　　萧逸的身影，忽然停下。
<br/>　　“风刹地域，到了。”萧逸淡淡一笑。
<br/>　　原本，按他的估计，起码要两个月，才能到达风刹地域。
<br/>　　没想到，之前被北隐宗大长老追杀时的极限逃生，倒给他省了不少时间。
<br/>　　他逃离远遁时，也是朝着风刹地域的方向而全力远遁。
<br/>　　算上从天藏地域风刹殿主殿出发，到现今，也不过是一个月多些的时间。
<br/>　　嗖…
<br/>　　萧逸的身影，瞬间从高空落下。
<br/>　　风刹地域，同样是中域之内的闻名地域。
<br/>　　整个地域，异常辽阔。
<br/>　　而敢以‘风刹’二字命名之，也正是因为风刹殿总殿便坐落于此。
<br/>　　风刹殿总殿，这座上古八殿之一，统领着全中域所有风刹殿主殿、分殿。
<br/>　　那座散发着亘古气息的巨大建筑，哪怕萧逸远远观之，亦为之震撼。
<br/>　　进入风刹地域，呼…一阵罡风迎面而来。
<br/>　　整个地域，终年罡风不断，凌厉异常。
<br/>　　对于寻常武者来说，这里说不上好。
<br/>　　但对于风使来说，这里便终日都是极好的修炼之地。
<br/>　　剐人面庞得罡风，对于修习风之一道的武者来说，充斥着舒服以及惬意。
<br/>　　同时，整个风刹地域，没有任何势力。
<br/>　　唯独最中心，便是总殿坐落之地。
<br/>　　周遭，是一座座大型分殿。
<br/>　　中域之内，地域无数。
<br/>　　但唯独上古八殿，八座总殿的坐落地域，不存任何别的势力。
<br/>　　这是偌大个中域，对这八座超然之殿的一份崇敬。
<br/>　　萧逸一路疾行，不多时，来到了总殿之外。
<br/>　　亲身来到总殿之前，看着这巨大的建筑，萧逸不禁倒吸一口凉气。
<br/>　　整个总殿的占地范围，怕是不亚于一座大城。
<br/>　　萧逸缓步踏入其内。
<br/>　　率先映入眼帘的，便是前堂处与别的分殿一模一样的格局与布置。
<br/>　　只不过，这里的前堂，更为巨大，更为宽阔。
<br/>　　单单前堂这里，怕是就比别的整座分殿还要大。
<br/>　　任务处、办事处，一个个修为深厚的风使穿梭其中。
<br/>　　萧逸来到任务处，出示了令牌。
<br/>　　“萧逸副殿主？”任务处，一个执事先是一惊，随后苦笑一声。
<br/>　　“你就是那位总殿连下十道传令都未肯来，时常寻不到踪影的萧逸副殿主？”
<br/>　　“额。”萧逸尴尬地摸了摸鼻子，歉意道，“有些杂事缠身，脱不得身。”
<br/>　　执事摇了摇头，道，“你随我来吧。”
<br/>　　跟随这位执事，萧逸到了后堂。
<br/>　　后堂处，更为让萧逸一惊。
<br/>　　后堂之内，简直是一处城中城。
<br/>　　不过与正常城池不同的是，里头皆是一个个忙碌的风使。
<br/>　　中域之大，自不必多说。
<br/>　　而总殿这里，统领全中域所有分殿，自是任务与事务皆异常繁杂。
<br/>　　执事领着萧逸来到其中一个房间。
<br/>　　“承风殿主。”
<br/>　　房内，一老者批阅着卷宗。
<br/>　　执事躬身行了一礼。
<br/>　　“何事？”老者头也不抬，道。
<br/>　　“是萧逸副殿主来了。”执事回答道。
<br/>　　“嗯。”老者下意识地点点头。
<br/>　　下一秒，手中批阅，忽然一顿。
<br/>　　“萧逸？那个麻烦精？”
<br/>　　老者抬起头，锐利的眼睛，直视萧逸。
<br/>　　“这…”萧逸脸色一变。
<br/>　　老者那锐利的眼神，竟瞬间让他浑身难受，乃至浑身刺痛。
<br/>　　“哦。”老者收回了目光，看向执事，道，“通知另外九人前来总殿吧。”
<br/>　　“是。”执事领命，躬身离去。
<br/>　　房内，只剩老者与萧逸。
<br/>　　“参见承风殿主。”萧逸微微躬身。
<br/>　　承风，风刹殿十大殿主之一，中域当世有名的武道大能。
<br/>　　成名时间，无从考究。
<br/>　　只知，上一次出手时，已是数百年前。
<br/>　　承风殿主的名字，萧逸听说过，而所知道的信息，也仅仅是这些。
<br/>　　“不必多礼。”承风殿主摆摆手。
<br/>　　下一秒，承风殿主身影一闪，瞬间出现在萧逸身前。
<br/>　　“随我来吧。”
<br/>　　承风殿主一把捉住了萧逸的手臂。
<br/>　　“好快。”萧逸瞳孔一缩，不仅丝毫未能跟上他的速度，甚至反抗不了他这看似随意地一捉。
<br/>　　一个闪身间，承风殿主与萧逸同时消失在原地。
