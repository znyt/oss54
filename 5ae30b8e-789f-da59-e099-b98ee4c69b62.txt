　　第173章
<br/>　　青州，天一城。
<br/>　　天一生水，此地临着大海大河，渔业和船业都很发达，又因为距离边关极远，不像幽州苦寒之地经常遭受戎族骚扰。
<br/>　　可方伯山到了这里，距离戎族远远的，却依旧难以开心起来。
<br/>　　天一城城主与他是旧识，天一城落叶城挨着很近，双方生意上也有很多来往，平常关系还算不错。这次方伯山逃到这里，就是要投奔对方，等戎族杀够了抢够了，自己再回去还做自己的城主。
<br/>　　但几天待着，感受却不是很愉快。
<br/>　　对方也尽了地主之谊，请自己吃过几顿饭，酒席中便说起：你老兄来投奔我，自然保你安全，咱这里家大业大，还多你一张嘴嘛。当然，家大业大有家大业大对难处，一睁眼就有几百人管你要吃的，这种难处老兄你肯定会懂的。
<br/>　　懂，方伯山有什么不懂的，既然来投奔，怎么能一点好处不给对方。
<br/>　　今天一锭金，明天一锭银，珍珠玉器夫人也一定是喜欢的，拿去，拿去……
<br/>　　老兄，我这怎么好意思，不过说实在的，你那个小侄子还想要柄剑，听说老兄有把宝剑……
<br/>　　来天一城避难，方伯山是带了不少金银珠宝，可到了这里，他忍不住会想，自己带的家产还是自己的么……
<br/>　　在落叶城他是一人之下万人之上，除了幽州王他谁也不用看在眼里。而到了此地，自己和家眷只能住个小院子，吃穿用度自然不能和落叶城比，关键是到那里也得看别人脸色，哪怕对方仅仅是个丫鬟仆人呢，这感觉……就令方伯山有些不痛快了。
<br/>　　“城主，城主……”
<br/>　　乔野从院外跑进来。
<br/>　　“嘘，噤声！”方伯山警惕的看了看左右：“隔墙有耳，在这里别这么称呼我，免得别人觉得咱们有异心。”
<br/>　　“城主……没必要这么小心吧？”
<br/>　　“还是小心些好，我让你打听得事情打听得怎么样了？”
<br/>　　“我问了从幽州过来的人，他们走之前，戎族还没有杀过来。”
<br/>　　“还没杀过来，难道他们抢了黑石城就跑了。”方伯山琢磨着，如果是这样，自己就赶紧回去，不用寄人篱下，吃人白眼。
<br/>　　“城主，你说会不会是程大雷挡住了戎族？”
<br/>　　方伯山白了他一眼：“你是早上没睡醒么，凭几个山贼挡得住戎族，那可是北蛮部！”
<br/>　　“戎族既然破了黑石城，断然没有不继续南侵的道理。可时间过去这么久，他们还没杀到落叶城，或许，真的是程大雷挡住了戎族。”
<br/>　　方伯山陷入思索中，当排除掉所有可能后，最不可能的也成了真相。
<br/>　　可一个山贼，真能挡得住上万北蛮部铁骑么？
<br/>　　“再查，再探！”方伯山道：“摸清楚青牛山的动静！”
<br/>　　“是！”
<br/>　　……
<br/>　　幽州城，城头之上。
<br/>　　“多备硬弩大弓，若想守城，弓箭必不可少。”
<br/>　　杨龙停身披盔甲，快步行走在城墙上，巡视着自己的城池。
<br/>　　“已经让城里的工匠连夜赶制，一月之内，必能造好十万羽箭。”李善言紧紧跟在他身后。
<br/>　　“一月，北蛮部会给我一月时间么，快，要快！此战决不容有失！”
<br/>　　杨龙停快步行走，走到一个士兵身边停下，这兵丁模样还很年轻，脸上带着稚气。
<br/>　　杨龙停上下打量着他，拍了拍他帽子上的灰尘。
<br/>　　“今年多大了？”
<br/>　　“禀王上，十六！”
<br/>　　“好，好！”杨龙停重重拍了拍他的肩膀：“是个好男儿，要打仗了，怕么？”
<br/>　　“不怕！”兵丁挺直胸膛。
<br/>　　“哈哈，我都害怕，你怎么能不怕。”杨龙停笑了笑，忽然严肃道：“保家卫国，怕就不做了么！我和你们一起怕，大战来时，本王和你们一起杀敌！”
<br/>　　兵丁腰杆站得笔直，身体因激动而颤抖。
<br/>　　杨龙停拍拍他的肩膀，带着手下谋士武将离开。
<br/>　　直到杨龙停走远，兵丁依旧激动得满脸涨红。
<br/>　　“喂，那就是幽州王吧！没想到他会和小魏说话。”
<br/>　　“咱们的幽州王是个好人呐，我听说当年就是他赶跑了戎族！”
<br/>　　“跟着他，总是错不了的！”
<br/>　　一些士兵议论，而此刻杨龙停已经走远。他这几日的确没有好好休息，一直在检查城防。
<br/>　　大战来临，他不得不谨慎，虽然戎族可怕，但戎族也是欺软怕硬的。而幽州城算是整个幽州最坚固的城池，只要让他们知道难打，也就退了。反正幽州这么多城池，他们抢什么地方不是抢，不一定偏偏要抢最难打的幽州城么。
<br/>　　“下面那么多人在做什么？”杨龙停站在城门上往下望。
<br/>　　“都是从各地过来的难民，想要到城里避难。”
<br/>　　“将他们赶走，若有匪人趁机混进来怎么办！”杨龙停。
<br/>　　“已经派人赶过，可他们铁了心要进城。”
<br/>　　“赶不走就杀，这样简单的道理还用我来教你么！”杨龙停重重冷哼一声。
<br/>　　“报！”
<br/>　　一斥候爬上城头，在杨龙停面前跪倒：“禀王上，我已经打探过，戎族还未过青牛山！”
<br/>　　“还没过来。”杨龙停皱起眉头，按照时间推算不应该啊，毕竟戎族的行军速度还是蛮快的。
<br/>　　“青牛山那些土匪呢，他们去往那里逃了？”
<br/>　　“也没见到他们从青牛山出来。”
<br/>　　“咦！”杨龙停皱紧眉头，他下意识想起程大雷当初拦路的样子，难道这个傻瓜真的要挡戎族。
<br/>　　“再查，再探！弄清楚青牛山的情况！”
<br/>　　“是！”
<br/>　　青牛山是幽州最后一道险地，幽州又是帝国北地最后防线。青牛山破，则幽州无险可守，幽州破，则戎族可以长驱直入，冀州、青州、乃至京州皆陷入危地。
<br/>　　在被恐惧笼罩的疆域，大家皆知道戎族大举来犯，三十年前的事情会不会重演，众人心里都在打鼓。
<br/>　　诸多人与杨龙停一样，都派了斥候潜入青牛山，无形中，无数双眼睛搁在这里：各地城主、无力逃命的匪人，心心念念家里那点儿产业的富户……
<br/>　　名不见经传的青牛山从未像今日一样被人瞩目。
<br/>　　而对这些事情，程大雷一无所知，他正忙着磨刀霍霍。
<br/>(本章完)
