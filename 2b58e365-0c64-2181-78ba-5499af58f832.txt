　　江左心下一惊，脱口而出：“你怎么知道？”
<br/>　　鲁初雪继续翻着白眼，用笔轻敲着桌面：“人之常情，加上经验之谈。”
<br/>　　江左目瞪口呆，呐呐道：“你们这些能当上管理的真是厉害，什么都能猜的七七八八。”
<br/>　　鲁初雪难得的露出一抹笑意，带着些许认真道：“你们研究人员才是真厉害，屡屡创造奇迹。”
<br/>　　江左沉默了一会儿，正色道：“解毒剂其实没有等阶，也与其他药剂的状况不同。”
<br/>　　听到这个答案，鲁初雪的笔顿住，发愣的抬起头：“没有等阶？难道不该是不同程度的感染，配以不同的解毒剂么？”
<br/>　　“并非。”江左摇头道：“一个人受到感染的时候，我们并不能立即确定他的感染程度为几，所以解毒剂为统一标配。”
<br/>　　鲁初雪皱眉，不解问道：“难道不能通过采取感染者的血液分析？”
<br/>　　江左看着她露出一个笑容，有些玩味：“等到分析出结果的时候，还来得及吗？尤其是那些受到高等级丧尸或异种的攻击，而遭受感染的人呢？”
<br/>　　鲁初雪顿时愣住，这才发现她把问题想的过于简单，也考虑不周。
<br/>　　随即她又继续发问：“可如果为统一剂量，轻微感染和重度感染岂不是效果不一样？那解毒剂能否成功救下一个人不是也待定么？这原材料可不是大白菜，每一颗特殊丧尸的晶体都异常珍贵啊！”
<br/>　　江左的脸上依然带着莫名的笑意：“这就是解毒剂制造过程中的难点之一，不过我们院长已经解决了，其关键点就是我们的固有思维与实际情况有着偏差。”
<br/>　　鲁初雪眼神带着询问，显然听得一知半解。
<br/>　　“因为感染的并非普通病毒，是丧尸病毒，重点是基因而不是毒。”江左说着，解释道：“只要实现变异基因的可逆原理，一切就不在话下，所以剂量的区别并不是重点。”
<br/>　　鲁初雪听着，只觉得大脑思维跟不上节奏，只能打断江左滔滔不绝的解释，直接道：“我也不细问这其中的研究重点了，既然没有等阶区分，那你就说一下解毒剂的注意事项，我记下吧！”
<br/>　　江左将思维拉回，点头道：“只有一点，就是这解毒剂只对新人类有用。”
<br/>　　听闻这一回答的鲁初雪，再次愣住，忍不住下意识发出了一声惊疑：“啊？”
<br/>　　一次又一次的特例，让她对这回狼牙研究部的新发明，完全不得其解。
<br/>　　“为什么？”鲁初雪又问道。
<br/>　　江左皱起眉，回答道：“具体的原因我们也还在探索，但依照袁熙叶院长的推断，是因为新人类都已经打开了基因锁，实现基因变异的逆转相对简单，而普通人的基因锁处于封闭状态，进都进不去何来改变一说？”
<br/>　　鲁初雪愣愣点头，倍觉研究深奥的她只能附和着开口：“有道理……”
<br/>　　“还有，你继续写。”这时候的江左又想到了什么，道：“解毒剂也并非百分百有效，能否将感染祛除，除了这一关键药剂之外，还与个人的意志力有关。”
<br/>　　鲁初雪点头：“这一点可以接受。”
<br/>　　江左看着鲁初雪一点点记下要点，继续说道：“由于解毒剂的数量有限，并且与超级丧尸的数量存在直接联系，所以这些药剂依照一开始我们长官的意思，要求优先用在战争中遭受感染的新人类身上。”
<br/>　　“当然。”鲁初雪立即回答：“军队为先，狼牙为先。”
<br/>　　“那就好。”江左露出了笑容，松了一口气道：“要点都说完了，还有什么需要解释的么？”
<br/>　　看着有什么就说什么的江左，鲁初雪有些惊讶和不解：“你就不怕我问的多，暴露了解毒剂的调配原理么？”
<br/>　　江左好似听到了一个笑话一般的看着她：“我所知道的东西哪怕全部昭告天下，也不可能有人知道解毒剂调配的关键，这玩意儿我们研究院的几个人包括院长在内同时进行研究，过程中也没有相互隐瞒，但只有院长走完了这条路，我们哪怕知道如何继续下去却也只能中途退出，你难道不知道为什么？”
<br/>　　鲁初雪又一次呆住：“为什么？”
<br/>　　江左伸出一只手点了点自己的脑袋，神色认真无比：“智商问题。”
<br/>　　鲁初雪目瞪口呆，惊愕的一句话都说不出来，在这一瞬间她忽然觉得自己似乎在被无止境的嘲讽，但望着眼前江左无比认真的神色，她又确定对方没有这个意思。
<br/>　　果然，江左又继续说道：“不是我们不想继续，而是实在解读无能，袁熙叶院长的智力有多高难以估计。”
<br/>　　“好的。”鲁初雪抽·搐着嘴角，无奈的接受了这一现实：“如果方便的话，请说一下解毒剂制造的关键，既然说出来也没几个人能调配成功，那暴露也无妨，这将会标记为一则要点，方便我在上报的时候为你们提供更多便利。”
<br/>　　鲁初雪说的是实话，以狼牙研究部的名头提出的要求，其被重视的程度完全不及楚涵亲自出面的一次胡乱搅和。
<br/>　　所以在当下时间紧迫的情况下，研究部想要得到更多的特殊丧尸晶体，唯有让给出这些资源的基地了解解毒剂的真实性。
<br/>　　这种事情可不是你说研究出来了就研究出来了，一切都需要证据，更需要事实。
<br/>　　鲁初雪在狼牙研究部有了多次惊人成果后，才敢在这时候给予完全信任，这其中还有着段氏基地和狼牙基地之间的交情，以及有了何峰的传话。
<br/>　　若不然，鲁初雪也断然不敢仅凭江左的几句话，就如此冲动。
<br/>　　江左点点头，没有任何犹豫的回答：“可以告知，关键点就是特殊晶体内的成分，你应该知道特殊晶体来自于超级丧尸，而超级丧尸是每五十万丧尸中才会出现的二代变异丧尸，还是五阶之后才会有的。”
<br/>　　“这其中的联系就是变异进化。”江左说着的时候，语气带着一种神圣感：“丧尸会进化？从人类到丧尸再到超级丧尸，这其中有个规律，那就是一种循环。”
