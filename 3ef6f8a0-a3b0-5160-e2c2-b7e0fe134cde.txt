　　萧逸率先拿起书籍，认真翻看起来。
<br/>　　书的首页，赫然写着《修罗战体》四个大字，显然是一本功法。
<br/>　　书中所写，自然是关于修罗战体的修炼方法。而到了书的末尾，则是狂血玄君死前留下的感悟，以及关于他自己的一些事情。
<br/>　　修罗战体，炼体功法，分七层。
<br/>　　这门功法很是神奇，需吸收妖兽精血淬炼肉身，使武者的肉体强度无限接近于妖兽。
<br/>　　第一层圆满，肉体力量媲美先天九重实力的妖兽。
<br/>　　第二层圆满，肉体力量媲美洞玄九重实力的妖兽。
<br/>　　第三层圆满，肉体力量媲美破玄九重实力的妖兽。
<br/>　　依此类推....
<br/>　　七层圆满，成就修罗之身，战威无可匹，天地万物皆可一拳粉碎。
<br/>　　“好强大的功法。”萧逸不禁吃了一惊。
<br/>　　妖兽的身躯，远强于人类。
<br/>　　同境界同实力的人类武者与妖兽对战，几乎绝对不是对手。
<br/>　　若这本功法真如描述的这般神奇，那就真的强大得骇人了，甚至可以用逆天来形容。
<br/>　　“难怪狂血玄君有着斩杀地元境的实力，更是地元境之下第一武者。肉体媲美破玄九重妖兽，自身又有着破玄九重的修为和真气，两者结合，实力的确恐怖。”萧逸惊讶地自语道。
<br/>　　不过，修罗战体虽然强大，但修炼难度也极大。
<br/>　　哪怕是狂血玄君，也仅仅练到第三层圆满。
<br/>　　首先它极难领悟，就算绝世天才，也只有不足一成的几率能修炼。
<br/>　　其次，妖兽精血极为狂暴，内含狂暴气息，人类无法直接吞服。
<br/>　　若随意吞服，会导致真气紊乱，最终走火入魔。
<br/>　　修罗战体这门功法能巧妙地化解这些狂暴气息，但并非全部，只是大部分。
<br/>　　所以久而久之，这些残暴气息还是会不断积蓄，导致走火入魔。
<br/>　　根据狂血玄君的随笔记载，他就是炼到第三层圆满时，体内残暴气息已经多得快压制不住了。
<br/>　　令他每每战斗时，都会血脉喷张，状若疯狂，才被人誉为狂血玄君。
<br/>　　而天星雷鳄的精血中带有雷电属性，可以灭杀掉这些残暴气息。
<br/>　　加之恰好天星雷鳄意欲摧毁破玄城，他这才出手对付天星雷鳄。
<br/>　　当然了，结果是他以死封印了天星雷鳄。
<br/>　　萧逸暗暗皱眉，“这功法强大归强大，可是也太危险了。连狂血玄君这等惊采绝艳之辈都拿这些狂暴气息没辙，我若去修炼...”
<br/>　　“罢了。”萧逸忽然眉头一松，微笑起来，“萧家还等着我回去呢。”
<br/>　　如此强大的炼体功法，就算再危险也没人忍得住这等诱惑。
<br/>　　自己如今迫切需要实力。
<br/>　　加之，妖兽精血的狂暴气息问题，有不少方法可以解决。
<br/>　　有些特殊的天材地宝或者奇异宝物，可以消除这些狂暴气息。
<br/>　　萧逸收起修罗战体功法，看向第二样东西。
<br/>　　那是极品灵器血戮剑，但放在洞府中百年，被洞府内的禁制吸取了不少灵气。
<br/>　　剑内力量大减，如今勉强只能算是中品灵器。
<br/>　　但只要用真气去温养，数十年后还是可以恢复到极品灵器品阶的。
<br/>　　第三样东西，那玉瓶内装的是六级妖兽，紫炎狮王的精血。
<br/>　　六级妖兽可是地元境实力了，狂血玄君这都能斩杀，可想而知当年的他强大到什么地步。
<br/>　　不过当时他体内的残暴气息已经极多，到了压制不住的地步，导致暂时没有服用。
<br/>　　他本打算收服了天星雷鳄，消除体内的残暴气息后再服用的，没想到最终身死，没机会服用。
<br/>　　说起来，天星雷鳄不过是五级妖兽，狂血玄君本来是能够轻松收服的。
<br/>　　但交战时，天星雷鳄忽然突破到地元境实力，且又仗着雷鸣荒泽是自己的地盘，控雷力量大增，这才逼得狂血玄君以死来封印它。
<br/>　　言归正传。
<br/>　　书籍的末尾，记载了狂血玄君许多事情，包括与天星雷鳄的战斗，包括自己宝物的情况，怕是临死前不想自己就这么孤寂地死去，想在这世上多留一些念想吧。
<br/>　　说到宝物，幽魂面具、血戮剑、修罗战体，书上也详细交待了。
<br/>　　这原本不是他的东西，而是年轻时闯荡大陆，在某处神秘遗迹中所得。
<br/>　　幽魂面具和血戮剑没什么好说的，倒是修罗战体，只有上卷前四层，下卷后三层他并未拥有。
<br/>　　他本想将前四层炼至圆满后，便再次闯荡大陆，寻找下卷，可惜他没机会了。
<br/>　　死前，他将这三样东西留下，有缘人得之。
<br/>　　也希望有缘人能替他完成最后的心愿，有朝一日找到下卷。
<br/>　　萧逸无奈一笑，“也罢，就算只有上卷，也足够我修炼到地元境了。下卷能否找到，看机缘吧。”
<br/>　　说罢，萧逸对着床榻认真地鞠了一躬。
<br/>　　好歹得了人家这么多宝贝，鞠个躬道个谢也是应该的。
<br/>　　萧逸再次坐下蒲团，打开了玉瓶，准备将修罗战体修炼一番再出去。
<br/>　　一头妖兽，全身鲜血中，精血只有一丁点儿，装起来也只有一小瓶。
<br/>　　虽说会有残暴气息存在，但这才刚开始修炼，还不至于造成影响，离开洞府后再想办法解决便是。
<br/>　　天星雷鳄的雷电能灭掉残暴气息，它被囚禁在洞府内，萧逸凭借幽魂面具可随时去它那。
<br/>　　但以萧逸现在的微末实力，可不敢去惹它。
<br/>　　强大的妖兽，手段莫测，到了一定境界后，有些能口吐人言，甚至化成人形。
<br/>　　萧逸想着，日后自己实力有成时，再来会一会这天星雷鳄也不迟。
<br/>　　想罢，彻底开始修炼。
<br/>　　玉瓶中的妖兽精血，没敢一次性喝下，而是先喝一小口试试。
<br/>　　血液入腹，立刻按照修罗战体的运转方式转化能量。
<br/>　　不愧是六级妖兽紫炎狮王的精血，蕴含的能量极多，快速地游走于体内四肢百骸。
<br/>　　萧逸几乎可以明显感觉到，自己的肉体正以一种快速的方式增强着。
<br/>　　同时，一股暴动的气息在体内涌动，搞得自己心神不安，甚至让自己浑身难耐，有一种杀人的冲动。
<br/>　　“稳。”萧逸心头暗喝一声。
<br/>　　这也是修炼修罗战体的难处之一，妖兽精血上的不安和疯狂气息，很容易让人心神崩溃，陷入疯狂状态。
<br/>　　武者在修炼时若陷入这种状态，是非常危险的。
<br/>　　亏得萧逸心智过人，不多时便稳住了心神。
<br/>　　待得这股力量消耗光，萧逸发现自己所受的影响不大，便将玉瓶中的所有妖兽精血一口灌入。
<br/>　　再次按照修罗战体的修炼方法转化力量。
<br/>　　霎那间，肉体飞速增强，而体内那股残暴气息也浓郁得惊人。
<br/>　　萧逸的眼睛，竟开始变为猩红，浑身杀意尽出，面貌恐怖而狰狞。
<br/>　　“该死，紫炎狮王的残暴气息竟可怕如斯。”萧逸心头暗骂一声，竭力忍住那股暴走的冲动。
<br/>　　慢慢地，他再次稳住心神。
<br/>　　虽然双目仍旧猩红，仍旧有一股杀人的冲动。
<br/>　　但已对他不构成威胁，他完全可以忍到这些妖兽精血力量完全被转化。
<br/>　　等到结束修炼，再将这些气息给压制下去，不会对自己造成任何影响。
<br/>　　但，恰在此时，他的双目，竟开始出现了变化。
<br/>　　原本的猩红，渐渐变成了一蓝一红。
<br/>　　太阴太阳之眼猛地出现，彻底将猩红压下，连带着将残暴气息也瞬间压制。
<br/>　　“怎么回事？”萧逸忽然一惊，正常情况下，太阴太阳之眼只会在吸收武魂，或者观看丹方时出现。
<br/>　　下一秒，萧逸差点没惊得合不拢嘴。
<br/>　　体内的肆虐的残暴气息，竟被冰鸾剑武魂的力量强行禁锢，让其无法在自己体内四处游走。
<br/>　　而后，太阴太阳之眼的无形漩涡出现在自己体内，快速吸收被禁锢住的残暴气息。
<br/>　　不多时，残暴气息被吸收一空，在无形漩涡中自动溃散，再不复存在。
<br/>　　“不会吧，连残暴气息也能被你吸收消灭掉？”萧逸内视，看着体内的冰鸾剑，暗暗咋舌。
<br/>(本章完)
