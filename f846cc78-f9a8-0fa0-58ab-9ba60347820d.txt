　　在场剩下的三人，一齐出手了。

<br/>　　白萱脸色苍白的说道，这样三尊天才联手，所爆发出来的力量，简直是惊天动地，谁也无法正面硬抗这锋芒。
<br/>　　“退？”
<br/>　　楚风眠的嘴角，露出一抹冷笑。
<br/>　　“为何要退！”
<br/>　　“楚某说过，我要杀这四人，不过就是屠鸡宰狗罢了，一群蝼蚁，联合起来，也想翻天？”
<br/>　　楚风眠开口之际，他的身躯动了，不闪不避，脚下一动，直接手持剑锋，踏空一步，迎了上去。
<br/>　　“杀戮剑道！杀神降临！”
<br/>　　在楚风眠的背后，无尽杀意，猛然凝聚，化为了远古杀神，突然降临。
<br/>　　这远古杀神降临一刻，其中蕴含的恐怖血气，也是全然凝聚在了楚风眠的剑锋之上。
<br/>　　在这刹那之间，这杀戮一剑，已经斩出。
<br/>　　万丈血痕，当空爆开，在这无尽血气的力量之下，这三人的无数武技，近乎被吞噬。
<br/>　　没有任何一丝力量，可以逃过这血气的吞噬，在这无尽血气之中，楚风眠仿佛便是如同无敌的存在。

<br/>　　一股庞大的灵力波动，碰撞，交锋，这三人联手，在瞬间就被击败。

<br/>　　“这，绝不可能！”
<br/>　　只有云公子一人，还能勉强的站起来，他的眼神中，都满是不可置信。
<br/>　　他们三人联手，居然会败，这除非是生死境四重，五重之上的强者，才有可能做到这一切。
<br/>　　如今楚风眠以神力境四重的境界，足以达到这一步，这让他如何相信。
<br/>　　楚风眠的实力，在突破了神力境之后，几乎是有了质的飞跃，神力境的天才，就算是在妖孽，在楚风眠看来，也不过如同猪狗一般。
<br/>　　楚风眠心神一动，那罗辰，麟木的血气，直接都被他所吞噬。
<br/>　　这二人，虽然都只是神力境九重的境界，但是一身血气，丝毫不比生死境二重，三重的武者要差。
<br/>　　毕竟都是一方天才的人物，不知道是得到过多少奇遇，这些奇遇，都留在了他的力量之中。
<br/>　　如今楚风眠吞噬了他们的力量，也就意味着吞噬了他们之前所得到的一切奇遇。
<br/>　　“这两人，就这样死了？”
<br/>　　白萱在一旁，看着这一幕，都有些不可置信。
<br/>　　麟木，罗辰二人，都是北邙学院内门中的风云人物，将来定是板上钉钉晋升种子弟子的人。
<br/>　　如今居然就这样，死在了楚风眠的一剑之下。
<br/>　　楚风眠杀了他们，真的是如同屠鸡宰狗，没有任何的区别。
<br/>　　“这些人，敢来算计我，如今的陨落，便是命中注定的。”
<br/>　　楚风眠冷冷的开口道，他的视线，一下子转移了过去，看向了那云公子。
<br/>　　这云公子，居然是足以从楚风眠的这一剑下抵挡下来，倒是让楚风眠微微有些吃惊。
<br/>　　“哦？”
<br/>　　楚风眠看着云公子，一下子便是看出来，云公子足以抵挡得住刚刚那一剑的威力，不是因为他的实力。

<br/>　　楚风眠看着云公子，突然大手一挥，直接冲着那云公子手中的羽扇，抓了过去。
<br/>　　“神羽！”
<br/>　　云公子看到楚风眠的一只大手，破空而来，抓了过去。
<br/>　　云公子连连后退，煽动神雀羽扇，顿时又是五行灵力，凝聚而出，向着楚风眠的手掌，打了过去。
<br/>　　但是这无数武技，打在楚风眠的手上，居然是没有让楚风眠的手，产生一点伤痕。
<br/>　　楚风眠灵力一动，手一抖，这在上面的五行灵力，全然消散，同时楚风眠的手，也直接向着那神雀羽扇抓了过去。

<br/>　　云公子死死的抓着神雀羽扇，疯狂抵挡着楚风眠的力量。
