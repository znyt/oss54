其实他已经习惯了，他的生活一直是这样的。
<br/>
<br/>        但是现在，他的生活发生了翻天覆地的变化，他从来没有敢想仿佛一夜间奶奶就好了起来，在沐浴着阳光的厨房里，听到的都是欢声笑语，还有…女孩似娇还嗔的哼哼声。
<br/>
<br/>        陆寒霆的心慢慢的变柔，柔软到不可思议，他将目光锁定在夏夕绾那道纤柔绝丽的身影上，这一切都是因为她的到来吗？
<br/>
<br/>        为什么他身边的人都这么喜欢她？
<br/>
<br/>        奶奶，小圆圆，甚至是吴妈，都对厉嫣然不冷不热的，但是她们竟然都这么喜欢她。
<br/>
<br/>        这时老夫人一扭头就看到了站在门外的陆寒霆，她迅速招手，“阿霆，你起床了，今天你可起晚了，昨晚很晚才睡吗？”
<br/>
<br/>        陆寒霆下来了。
<br/>
<br/>        夏夕绾侧头，澄亮而水汪的翦瞳向他看了过去。
<br/>
<br/>        陆寒霆也在看她。
<br/>
<br/>        四目相对，彼此想到的都是昨晚发生的故事，夏夕绾先移开了目光，不看他了。
<br/>
<br/>        陆寒霆也收回了视线，他走到老夫人的身边半蹲在地，“奶奶，你今天怎么下来晒太阳了，平时请你下来你都不下来。”
<br/>
<br/>        老夫人拍了拍陆寒霆的大手，“那是因为绾绾说多晒太阳对我身体好啊，我就下来了，哦对了，我还没有给你介绍一下，这位就是我请回来的美厨娘夏夕绾，你叫她绾绾就可以了。”
<br/>
<br/>        老夫人一本正经的介绍道。
<br/>
<br/>        吴妈迅速笑道，“我看绾绾是小厨娘还差不多，不过美厨娘更对，哪家的厨娘有我家的美啊，少爷，你说对吧？”
<br/>
<br/>        听着这主仆俩在这里一唱一和，陆寒霆深邃的狭眸又落在了夏夕绾的身上，他知道她是来爬他床的，但是没想到她真的会下厨。
<br/>
<br/>        像厉嫣然就是十指不沾阳春水，昨晚她端着吴妈准备的营养餐来讨好他，他并没有戳穿。
<br/>
<br/>        夏夕绾看着才20岁左右的样子，现在这么小年纪的女孩子很少会下厨的了。
<br/>
<br/>        至于美不美…
<br/>
<br/>        她是很美。
<br/>
<br/>        那张纤柔绝丽的小脸让他很是心动。
<br/>
<br/>        她不就仗着自己长得漂亮，十分有姿色，才想着勾引他的吗？
<br/>
<br/>        陆寒霆抿了一下薄唇，“既然奶奶喜欢，就让她留下吧。”
<br/>
<br/>        老夫人很开心，“阿霆，那你不要忘记付给绾绾工资，绾绾还会医术，昨晚给我扎了一针，我今早就觉得神清气爽的，医生和厨娘的活儿都让绾绾给干了，你可要按照世面上最贵的酬劳来付给绾绾。”
<br/>
<br/>        陆寒霆点头，“知道了。”
<br/>
<br/>        “吴妈，你推我出去透透气，今儿的天气真好啊。”老夫人适时叫住了吴妈，两个人退了出去，将空间让出来。
<br/>
<br/>        ……
<br/>
<br/>        见奶奶和吴妈走了，陆寒霆单手抄裤兜里来到了夏夕绾的身后，他嗓音低沉磁性的开腔道，“我虽然不知道你用了什么办法讨得了我奶奶和吴妈的欢心，但是你最好不要在我的眼皮底下搞什么小动作，若是让我发现你利用她们，我会立刻让你消失，明白么？”
<br/>
<br/>        “搞什么小动作？”夏夕绾抬起澄亮的眸子看着他，“昨晚我都说了我不想勾.引你了，所以，你也不要庸人自扰，自作多情了。”
