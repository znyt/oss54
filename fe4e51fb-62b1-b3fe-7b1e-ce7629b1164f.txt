　　“混沌钟却是不能给你们了，但，我这里有两件不弱于混沌钟的存在，却是比混沌钟更适合你们。”
<br/>　　元始收起剩下的三个葫芦，对着老子与通天道。
<br/>　　两人听到前一句还有些失望，毕竟，混沌钟可是先天至宝，先天至宝谁不想要，但他们又不能为了一个先天至宝而搞得兄弟不和。直到听到后一句，他们就有些奇怪了，先天至宝可是就这么几件，不比先天至宝差，那又是什么，两人对此十分好奇。想来元始是不会让他们失望的。
<br/>　　“看。”
<br/>　　元始也不吊人胃口，直接袖袍一挥，洞府空中顿时便出现了两件散发着强大威压的灵宝。
<br/>　　两件灵宝，一件是拥有二十四节骨节，形似脊骨的长剑，上面刻着“昆吾”两字，剑身上散发着煌煌之威，凌厉无比。
<br/>　　正是元始战胜昆吾老祖的战利品——昆吾剑！
<br/>　　另一件灵宝是一枚灰色石印，它散发着苍茫悠远的气息，一看就知道这东西不属于洪荒，而是属于混沌。
<br/>　　这正是生死石印！
<br/>　　不管是昆吾剑，还是生死石印，都是元始战胜昆吾老祖之后的战利品。尤其是其中生死石印，给了元始极深的印象。他不仅通过生死石印对战领悟了道一印，而且后来通过生死石印领悟了生死法则的生死阴阳印。
<br/>　　这两件灵宝都是不弱于先天至宝的存在，其中昆吾剑是昆吾老祖的伴生灵宝，生死石印是昆吾老祖身为混沌魔神之时在混沌之中得到的，两者都十分非凡。
<br/>　　但对于元始来说，不管是昆吾剑，还是生死石印，都不适合他，虽然两者都威力巨大。昆吾剑明显对于剑道修士更适合，生死石印对生死阴阳太极之道更为适合，他的元始大道包容太广了，两者都不太适合他。他觉得，混沌钟比这两者都适合他。
<br/>　　没有为什么，元始觉得，他心里天生就喜欢鼎、钟、镜、塔，印之类容易演化道则的灵宝，混沌钟就符合他的选宝要求。
<br/>　　“这是什么？”两人问道。
<br/>　　“昆吾剑，昆吾老祖的半身灵宝，攻击力极为强大，不比先天至宝弱。最为适合通天。”说着，他将昆吾剑递给通天，之后，接着说道：
<br/>　　“生死石印，混沌中所得，含有生死太极法则，与老子的阴阳太极一道极为契合，最为适合老子。”
<br/>　　接着，他将生死石印递给老子。
<br/>　　“好剑，剑形如同脊椎骨，煌煌如烈日，白虹时切玉，紫气夜干星。真乃剑中之祖。”通天感受着剑身传来的精神，顿时毫不吝啬的称赞道。
<br/>　　“真是好剑，我喜欢！多谢大兄赐剑。”
<br/>　　接着，他对元始躬身谢道。
<br/>　　“嗯。”元始面不改色，接受他的谢意。
<br/>　　“多谢大兄赐印。”一旁的老子接过石印，他同样感受到石印之中的法则，对他很有帮助，他极为欣喜，对着元始感谢道。
<br/>　　“嗯，好了，此次紫霄宫一行，还需要消化，你们还是回去早早整理参悟，顺便简单炼化自己的灵宝。”
<br/>　　“我等先行告退。”
<br/>　　两人向着元始说了一句，便急匆匆地回去各自的宫殿，却是向着赶紧炼化灵宝去了。
<br/>　　元始摇了摇头，不再管他们，身形一闪，便来到了玉虚宫中闭关之所，吩咐四位弟子一声，便开始闭关了。
<br/>　　他并没有急着炼化灵宝，而是先将鸿钧讲道所得消化，只有完全消化的道，才能算自己的东西。
<br/>　　他结合自己的大道，将鸿钧所讲的大道转化为自己的，并从炼精化气开始，到大罗金仙境界，重新整理修炼一番，弥补自己的不足之处。
<br/>　　之后，他根据自己的玉清气特点，创出不同于之前《元始无名天经》的道经《玉清元始经》。
<br/>　　这可是他日后道统的传承，由不得他马虎。之前的《元始无名天经》明显是他自己修炼的功法，而从这部《玉清元始经》开始，才算得上是他以后的传承。
<br/>　　《玉清元始经》记载了从凡人到大罗金仙境界的修炼功法，以及一些神通秘术。当然，这只是暂时的，以后肯定还会有准圣，圣人境界的功法。
<br/>　　只是如今鸿钧道祖还没有讲到那些，他也就还不急着创出来。
<br/>　　花费了六千年时间的消化讲道所得并创出道经，之后，元始并没有出关，而是继续闭关。此次他却是要炼制三个葫芦灵宝与炼化先天至宝混沌钟。
<br/>　　元始一挥手，将三个葫芦取了出来。紫红、紫白、紫黑色的三个葫芦静静地漂浮在密室之中，绽放出耀眼的光彩，一股股强烈的法力波动在密室中动荡着，散发出惊人的威压。
<br/>　　元始仔细感应了一下，发现三个葫芦虽然同源，但并不是同属性。其中，紫红色葫芦中带有一股强烈的摄魂能力，其内宛若一个幽暗深邃而又恐怖的黑洞空间，隐隐给人一种魂魄不稳的感觉，仿佛魂魄下一刻便会被它吸走一般。
<br/>　　元始知道，若是没有他的干扰，此葫芦原本是会被红云道人取走，炼制成九九散魄葫芦，专门散人魂魄，极为厉害。但如今却是已经不可能到红云手中了。
<br/>　　第二个紫白葫芦，内蕴一股强烈的先天庚金本源。若是没有元始干扰，它会被太一取走，日后被炼成斩仙飞刀，葫芦之上有三尺毫光，只要对着葫芦一拜，唤一声“宝贝请转身”，便可斩下敌人六阳之首，端是厉害。
<br/>　　第三个紫黑葫芦内部蕴有一小型混沌空间，能够自发产生混沌之气，这才是元始本来应有的。
<br/>　　混沌之气谁也不嫌会多，更何况这里面还有一个混沌空间，若是装人、困人，大罗金仙以下者，不消一时三刻，便会被同化为混沌之气，也很是厉害。况且，这其中的混沌之气不管是用来修炼，还是炼器，都是不可多得的好东西。
<br/>　　而且，这里面的混沌之气还是最为精纯的混沌之气，并不像混沌之中的混沌气，那是最为狂暴的混沌气，一般人并不能用它来修炼，或者炼器。只有法里高深者，才能以大法力将之驯服炼化。而这葫芦之中的混沌气却并不需要如此。
<br/>　　故而，这葫芦也是一件不可多得的宝物。
<br/>　　接下来就是炼化时间了，元始花费时间将紫红葫芦炼制成一个散魄葫芦，他在里面更加入了吞噬同化道则神纹，威力强大无比。不仅能够摄人魂魄，还能将人吸入其中，将他一声修为吞噬，转化为精纯的能量，供人吸收炼化，端是不可思议。
<br/>　　名字不变，元始仍旧称它为“九九散魄葫芦”。
