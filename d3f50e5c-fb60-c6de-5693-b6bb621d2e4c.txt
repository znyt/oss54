        &lt;h3&gt;第三卷第一百零五章  凌少&lt;/h3&gt;
<br/>
<br/>        凌辰上飞机之前给宁雅发了一个信息；“蓝天白云，即将飞行，首都的天，是雾蒙蒙的天，因为有你，天空别样的蓝。”
<br/>
<br/>        宁雅没有及时到这条信息，因为工作太忙，公司里每个人就像是飞旋的陀螺，所以她就没有注意到这条信息，也没有回信息。
<br/>
<br/>        经过几个小时的飞行，凌辰和助手成紫安全抵达贵州。
<br/>
<br/>        贵州地处云贵高原，东靠湖南，南邻广西，西毗云南，北连四川和重庆；境内地势西高东低，矿产资源丰富。气温变化小，冬暖夏凉，气候宜人。
<br/>
<br/>        贵州有5个5a级旅游景点，7个4a级旅游景点。著名的黄果树瀑布就是国家5a级风景名胜区，1999年被大世界吉尼斯总部评为世界上最大的瀑布群。
<br/>
<br/>        世界自然遗产-中国丹霞，4a级风景区，唯一一个直接以行政区命名的风景区也是在贵州。
<br/>
<br/>        凌辰不是第一次来贵州了，他基本年年都会来，对这里已经非常熟悉。
<br/>
<br/>        打开手机，先看看有没有宁雅的信息，虽然有点失望，但是凌辰想着宁雅这几天很忙，心下又释然。
<br/>
<br/>        他是全国总代，经销商很多，总代却只有一个。
<br/>
<br/>        他的下面还会有一级代理商、二级代理商、经销商。
<br/>
<br/>        总代也不是说会一直在哪个位置上，有能力居上，要有销售业绩，而厂家也会根据每年的销售业绩来考核各级代理商。
<br/>
<br/>        而有能力取代他位置的就是几个一级代理商，在他们的这个行业里，有钱有销售业绩就是老大。
<br/>
<br/>        每一个代理商下面都会有很多的商店，有一些有钱的，而且销售业绩好的都会直接定制以自己公司名称的产品。
<br/>
<br/>        凌辰的实力不算是最强的，最强的当属甸城的一级代理商和沈阳的代理商，但是凌辰做人却是做得最好的。
<br/>
<br/>        凌辰和成紫到达酒店之后，稍事休息便准备去拜访厂家老总。
<br/>
<br/>        这是每年的例行习惯，每个人都是。
<br/>
<br/>        但是今年的老总拒绝了大家的拜访，让他们等消息，大家一起聚一聚。
<br/>
<br/>        成紫：“凌总，今年是不是会有变动？”
<br/>
<br/>        “看这样子估计会有情况。”凌辰不知道厂家会是什么想法，但是没有见到人谁也不知道情况。
<br/>
<br/>        “我去看看那几位有没有什么情况？”成紫说。
<br/>
<br/>        “先不着急，等着他们上门吧。”凌辰笃定的说，他是总代，他们肯定会来找他的，如果他去找他们似乎有点不妥，尤其是在情况不明的时候。
<br/>
<br/>        事实如此。
<br/>
<br/>        下午陆陆续续的有人来找凌辰，凌辰在他那总统套房里接见了各地的代理商，大家纷纷询问明年的情况，有的要福利，有的要销售量。
<br/>
<br/>        “凌总，这个我每年的量基本不够，明年能不能多给我申请一些？”
<br/>
<br/>        “凌总，市场价格不好控制，能不能统一一下价格，严格控制价格？”
<br/>
<br/>        “凌总，市面上有伪劣假冒的产品出现，建议厂家严查。”
<br/>
<br/>        “凌总，今年厂家都不和我们见面是个什么意思？让我们等通知？等到什么时间？”
<br/>
<br/>        大家七嘴八舌的问着，每个人似乎都急切的想从凌辰这里打听到情况。
<br/>
<br/>        凌辰脸上没有任何表情，大家对这个年轻的，浑身充满着凌厉气势的年轻总代从心里有一丝莫名的畏惧，虽然他对大家都很照顾，而且也会给大家争取到很多的福利，有时候会想办法给大家拿到单子，但是他似乎是冷血动物，感觉浑身都是冷冰冰的。
<br/>
<br/>        “大家就先不要打听情况了，厂家有厂家的计划，总代有总代的计划，你们提的要求我们会考虑的，我们会考虑到每一个人的利益的。”成紫替老板回答大家的问题。
<br/>
<br/>        大家还是看着凌辰，似乎想从凌辰这里得到准确的或者说是能让他们觉得安心的问题。
<br/>
<br/>        “都不要着急，将你们的资金准备充足就好。有钱还怕没事情干？”凌辰微微一笑，看着大家。
<br/>
<br/>        大家见凌辰笑了，心里才松了一口气，感觉房间的空气似乎流通起来了。
<br/>
<br/>        “那，凌总，晚上我们一起喝几盅？”
<br/>
<br/>        “是啊，是啊，好不容易聚在一起了，喝几盅吧？”
<br/>
<br/>        “我今天有些感冒，这几天跑了一趟北京，有些累了，改天吧！”凌辰拒绝了，顺便故意咳嗽了几声。
<br/>
<br/>        厂家不见他们，而他们都聚在一起吃吃喝喝，联络感情，岂不是让人误会他们抱团取暖？
<br/>
<br/>        “那，凌总，你休息，我们就不打扰你了。”大家纷纷告辞。
<br/>
<br/>        凌辰以感冒为借口，拒绝了和大家一起聚聚，本来他们也有机会，凌辰会天南地北的到处跑，见他们也是会见到的，但是，这一次是不能在厂家的眼皮子底下聚在一起。
<br/>
<br/>        往年大家到了，厂家大老总会来看他们的，而且还有厂家的人陪同，今年只是给大家安排了住处，让大家等通知，就连他这个的总代也是冷落下来，要么是要做重大的调整，要么就是厂家内部出现了问题，现在不是和大家一起吃吃喝喝的时候，只能低调的等候。
<br/>
<br/>        凌辰这边低调行事。
<br/>
<br/>        那些代理可不低调行事，大家相互打听消息，最后的结论是厂家连凌辰也不见，有可能会要换总代。
<br/>
<br/>        风向变了，明年最有资格拿到总代的是甸城的和沈阳的。
<br/>
<br/>        有些人纷纷的去找甸城和沈阳的。
<br/>
<br/>        甸城和沈阳又自立成派，想寻求大家的支持。
<br/>
<br/>        也有几个代理商找到凌辰发唠叨：“凌总，你看，这厂家还没有宣布呢，他们就嘚瑟起来了。看看那些趋炎附势的人我都觉得恶心。”
<br/>
<br/>        凌辰：“人之常情，踩低捧高是人的本性，不足为怪。”
<br/>
<br/>        “哼，甸城的何少华表面仁义道德，其实坏事做绝，听说对他们当地的经销商进行打压，利润基本都被他拿去了。”
<br/>
<br/>        “没有见过的事情就不要说，大家都是同行。再加上每个地方的策略都不太一样。这个事情还没有定夺呢，别担心。”凌辰出言安慰。
<br/>
<br/>        甸城的何少华他是知道的，据说背景强大，认识北京的一个老总，有钱又有势力，真是要和他抢夺总代的身份，他怕也是难以pk得过的。
<br/>
<br/>        不过事情不到最后，谁也不知道最后的结果是什么。
<br/>
<br/>        与其着急上火，不如安心等待，车到山前必有路，该是你的就是你的！
<br/>
<br/>        凌辰安下心来等待，刚好可以静静心，看看书，想一想他的小雅，小雅这几年的经历让他有些心疼，但是又无可奈何，好的是，陈子昂他也认识，人不错，小雅在她公司，他还能安心。他需要成长起来，为小雅撑起一片天。
<br/>
<br/>        总代的这个位置，谁也不能从他的手中拿走。
<br/>
<br/>        如果要想娶回小雅，只能有与其家族匹配的实力。
