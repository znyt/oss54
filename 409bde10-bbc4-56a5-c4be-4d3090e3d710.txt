
<br/>　　<br/>　　    齐峥坤的态度，自然就代表了玄门山字脉的态度，之前大师爷和易天逸他们就已经得到了玄门总堂开会商议的消息，当时相字脉没有商量出结果打算去总堂观望，而总堂的意思非常明显，连山和归藏必须要到手。<br/>　　<br/>　　    这两书包括奇门遁甲对玄门五脉的意义非常重，就像九阴真经对郭靖，乾坤大挪移对张无忌的意义一样，这两人学会之后顿时都一飞冲天了，而玄门五脉要是得到这三书，说是起飞都有点低调了，简直可以说是上天了。<br/>　　<br/>　　    玄门就是道门，他们的传承几乎有几千年的历史了，比茅山，龙虎和天师教这些道门正统传承还要久远得多，据说玄门的祖师就是道家三清嫡传弟子。<br/>　　<br/>　　    玄门对于这三书的渴望就跟一个人在沙漠里马上就要渴死的时候看见了一汪清水一样，恨不得一头扎在水里。<br/>　　<br/>　　    得到连山，归藏和奇门遁甲，就能光复往日玄门正宗的辉煌了。<br/>　　<br/>　　    王惊蛰很不解的看着齐峥坤和青爷，说道：“以你们和我家的关系，如果说借看我未必不会答应，为什么非得要采取这种方式呢？”<br/>　　<br/>　　    青爷脸色一僵，别过脑袋不知如何回答，齐峥坤替他答惑了：“借，就意味着是份天大的人情，你若借给我们了我这份人情我以后可能还不起，如果不借那又伤了感情，就算是你借，你也不会借出全部给我们，总归得留一些压箱底的在身上吧？我可不信你会那么无私，大方的把连山，归藏和奇门遁甲里的内容全都告诉了我，那你还不如去信佛好了，你这不比佛祖割肉喂鹰差到哪去了，人啊都是贪得无厌的，当官的想当大官，做生意的想做大生意，我们这些人想要的自然也更多，所以就只好从你身上下手了，不要怪我们太现实，而是所有人的人性都是贪婪的”<br/>　　<br/>　　    王惊蛰苦笑着说道：“说的真好，但也真残忍”<br/>　　<br/>　　    齐峥坤问道：“不过，我有点奇怪的是，你既然已经看出来了，那怎么早先不走呢？”<br/>　　<br/>　　    王惊蛰看着青爷说道：“叔，你是我爸爸的朋友，如果我和你们有了冲突，打伤了你或者他，可能还会死人，我总归得有理由和我爸爸交代吧？现在好了，我有交代的理由了，不过我还想问的是，同样我要是伤在你们手里，你又如何和我父亲交代呢？”<br/>　　<br/>　　    齐峥坤拦了下青爷，冷笑着说道：“既然已经撕破脸了，那还需要什么交代？有交代了也不能相逢一笑泯恩仇吧？那就索性把脸撕的粉碎好了，你在我们手里或者连山三书在我们手里，还有在乎你父亲是什么态度嘛？你都说了他都好多年没有出现了，没准以后永远都没办法再出现了！”<br/>　　<br/>　　    王惊蛰皱眉说道：“你说的最后一句话，把我心里对你们仅有的一点感情，给击垮了······”<br/>　　<br/>　　    “呵呵，这个时候还感情两字，不是太幼稚了么？”齐峥坤的巴掌忽然重重的拍在桌子上，随后院门外就传来了一阵嘈杂的脚步声。<br/>　　<br/>　　    自从王惊蛰和王冬至在陇西露面，南昆仑就把他们姐弟俩的消息来了个公之于众，消息就像是插了翅膀一样飞了出去，北出马的人知道了，玄门五脉的人知道了，茅山，龙虎，天师等教也知道了，但谁都没有玄门五脉的脚步最快，他们甚至还查到了王惊蛰已经来了长安的消息。<br/>　　<br/>　　    玄门五脉的弟子，抛开正统弟子不说，世间还有很多很多数不清的外门弟子，多到让人不可置信的地步了，这些人就像一棵大树延伸出去的枝叶和脉络，遍布整个华夏大地九百六十万平方公里。<br/>　　<br/>　　    玄门，山医命相卜，其中的医也就是医道，几乎现存的所有中医里能被称为大师级的人物，差不多都和医字脉有所关系，还有就是命卜两脉，精通算卦卜问，所以很多算命的先生都和这两门有密切联系，而一些出名的风水大师几乎都处于相字脉。<br/>　　<br/>　　    玄门，可能得算是古往今来最庞大的一个机构了。<br/>　　<br/>　　    这个机构一旦全力运转起来，其能量恐怕会超出所有人的预计。<br/>　　<br/>　　    玄门的能量甚至在几十年前还曾经引起过上面的注意，特别是那场天下皆知的破四旧事件，让玄门这个机构不得不变成了一团散沙，不在拧成一股绳了，五脉全部分开自治，总堂的凝聚力还有指挥力度也被削弱了不少。<br/>　　<br/>　　    很少有人知道的是，这是玄门和上面商量过后的结果，上边的意思是玄门这种组织机构不能在这么存在下去了，也是绝对不允许的，但玄门又不可能解散，毕竟涉及到的人太多了。<br/>　　<br/>　　    所以，玄门后来就变成了现在这样，五脉全部分开自治，各自为政，同时上面派遣专门的人员入驻玄门进行监控，然后玄门中也有人为国家服务，这算是个皆大欢喜的局面了。<br/>　　<br/>　　    不过，玄门虽然分家了，但有句话叫瘦死的骆驼比马大，就算分开了，山医命相卜的统称还是叫玄门，他们在面对利益或者生存相关的一件事时，还是会选择站在一起的。<br/>　　<br/>　　    这一次，面对连山，归藏和奇门遁甲三大奇书现世，一盘散沙的玄门就罕见的统一了意见。<br/>　　<br/>　　    长安城，这条不起眼的胡同里，这个狭小的院子中，一下子挤进来六七个人，这是玄门能用最快时间抵达长安的几个人，其中还有一些正奔波在路上。<br/>　　<br/>　　    齐峥坤缓缓的站了起来，青爷神色复杂而纠结的也站了起来，他看着王惊蛰说道：“惊蛰，我劝你还是束手吧”<br/>　　<br/>　　    “叫我王惊蛰吧······”王惊蛰说道：“从现在开始，你和我们姓王的关系，已经不允许你再叫的这么熟络了”
