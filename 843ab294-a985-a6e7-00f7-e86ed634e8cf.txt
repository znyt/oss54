　　第555章
<br/>　　读书人说来说去，也无非修身齐家治国平天下几件事而已。
<br/>　　夫子似乎没成过亲，也未出仕做过官，虽然他常常作出一副以天下为己任的模样，而事实上，这几件事都和他没多大关系。他能做的似乎也就只剩下修身一件事。
<br/>　　老实说，在这件事上他做得相当不错。一日三餐，按时睡觉，早晚静思，静坐常思自己过，闲谈不论他人非。
<br/>　　今天就算书院发生了这样的大事，也没能打乱他的作息时间。在童子伺候着洗过脚后，独自上了二楼。
<br/>　　夫子住的地方取名叫做三省楼，取一日三省吾身的意思。他踱着脚步，缓缓上了楼，点燃一支清香。每晚临睡前，他都要在房内打坐经思，回忆自己一天做过的事。
<br/>　　我今天有没有读书，有没有做坏事，有没有动邪念啊。
<br/>　　琢磨来琢磨去，等把所有事情都想通了，才肯睡去。
<br/>　　咚……咚……咚
<br/>　　打坐静思的夫子突然听到外面传来敲窗声。他猛然睁开眼睛，一向浑浊的双眼，在此刻变得清澈犀利。
<br/>　　咚……咚……咚
<br/>　　一声连着一声，不紧不慢，不缓不急。
<br/>　　夫子握住了刀，一柄割纸刀，身体向后，靠在桌上，盯着窗户。
<br/>　　咚……咚……咚
<br/>　　夫子抓起棋盒中的棋子，挥手投出，棋子穿透窗户纸，外面的响声终于停止了。
<br/>　　嗖。
<br/>　　耳边忽然响起破空声，夫子丢出的七枚棋子原样还了回来。夫子出手如电，手若鬼影一闪，将七枚棋子尽数接在掌心。
<br/>　　双方隔着窗户沉默着，夫子掂着手中的棋子，心中一惊，上面都有轻轻的裂痕。这当然不是夫子造成的，而是外面那个神秘人用手指捏裂。如此，指力堪称恐怖。
<br/>　　来得究竟是谁……
<br/>　　夫子心如电转，对方还未露面，便以令他觉得危险。这种危险的感觉，对夫子来说，却已是很久没体会过的事情了。
<br/>　　他目光看过去，窗户上出现七个破洞，而一个破洞中，杵着一只眼睛。
<br/>　　夫子在注视对方的时候，对方也在注视他。
<br/>　　“既然来了，为何不现身一见，装神弄鬼，又是为何？”夫子开口。
<br/>　　吱呀呀。
<br/>　　伴随着一阵声响，窗户被推开，一道人影闪进房间，立在夫子面前。
<br/>　　对方一身黑袍，披头散发，一张脸消瘦枯槁，宛若骷髅蒙了人皮，一双眼睛却是雪亮。
<br/>　　对方不人不鬼，半妖半魔，但夫子还是第一眼就认出了对方。
<br/>　　“好久不见。”夫子叹了口气，手中的割纸刀可没有放下：“没想到你还活着。”
<br/>　　“你不是也还没死么。”孟玄青道：“看上去你比我活得好，但能不能先把刀放下，见到老朋友，你不准备好好招待也就罢了，一直举着刀算是怎么回事。”
<br/>　　“这么多年不见，我以为你已经死了。”夫子把刀搁在桌上，但仍是一伸手就可以握到得位置：“无事不登三宝殿，这次过来是为了什么事。”
<br/>　　“公主死了。”
<br/>　　“啊，她怎么死了。”夫子一惊，随之平静下来：“这么多年过去，你我都老成这般模样，她……也该死了。”
<br/>　　孟玄青盯着他脸上得表情，半晌也叹了口气，并没有说什么。
<br/>　　李家皇族当年横扫十九国结束乱世，过程中自然造了很大杀孽，对于十九国得余孽当然是赶尽杀绝。亡国人复国之心不灭，共同联合在一起，组成十九国反武联盟。在那段时间，针对帝国皇族的刺杀行动层出不穷，自然也迎来了帝国的强力打压。
<br/>　　而这所谓的反武联盟，其实内部也不团结，各自有各自的利益诉求，往往秉着一种杀人放火你去，酒池肉林我来的心思。最后的分崩离析，也就成了自然而然的事情。
<br/>　　孟夫子便是故鲁国的王子，他在十九国联盟的帮助下建立书院，为复国联盟培养人才。只等得日后时机到了，振臂一呼，揭竿而起。
<br/>　　当然，时间过去太久了。当年制定这计划的人差不多都死了。而复国联盟年轻一代，其实也是处于后继无人的局面。就连孟夫子这样的人，也渐渐忘了当年建立书院的目的。他已这把年纪，所想的无非是多教出几个学生而已，至于什么时候死那也不是他能决定的。
<br/>　　……
<br/>　　今夜的行动对燕不归来说，目前还算顺利。只要将这位华神医带走，为黑贞治好伤，一切就又回到正确的轨道上。
<br/>　　但事情不可能按他所想的发展，最后他还是被人发现了。
<br/>　　发现他的是马猛。马猛肚子不舒服，晚上出来起夜，就看到一道人影鬼鬼祟祟，形迹可疑。
<br/>　　“什么人。”他当场就吼出声来。
<br/>　　燕不归一激灵，毕竟是做贼心虚，他那里敢耽搁，加快脚步向山下逃去。
<br/>　　若是不跑，马猛还未必敢确认。但对方如此一逃，立刻就令马猛察觉出来不对。
<br/>　　“来人呐，有贼来了。”
<br/>　　马猛身子原地纵起，一跃数丈，紧紧随在燕不归身后。这位老武师，虽然年岁已大了，可身上的轻身功夫仍旧是半点不弱。
<br/>　　燕不归毕竟背着一人，华正则的生得又肥胖些，速度自然被拖了下来。走到半山腰的观春台时，他被马猛赶上了。
<br/>　　“那里来得贼人，还不束手就擒，小心马爷爷的大刀。”马猛虚张声势，事实是他出来得急，没有带刀。这如果赤手空拳与对方动起手来，可有些危险。
<br/>　　马猛大呼小叫，也惊动了书院巡夜的人。张六一杆大棍扛在肩头，堵住了燕不归的去路。
<br/>　　“呵，好小子，又是你，今日六爷爷好好训训你。”
<br/>　　宋寒之手持一柄青锋剑，一跃数丈，沿着崎岖山路而来。
<br/>　　“好贼，你就是那盗书贼么。”
<br/>　　燕不归不露面则已，一露面就被三大高手围住。这三人任何一人的功夫都不弱于他。
<br/>　　程大雷隐在暗处，下意识咧起嘴：事情怎么搞成这般样子，书院的警惕性也太高了吧。
<br/>　　他寻思着自己要不要暗中帮燕不归一把，但面对三大高手，程大雷确实也是有心无力。
