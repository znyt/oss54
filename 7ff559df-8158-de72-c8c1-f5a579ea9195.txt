　　第580章
<br/>　　夫子一张脸由白变黑，冰魄神针见血封喉这一点，在他身上得到验证。他刺向燕不归的一剑，终究是没有递出。
<br/>　　燕不归满脸焦黑，爆炸将他轰成炭人。但他仍然活着，重剑砍在李莫愁身上。
<br/>　　程大雷不得不承认一个事实，李莫愁已经被杀死。在刚才的一瞬间，究竟发生了什么，怕是身处当场的人都不知道，更何况距离稍远的李莫愁。
<br/>　　但她的确被杀死了。
<br/>　　燕不归哈哈大笑，口中道：“程大雷，我也杀你一个女人，天大地大，看你去那里杀我。”
<br/>　　燕不归的确有几分走火入魔，他以为程大雷这等人物，自然好色如命。但程大雷在这一点上，的确还蛮检点的。而这李莫愁其实和程大雷并没有多大关系。
<br/>　　燕不归自以为杀了程大雷，便能让程大雷体会自己所经历的痛苦，哈哈大笑一声，转身就向外逃去。
<br/>　　“想走。”
<br/>　　程大雷冷眉倒竖，百步飞剑这一招再次祭出。但燕不归逃得的确有些快，程大雷这一招也不是必杀之剑，只能眼睁睁看着他跃到悬崖边，噗通一声跳入大海。
<br/>　　这一招的确用得急了，程大雷疏忽了还有一个孟玄青。孟玄青虽然刚才也被爆炸波及，但并没有受太大的伤，而他今天出现在此地，早已存了必死之心。
<br/>　　死也要和程大雷玩命。
<br/>　　书院弟子见到夫子被杀，胸中也是一腔怒火，冲上来就要将程大雷碎尸万段。
<br/>　　程大雷用脚尖挑起夫子那柄竹中剑，一剑挥了出去。盛怒之下的一剑，威力又比平日暴涨几分。只是一剑，就穿了孟玄青的喉咙。
<br/>　　这老怪物一身毒，程大雷连竹中剑都弃了，就怕被毒物沾染。
<br/>　　他的尸体噗通一声倒地，血从喉咙中涌出，可奇怪的是，他脸上竟浮现一丝笑意。
<br/>　　他本是求死来的，死了反而心安，他或许早已明白，凭他这把老骨头，其实也做不了什么，这个江湖已不属于他了。
<br/>　　“谁来。”
<br/>　　程大雷回头爆喝一声，目光如冷刀一般扫过，张明理率领的书院弟子，却都被这一幕镇住，一时竟没人敢上前。
<br/>　　“好啦，好啦。”
<br/>　　李行哉这个时候站出来，大声道：“本王在此，你们还想做什么。也是读圣贤书长大的，为什么就不能学点好，怎么，想学人家造反啊？”
<br/>　　所以说秀才造反，三年不成，张明理一帮人的确被李行哉率领的官兵给镇住了。
<br/>　　可若让他们放下夫子被杀的仇恨，却也是不能。张明理咬着牙道：“这程大雷罪大恶极，乃是帝国通缉的要犯……”
<br/>　　“我当然知道他是要犯，我干嘛来的，我不就是来缉拿要犯的。”李行哉在蛤蟆寨住过一阵，深得蛤蟆城无耻的真传。他拍拍程大雷的肩膀：“他，被捉拿住了。”
<br/>　　“……”
<br/>　　一众人看着李行哉发号施令，两个官兵将程大雷押到大船之上。
<br/>　　“我说你们这些人，帝国要犯自然有帝国来拿，你们瞎操什么心。好了，都赶快回去吧，这夜黑风高的，你们这帮文弱书生，再遭了海盗什么的。”
<br/>　　就在书院众人的眼皮地下，李行哉将程大雷和李婉儿还有那头黑牛带到大船上，也不忘收敛李莫愁的尸体。
<br/>　　书院这帮弟子可不是什么文弱书生，七十二剑阵摆开，程大雷和燕不归两大高手都无法破阵。
<br/>　　但他们却也只能眼睁睁看着程大雷被带走，并没人想到能作什么。
<br/>　　书院本是一座为反抗帝国建立的学院，如今，他们显然是将建校的初心忘了。
<br/>　　船分波浪，开始返航，在船舱内，李婉儿、程大雷和李行哉坐在一起，互相述说经历的往事。
<br/>　　程大雷搞明白李行哉是为何来的，程大雷也将自己的经历说了一遍。
<br/>　　听罢之后，李行哉睁大眼睛，道：“我说书院的藏书楼怎么被烧了，原来是你干的呀。”
<br/>　　“什么我干的，那件事和我没关系。”程大雷心里有怨气，虽然我名声不好，但你也不能什么坏事都往我身上按。这件事要传出去，天下读书人话不把我家祖坟骂裂了。
<br/>　　“那个……该不会已经传出去了吧？”
<br/>　　“说不太好。”
<br/>　　程大雷咧咧嘴，这件事怕是会惹成大麻烦。
<br/>　　李行哉差人送进来酒，两人许久未见，却也有许多话想说。既然李行哉已经站稳了脚跟，程大雷也替他开心。
<br/>　　程大雷忽然响起一件事，问李行哉有没有在岛上见到一柄铁枪。
<br/>　　李行哉皱眉思索一阵，命手下去船上取来，道：“看着有些稀罕，是你说得这柄么？”
<br/>　　正是王无敌留下那柄霸王点金枪，程大雷交给李行哉，道：“你现在起码也是有身份的人了，该有柄趁手的兵器，我借花献佛，算是送你了。”
<br/>　　李行哉这才仔细打量这把枪，嘴里嘟囔一声：“我一直是有身份的人。”
<br/>　　“连婉儿我也一块儿送来了，安然无恙，交给你我也能松口气了。”
<br/>　　李行哉看了看程大雷，再看了看李婉儿，眼神颇有种意味深长的味道。半晌，他哈哈笑了一声：“收下了。”
<br/>　　随之，他话锋一转，道：“你下一步准备去哪儿，回凉州？”
<br/>　　“凉州是要回的，不过中途怕是要去一趟草原。”
<br/>　　“去报仇？”
<br/>　　“报仇是一件事，还有另外一件事。”程大雷道：“戎族这次专门派人来盗山水经，所图怕是不小，倒也要看看，他们这次想折腾什么。”
<br/>　　李行哉脸上的表情凝重起来，叹口气道：“是呐，若是戎族再打过来，现在的帝国怕挡不住呐。”
<br/>　　“原先好像也挡不住啊？”程大雷。
<br/>　　“呃……”李行哉无奈的叹口气：“所以你我都要早作准备才是，未来这世道还不知会变成什么样。”
<br/>　　二人在船舱中饮酒，各自也说了许多话。李行哉如今有了些家业，可这一路走来也不容易，扫荡海盗更是多有凶险，说起这些，他也是感慨颇多。
<br/>　　最后，他醉眼惺忪的看着程大雷。
<br/>　　“二帝十八王，未来必然是群雄逐鹿，老将折戟，英雄辈出。程大雷……生在这世道，你我不寂寞呐。”
<br/>(本章完)
