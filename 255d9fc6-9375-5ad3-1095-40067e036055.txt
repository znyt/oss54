　　“咔嚓！”一声惊雷。
<br/>　　敬事房管事愣了一下，先是抬头看了看天，又看了看李承乾：“殿，殿下，这，这个，不，不是……”
<br/>　　李承乾对这样的巧合也是感到无语了，不过他并不相信什么应誓不应誓，所以也就没有在乎，只是低头看着那管事：“今天的事情我可以当成没有发生过，那杨常林我自然也会找机会问他，但今后该怎么做你知道吧？”
<br/>　　“知道，知道！”管事把头点的飞快。
<br/>　　当初鬼迷心窍想着能在某些事情上压这些皇子一头，结果没想到反而坑了自己，如果再不知道怎么办，估计脑袋就保不住了。毕竟眼前这位可不是什么一般人，手底下人命那也是好几十万，估计也不会在乎多他一个。
<br/>　　……
<br/>　　打发走了那些敬事房的家伙，李承乾收拾了心情回了房间。
<br/>　　此时的杨雨馨已经换好了衣服，正坐在炕边等他回来。
<br/>　　见他进来有些担心的问道：“殿下，您，您这样做不会有事吧？”
<br/>　　“你放心吧，他们一定不敢多说半句。”李承乾笑着答道。
<br/>　　开玩笑，‘睚眦必报’的美名那可是无数事实堆起来的，只要那些内侍不是一心求死就不会在这件事情上再生什么事端。
<br/>　　当然，那杨常林必须揪出来，如果不把这小子整明白，估计以后的日子安生不了。
<br/>　　可是，揪出来之后怎么弄？杀了？太便宜他了！
<br/>　　思来想去李承乾脑中灵光一闪，‘啪’的打了一个响指！
<br/>　　……
<br/>　　两日之后，庭掖局。
<br/>　　杨常林如往常以样，清点了需要清洗的衣物，脑子里不由自主的转着这些薄如蝉翼的衣料穿在宫中那些美人身上时的样子……。
<br/>　　该死的，那里不由自主又开始疼了起来。
<br/>　　好好的一条根，就这么没了，每天只能看着一群美女在眼前穿花蝴蝶般的走来走去，却只能有心无力的感觉让杨常林有些郁闷。
<br/>　　这些女人可比他家里的那几房小妾强多了，如果能……，算了，那里又开始疼了。
<br/>　　“杨常林，把这些衣物送东宫去。”就在他胡思乱想的时候，庭掖局掌令带着两个内侍走了过来。
<br/>　　“啊？东，东宫？”听到东宫二字，杨常林打了一个哆嗦，他并没有忘记是谁把他变成现在这个样子的，而且伤还没有伤利索就要出来干活。
<br/>　　“怎么，你有什么意见么？”那掌令见杨常林犹豫，立刻脸色一沉。
<br/>　　“没，没有，马上就去，马上！”杨常林混久的江湖，如何看不出来这掌令对自己并不怎么满意，可是他眼也没有什么办法与和这位顶头上司较量。
<br/>　　毕竟在人屋檐下不能不低头，偌大的庭掖局，死上几个人估计连问的人都不会有，他杨常林可不想死在这里。
<br/>　　……
<br/>　　带着一车的衣料，杨常林出发了，一路上小心的四下打量，想要再找机会遇到一次宫里的贵人，说不定可以凭着一张嘴忽悠点好处，就算不能摆脱眼前的困境，如果能再坑一下把自己丢进这里的那小子也好啊。
<br/>　　可这一次杨常林注定失望了，一路行来不但一个贵人没有遇到，甚至连人都没有遇到几个。
<br/>　　好不容易到了东宫，本以为交了东西就可以走人，结果却被一个比较面熟的汉子拦了下来：“站住，例行检查！”
<br/>　　“我，我没进去啊。”杨常林退后了两步。
<br/>　　“没进也不行，检查！到那边的房子里去！”那汉子推了他一把。
<br/>　　没招儿，好汉不吃眼前亏，那就过去吧，并不知道接下来会发生什么的杨常林走向那个侍卫执守用的小屋。
<br/>　　刚一进去，这货就发现有些不对了，因为屋子里竟然什么都没有，除了一盆仙人球！
<br/>　　这是什么情况？为什么屋子里这么干净？难道……。
<br/>　　还没等他想完，身后那汉子说话了：“杨常林，太子殿下本来是给过你改过自新的机会，奈何你自己不知道珍惜。”
<br/>　　“不，这位将军，我……。”
<br/>　　“少特么跟老子废话，看到那边的仙人球了没？给你半个时辰时间，要么你曰了它，要么它曰了你，否则老子也不知道这里会不会失火。”
<br/>　　“啥？！”杨常林看着那仙人球眼珠子差点没有瞪爆喽。
<br/>　　这东西要怎么曰？再说老子那东西已经没了，还怎么曰？！
<br/>　　“哗啦……”正想着，门外传来锁门的声音，透过窗子还可以看到有人不断在向房子上泼着什么东西，一种淡淡的油味冲击杨常林的味觉。
<br/>　　这，这特么是让我死么？难道前几天跟皇帝陛下说的那些事情被人发现了？不，不可能啊，这话怎么会传到太子的耳朵里？
<br/>　　再说，太子不是都应谨小慎微么？为什么这位这么能折腾，竟然还敢这样明目张胆的报复，难道他就不怕被皇帝知道以后……，算了，还是别想那么多了，外面那些人已经在点火把，天知道这些人会不会提前把房子给点着。
<br/>　　强烈的救生欲之下，杨常林将目光投向了桌上那盆满是尖刺的仙人球。
<br/>　　……
<br/>　　半个时辰之后，杨常林颤抖着双腿从房间里出来了，房间里除了剩下一个空空的花盆，再无一物。
<br/>　　“行，老小子表现的不错，你可以走了。”那汉子在检查完房间之后，用十分佩服的目光看着杨常林，甚至还对他竖起了一根大拇指。
<br/>　　杨常林的脸不断抽搐着，大颗在颗的眼泪还挂在眼角，撇着两条腿，带着跟着他来的小内侍和大车回了庭掖局，下辈子打死也不来东宫这地方了。
<br/>　　可是，事情真的会那么容易就结束么？李承乾这种腹黑男会如此容易就放过一个敢坑自己的骗子？
<br/>　　答案当然是否定的。
<br/>　　第二天一大早，就在杨常林好不容易处理好那个位置之后，那该死的掌令又来了，同样的随从，同样的大车，站在同样的位置，用同样的语气说道：“杨常林，把这些衣物送东宫去。”
<br/>　　瞬间，杨常林呆住了，体验了一回什么叫欲哭无泪！
