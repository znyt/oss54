　　小矬子们想的很好，但事实真的会按他想的那样发展么？
<br/>　　候君集也好，牛进达也罢，这些人全都知道归化人之所以去大唐请命到底是为了什么，他们会沾沾自喜？明明很快就可以吞下整个倭国，结果却硬生生被压制住攻势的老候会沾沾自喜？别扯蛋了好么？
<br/>　　所以那斋藤来献贺礼的行为在老候看来就是挑衅，连礼物是啥都没看就让人把那个斋藤一顿乱棍打了出去，另外还有些不解气，又命人去前面通知了那些归化人，给老子狠狠打他丫的，不在一个月内拿回一万人头，就特么等着在前面当炮灰吧！
<br/>　　……
<br/>　　八月的天气对于倭国来说已经开始变的有些寒冷，但好在没到降雪的时节，所以对打仗并没有什么影响。
<br/>　　正在与倭军对峙的秦吉胜听说了候君集的命令之后叫苦不迭，他们已经无力再进攻了，而且就算是攻过去，在无险可守的情况下也只能被对面的倭国再打回来。
<br/>　　也就是说这只是一场完全没有意义的消耗战，而消耗的全都是原本倭人自己的力量。
<br/>　　但是秦吉胜却没有办法违抗候君集的命令，毕竟人家十多万人在那里摆着呢，而且那种火铳的攻击力太强了，让他们望而生畏，根本提不起与唐军战斗的意志。
<br/>　　所以万般无奈之下他只能准备战斗，将进攻的命令传达下去，同时开始制定作战计划。
<br/>　　……
<br/>　　不得不说，这还真是计划不如变化，人算不如天算，山背大兄的计划虽然自认完美无缺，可是他万万没有想到，在他对面的那些归化人并没有松懈，反而正在集结，准备发动攻击。
<br/>　　所以当他率领二十万大军兵临城下的时候，愕然发现，在他的对面正有一支规模在十多万之间的归化人军队在严阵以待。
<br/>　　所谓狭路相逢勇者胜，计划被打乱的山背大兄这么一愣神的功夫，对面的归化人已经向他们发动了攻击，秦吉胜亲自带队，嚎叫着冲了上来，在他的身后跟着的便是无数的归化人。
<br/>　　大战瞬间暴发，两支军队几乎在没有任何准备的情况下撞到了一起。
<br/>　　曾经都是一个国家的子民，说着共同的语言，有着共同的习惯，但此时此刻他们却在刀兵相向。
<br/>　　惨叫声不绝于耳，杀戮蒙蔽了双眼，打了这么长时间的仗，双方的眼中只有仇恨。
<br/>　　山背大兄在双方刚的接触的时候就被打懵了，猝不及防之下他的第一个念头竟然是逃，但很快理智就告诉他，不能逃眼下这个时候若是逃了，后果只能是全军覆没，那样的话他就再也没有了反本的机会。
<br/>　　所以他硬生生的忍住了逃走的念头，咬着牙与扑上来的秦吉胜战到了一起。
<br/>　　没有什么招式，也没有什么技巧，大唐的横刀在他们手里像是斩马剑一样的巨大，挥动间带着呼呼的风声。
<br/>　　“哐，嘎吱……”刺耳的金属摩擦声中，秦吉胜与山背大兄撞到了一起，彼此间咬牙切齿的瞪着对方，大有将对方碎尸万段之意。
<br/>　　“腹部，你个叛徒，倭国到底有什么对不起你的地方，竟然让你引狼入室。”
<br/>　　“我的名字叫秦吉胜，我是归化人，不是倭人，这个国家本来就是我们的，是你们窃取了皇位，现在到了还给我们归化人的时候了。”
<br/>　　“腹部，你该死！”
<br/>　　“该死的是你，死吧，死吧，死吧！”
<br/>　　“哐哐哐……”一连串的战刀撞击声，两个在候君集看来没什么区别的矬子战到了一声，大声喝骂着，彼此将手里的刀砍向对方的要害。
<br/>　　……
<br/>　　远处，距离战场四五里的地方，一座不怎么高的山头上，李佑、杨天、松赞干布几人站在那里，各自举着望远镜观察着战场上的形式。
<br/>　　他们本来是准备看看秦吉胜是如何进攻的，可是没想到对面的倭人竟然也和他们有着一样的目的，所以当两支军队撞到一起的时候，李佑等人也是大吃一惊。
<br/>　　幸好当初候老头儿歪打正着的下达了进攻的命令，否则若是真的让这些倭人首先发起攻击，怕只真的会影响到最后倭国的战局。
<br/>　　“怎么样，松赞要不要赌一下谁会赢？”惊讶过后的李佑主动开起了盘口，对松赞干布发起邀请。
<br/>　　“殿下，数十万人的战斗现在下结论还为时过早，不如先看看再说如何？”松赞干布笑着摇了摇头，松甘岭一战之后，他从来不再去做没有把握的事情，尤其实会影响自己判断的事情更不会去做。
<br/>　　“你这人好生无趣！”李佑见松赞不与他赌便也没了兴趣，至于说杨天，很显然他并不够格，纵然他现在是太子的舅兄也一样，毕竟实力才是决定一切的根本。
<br/>　　“殿下，不如我们还是回去吧，这里太过危险，若是被那些倭人发现了，只怕我们再想离开就难了。”站在一旁的亲卫得了机会，立刻上前劝说，这是作为护卫的本职工作，马虎不得。
<br/>　　但李佑却摇了摇头：“没事儿，再看一会儿，你不觉得这很有意思么。”
<br/>　　有意思么？亲卫并不这样觉得，战场就是战场，不管战斗的人是谁，他代表的都只有死亡。
<br/>　　松赞干布对李佑的性格也有些无奈，叹了口气说道：“殿下，还是走吧，没什么可看的了，这么大的场面没有个把月根本不会有什么结果。”
<br/>　　“那是这些家伙怂，如果是我们的人上来，这二十万人早给他干挺了，还至于打一个月！切！”李佑这次没有反对，摇了摇头向山下走去。
<br/>　　而一边的杨天则是脸色有些难看，眼中闪过一丝莫名，但最后他还是没有说什么，跟在李佑的身后走下了小山。
<br/>　　一群人带着护卫顺着另外一条路向着唐军大营而去。
<br/>　　身后阵阵喊杀声依旧在持续，不断有人在战场中死去，而活着的人则继续冲向下一个对手，战争从来就不存在什么怜悯与同情，杀戮才是它的本质，自从它诞生的那一刻起，便已经失去了控制，人类只不过就是它的玩偶！
