　　开门的时候，刘哔习惯性的向前走了几步，突然脚下踩空，‘噗通’一声，摔在了下面的石板上。
<br/>　　抬头向上看了一眼，看见一排木质的珊栏，只是中间靠近自己房门的位置，不知为何却出现了一个缺口，正好容得一人通过。
<br/>　　刘哔坐起身来，揉着自己有些酸痛的腰部，就算是经过丧尸血统加成的身体，依旧受到了一些轻微的创伤。
<br/>　　似乎是被这边的动静吸引，外面传来了一阵嘈杂的脚步声，刘哔心中一惊，下意识的想要找个地方躲藏，临行之时却又止住了身体。
<br/>　　短短的一瞬间，他考虑了很多，系统既然把自己传送到了这个地方，那自然是有它的用意，或许给自己安排了什么身份也说不定。
<br/>　　毕竟是第一次传送，刘哔想要借着这一次机会，搞清楚一些事情，万一有什么意外发生，到时候再找借口脱身就可以了。
<br/>　　片刻，门外的脚步声逐渐清晰了起来，借着这段时间，刘哔也将周围的坏境查看了一遍，这是一个很复古的宅子，上下两层，房间不多，偶尔能看见一些黄色的空白符纸，平白增添了几丝阴森的感觉。
<br/>　　而他所处的地方，正是客厅，只是让他奇怪的是，这个地方似乎只有他一人居住，下面发生了这么大的动静，都没有人出来查看。
<br/>　　正当刘哔盯着二楼几个紧闭的房门出神的时候，“吱呀”一声，两个干瘦的奇怪中年男人推门走了进来。
<br/>　　“叮～您有新的系统任务，请宿主进入任务面板自行查看。”
<br/>　　见到这两个中年男人的瞬间，系统冰冷的声音也从脑海中传来，刘哔眯了眯眼，下意识的舔舐了一下口中的獠牙，站起身来。
<br/>　　“九叔好。”拱手打了个招呼，刘哔大概知道自己此刻所处的是什么世界了。
<br/>　　这是他很早之前看过的一个电影，但是一直都记忆犹深，这部电影也曾一度被奉为国产恐怖片的经典之作。
<br/>　　而那个满头银发的瘦弱中年人，就是人称‘九叔’的驱魔道长，一身法力不可小觑。
<br/>　　九叔旁边那个年龄要稍微小上一些的中年男子，脸上带着一副眼镜，面相斯文，手中还拿着一个模样怪异的铃钟。
<br/>　　这个人叫四目道长，是九叔的师弟，做着赶尸的活计，刘哔对他的印象不是特别深刻，但和九叔同出一门，想必实力也不会差到哪里。
<br/>　　“臭小子，三更半夜的，你在搞什么东西！还有，你小子胆越来越肥了啊，见到师父居然都不行礼了啊！”看到刘哔安然无恙的站在大厅中央，九叔明显松了口气，随后脸色一板，颇为严厉的冲刘哔训斥道。
<br/>　　讪讪一笑，刘哔脸上的表情有些尴尬，初来乍到，他还不太熟悉自己的身份，但是从只言片语上来判断，他似乎成为了九叔的徒弟？
<br/>　　这可不是一个好消息，毕竟自己现在的体质特殊，如果被九叔发现，难免会闹出什么事情来。
<br/>　　“师父，救命啊师父！”正在此时，外面隐约传来了一阵呼救声，随之而来的，是颇为杂乱的打斗声。
<br/>　　九叔脸色一惊，却再也顾不上刘哔，和师弟二人一起跑了出去，刘哔站在原地，稍一犹豫，也起步跟了上去。
<br/>　　按照原本的剧情发展，应该是义庄的那些僵尸发生暴动了，有九叔挡在前面，也不会发生什么危险，他想去近距离观察一下这个世界的僵尸。
<br/>　　如果逗留的时间够长，以后少不了要和这种东西接触，反正早晚都要面对，还不如趁此机会好好观察，这些经验说不定哪天就可以用上。
<br/>　　紧随着九叔的脚步，一行三人，来到一个错落有致的长廊，迎面撞过来一个留着西瓜头的矮小男子，一脸惊慌失措，跌跌撞撞的冲三人跑来。
<br/>　　“臭小子，慌慌张张的出什么事了！”见状，九叔眉头一皱，一把抓住这个矮小的男子，开口询问道。
<br/>　　跟在九叔后面的刘哔，眼中红光一闪，饶有兴致的打量了一下面前的矮小男子，这个人叫文才，是九叔的徒弟，也是他名义上的师兄。
<br/>　　听到九叔的声音，文才似乎找到了主心骨一般，眼中的惊慌逐渐褪去，一把抓住九叔的衣袖，指着身后的方向：“师父，里边，里边。”
<br/>　　狠狠瞪了文才一眼，九叔也知道情况紧急，招呼一声，三人脚也不停的赶了过去，转瞬之间，来到了义庄用来摆放棺材的屋子门前。
<br/>　　此时，屋门大敞，几只穿着官服的僵尸正向门外蹦跳着，脸上苍白一片，早就没有了先前贴上去的符纸。
<br/>　　刘哔仔细打量着这些僵尸，下意识的在脑海中进行武力判断，这些僵尸的行动笨拙，动作僵硬，看起来也不是很难对付的模样。
<br/>　　而这些僵尸中，有一只格外灵动，口中嘶吼着，向站在门口的三人扑来，刘哔嘴角一抽，他知道这个僵尸，是九叔的徒弟秋生假扮的。
<br/>　　果不其然，当九叔和四目道长把他放倒在地的时候，秋生连忙出声，表明了自己的身份。
<br/>　　门内的僵尸逐步逼近，九叔没空搭理自己的这个徒弟，连忙起身，手上掐了一个法决，和师弟二人同时冲了上去。
<br/>　　刘哔看了一眼地上的秋生，抬脚迈过，主动加入了战场，找到一只身处边缘的丧尸，出手试探了起来。
<br/>　　因为不太了解这个世界僵尸的实力，刘哔并没有丝毫留手，双手握拳，在高达二十点的力量加持下挥动着，带动起一片虎虎风声，显得英勇不凡。
<br/>　　但是这足以打穿墙壁的力量，落在这些僵尸身上时，都没起到太大的效果，这些僵尸的肉体如同一块钢铁一般，反倒是将刘哔的拳头震的生疼。
<br/>　　“不要用蛮力，用法术啊！”不远处，九叔正和三四个僵尸群殴着，百忙之中，看到了刘哔这边的险境，心中一突，急切的开口喝到。
<br/>　　看到刘哔这边有危险，先前被放倒的秋生，连忙来到了刘哔身边，双手颇为生涩的握了一个法决，点在了刘哔面前的僵尸额头上。
<br/>　　“小师弟不要怕，我来帮……”秋生得意的冲刘哔晃了晃手指头，口中的话语说到一半，却戛然而止。
<br/>　　他所看到的，是刘哔宛若野兽一般的赤红双目，在月光的照耀下，带着一股极度妖异的美感。
