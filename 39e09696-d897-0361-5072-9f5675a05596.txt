　　刷。
<br/>　　三阶丧尸举起大锤子就是砸了过来，林峰立即闪躲，完美的避开，顿时这个三阶丧尸露出了一大片的漏洞。
<br/>　　“力量很大，但是不够灵活。”
<br/>　　林峰立即一刀砍在对方后背。
<br/>　　刷。
<br/>　　这一刀直接给他一刀两半。
<br/>　　林峰摇了摇头，“不堪一击。”
<br/>　　挖走对方的晶石，林峰看了眼对方拿的大锤子，应该是什么建筑工地上的那种大号的。
<br/>　　林峰直接收入自己物品栏，是个好东西自然不能放过了。
<br/>　　这年头，武器也是很重要的。
<br/>　　拿着武士刀林峰不断杀戮。
<br/>　　这一条街道上少说有百来只丧尸，全部一窝蜂冲过来。
<br/>　　“来得好！”
<br/>　　林峰眼中杀机毕露。
<br/>　　狂龙出海！
<br/>　　一刀下去，瞬间十几头丧尸被一刀为二，其中一些丧尸没了下半身被腰斩了，居然还可以在地上爬，肚子里的肠子都在外面拖着，留下一地的鲜血看上去非常的恐怖。
<br/>　　“丧尸还真的是生命力顽强啊，这种情况下都不死？“
<br/>　　林峰一脚把头颅给踩爆彻底的结束他们可悲的生命。
<br/>　　吼吼吼。
<br/>　　然而剩下的更多丧尸依然不畏生死的冲了过来。
<br/>　　林峰武士刀疯狂的舞动，且战且退，利用地形来战斗。
<br/>　　刹那间就像是削水果一样，这群丧尸来多少死多少，在武士刀之下，全部毙命。
<br/>　　偶然几个突进到了面前的，林峰也都毫不犹豫的用飞镖爆头解决战斗。
<br/>　　林峰微微喘口气，看了眼一地的尸体，把他们的晶石一个个的收起来，然后继续出发。
<br/>　　而此时此刻。
<br/>　　曹龙的赌场门口。
<br/>　　梅姐满脸难看的站在外面。
<br/>　　“梅姐，龙哥死了。是被枪杀的。”
<br/>　　梅姐眼中精芒一闪。
<br/>　　之前他本来要走了，但毕竟他和曹龙夫妻一场，见到曹龙不肯跟他去安全区，一时之间心有不忍准备回来再劝劝。
<br/>　　没想到却看到了这一幕。
<br/>　　“梅姐，要不要让人查一查，看口径，应该是沙漠之鹰这种手枪近距离爆头造成的，整个江城有这武器的不多。而且基本上可以排除军队，军队的人，不会用这种武器。”
<br/>　　一个年轻人道。
<br/>　　“这件事情自然要查，不过不是现在，家族催我们催的紧，先去曙光基地再说吧。另外，把曹龙的尸体给我带回去，我要给他安葬。”
<br/>　　梅姐说完带上了一个墨镜上了车。
<br/>　　……
<br/>　　林峰此时找到了那个空投。
<br/>　　是个D级空投箱。
<br/>　　打开来这一次里面没有任何食物，只有一个东西。
<br/>　　“麻痹飞镖？”
<br/>　　“被击中的目标会产生肌肉痉挛。哪怕是丧尸都会痉挛。”
<br/>　　林峰扫了眼，不多不少，二十发，好东西，林峰全部收入物品栏，不知不觉，原本的80个物品格子居然又要满了。
<br/>　　“下一个距离这里不过三百米，不算远，看地图显示是在一个小区内。”
<br/>　　林峰立即继续出发。
<br/>　　这个小区是个老小区。
<br/>　　林峰扫了眼小区内，不少丧尸四处晃荡，“这个空投应该在小区居民楼的楼顶。”
<br/>　　林峰立即是冲了进去，将小区内的十几个丧尸砍翻，然后冲上了楼。
<br/>　　“握草。”
<br/>　　林峰刚进楼就吓了一跳，转角就是一个老奶奶丧尸，半边脸都没有了，牙齿都露在外面，林峰冷不丁的忽然看到这个吓了一跳。
<br/>　　不过直接一刀将她给砍了。
<br/>　　挖出对方晶石，林峰马不停蹄的上楼去了。
<br/>　　天台之上。
<br/>　　林峰刚来到这里就看到了几个尸体，估计是之前有人跑到这里来了，却依然难逃一死。
<br/>　　“这几个丧尸居然头没了。”
<br/>　　林峰看了眼那几个丧尸，都是身体还在，头部没了，而且也不在附近，明显被什么玩意给啃了或者被一些变态给拿走当收藏品了。
<br/>　　看伤口，有点像是被什么玩意的牙齿给咬下来的。
<br/>　　林峰暗自戒备，这附近应该有个实力很强的怪物存在。
<br/>　　林峰慢慢朝天台上的空投走去。
<br/>　　这个空投也是个D级的，这几天一直都没有刷新过C级的。
<br/>　　打开来，居然有个背包。
<br/>　　叮！
<br/>　　拓展物品栏！目前为100格！
<br/>　　除了背包之外，林峰看到了一个非常可耻的东西，居然是个避孕套。
<br/>　　“我去，大爷的，能不能给点有用的，给我来点子弹，给避孕套算什么，是给我暗示什么吗？”
<br/>　　林峰无语了，不过给了那也不能丢了，万一以后用得着呢。
<br/>　　林峰全部丢入物品栏内。
<br/>　　“看来那个杀死这几个丧尸的怪物已经走远了。估计到其余地方捕食去了。”
<br/>　　林峰没有任何停留
<br/>　　一路狂奔林峰回到学校。
<br/>　　路过几家小卖铺，里面都已经是被许多幸存者霸占了，每个人都在疯了一样的抢夺。
<br/>　　但可惜的是在，这些小卖铺早就已经被人给拿的差不多了，根本没有多少吃的。
<br/>　　林峰摇了摇头，自己物品栏里还有一些吃的，倒也根本犯不着为此去担心，为此去卖命了，他现在要做的主要事情就是提升自己的实力。
<br/>　　“现在我身上的晶石怎么说可以让我突破到五阶了吧。”
<br/>　　林峰回到学校。
<br/>　　学校内一如往常。
<br/>　　林峰回来后第一时间就是把身上的晶石都用了，越高级的晶石体积越大，其中蕴含的所谓基因原液就越多，而且似乎越纯。
<br/>　　“既然可以吞服的话，那么我拿这个玩意来泡澡是不是同样可以强化自己的身体。”

<br/>　　两颗三阶晶石外加几百颗晶石。
<br/>　　林峰总算是突破到了五阶。
<br/>　　“很不错的感觉，一拳力量最起码也6000斤了吧。”
<br/>　　林峰握了握拳头。力量是最直观，最清晰，也是自己最能够直接察觉到明显变化的。

<br/>　　林峰发现每个人目光都看着他不禁好奇道。
