道，人们又安静下来，在听着。<br/>　　<br/>　　  “你们若是无法接受大可离开，离开新奉县搬去他处！”<br/>　　<br/>　　  听了这话，人们虽然默不作声，但眼中都有愤怒……<br/>　　<br/>　　  是啊，这是自己的家乡，现在却被外人占据，还要让自己离开，这是谁都无法接受的事情！<br/>　　<br/>　　  骆宾也是一阵着急，康少爷怎么能说出这样的话来，这不是加深矛盾吗？<br/>　　<br/>　　  而紧接着王康又是道：“但只要你们愿意留下，我保证能让你们过上好日子！”<br/>　　<br/>　　  “新奉县很穷，基本上没有可种出庄稼的地，城外到处都是山丘，谋生都成了一种奢望！”<br/>　　<br/>　　  “因而有的人，做了山匪！”<br/>　　<br/>　　  王康的目光在四周扫过，面对这目光，有人坦然，有人心虚的低下了头！<br/>　　<br/>　　  “无论是为了生存，还是为了其他，山匪是我绝对不会允许！而且还会严厉打击！”<br/>　　<br/>　　  “不要挑战我的底线，谁敢造次，当如我脚下人头观，这就是下场！”<br/>　　<br/>　　  王康的声音冷厉，充满着肃杀之气，尤其是他此刻正好站在一堆人头之后，如一个喋血的将军，震慑全场！<br/>　　<br/>　　  “而你们竟然还来找我闹，甚至有的人还喊出要我偿命！”<br/>　　<br/>　　  王康的目光落在那帮前来拦车的人身上，沉声道：“我就在此，谁想取我的命，随时恭候！”<br/>　　<br/>　　  “在此我也奉劝你们所有人，跟山匪有勾结，乘早脱离！”<br/>　　<br/>　　  “现在我允许你们把你们的儿子，丈夫亲人的头颅领去埋葬，但这只是第一次，也是最后一次！”<br/>　　<br/>　　  王康的声音冷厉，“在这之后就没有这样的机会了！”<br/>　　<br/>　　  说完王康跳下平板车，直接回到了自己的马车上，无人阻拦，无人说话，整个街道都是一片寂静，他们都在消化着王康刚才所说的话！<br/>　　<br/>　　  除去留下几十士卒，剩下都随同车队继续出发前往县衙。<br/>　　<br/>　　  情势比他想象的还要严重，无论是外部还是内部都有很大的阻力，王康知道在这个时期，怀柔是没有任何作用的！<br/>　　<br/>　　  他必须要用铁血手段将新奉县快速拿下，迟则生变！若有不服者，那便杀！<br/>　　<br/>　　  直至杀到所有人都服了为止……<br/>　　<br/>　　  【作者题外话】：书评区卡了吗？别让我有种单机的感觉，打广告的都比你们发的勤<br/><br/><br/>
