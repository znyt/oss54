　　“让开，全都给我让开！”
<br/>　　黑桃9手中握着手枪面目狰狞的对着酒店大厅中的人群嘶吼，酒店中保安原本想要上前去组织，可是看到黑桃9手中的手枪后，又重新退了回去。
<br/>　　黑桃9拿着手枪，快速朝着酒店外跑去。
<br/>　　布谷身体落下的时候，一把抓住了横在酒店外墙的旗杆，所幸的是这根金属旗杆的质量还算不错，没有因为突然承受了布谷下坠的重量而断裂。
<br/>　　布谷抓着旗杆晃动了两下，看到冲出酒店的黑桃9，手一松，朝着一辆停在路边的汽车跳落下去。
<br/>　　咚！
<br/>　　布谷落在那辆停在路边的宝马车上，这辆在国内需要接近五十万才能买到手的宝马车车顶被布谷踩得都凹陷下去。
<br/>　　黑桃9看到突然出现在自己面前的布谷，脸色顿时为之一变。
<br/>　　妈的，这家伙是怎么从楼上下来的，难道是从楼上跳下来的？！从十二层楼的外墙跳下来，这家伙是疯子吗？
<br/>　　黑桃9的脸色变得格外难看，抬起手枪就对着布谷的方向打出一个点射。
<br/>　　哒哒哒哒哒！
<br/>　　铿铿铿铿铿！
<br/>　　布谷动作敏捷的避开了黑桃9朝着自己打来的子弹，子弹落在他身后的宝马车上，原本就被布谷一脚踩得车顶都凹陷下去的宝马车，又被一串子弹打出一排弹孔，这辆车的车主如果看到自己的车被人摧残成这样，估计眼泪都会掉下来。
<br/>　　“妈的！”
<br/>　　黑桃9咒骂一声，也不再犹豫，掉头就跑。这个时候要想再去开车基本是不可能的。
<br/>　　“想跑？怎么可能！”
<br/>　　布谷嘴角勾起一抹冷笑，说道：“韦斯利这混蛋虽然好色了一点，但是他经常给小布丁买零食，小布丁如果知道韦斯利受伤了，可是会很伤心的。”
<br/>　　布谷拔腿朝着黑桃9追去。黑桃9在逃跑的过程中，不时扭头对着紧追在自己身后的布谷开枪射击，但是却都被布谷敏捷的闪避了开来，停在路边的汽车和行人遭受到了池鱼之殃。
<br/>　　布谷拔出自己插在腰后的手枪，在奔跑的过程中拉开套筒，瞄准了距离自己不到五十米的黑桃9，他没有瞄准黑桃9的脑袋，而是瞄准了他握着手枪的右手，眼前这家伙暂时还不能死，至少布谷要从他的嘴里问清楚情况才行。
<br/>　　嘡！
<br/>　　子弹脱膛而出，朝着黑桃9飞去，可是准头却差了一点，并没有能击中黑桃9，而是击中了停在路边的汽车，被子弹触发的汽车警报系统立刻发出尖锐的警报声。
<br/>　　“抱歉，我不是故意的！”
<br/>　　“该死！”
<br/>　　感觉到自己身后有子弹飞来，黑桃9再次转身对着紧追在自己身后的布谷扣动扳机。
<br/>　　哒哒哒，咔！
<br/>　　弹匣中最后三颗子弹打完后，枪膛中传来撞针击空的声响，黑桃9咬了咬牙，按下弹匣扣，退出打空的弹匣，从口袋中摸出一个备用弹匣塞入手枪中。
<br/>　　而在黑桃9更换备用弹匣的时候，布谷已经重新瞄准，然后再次扣动扳机，这次他没有再射偏，子弹准确的击中了黑桃9的右肩，他握在手中的手枪也跟着掉落在地上。
<br/>　　黑桃9捂着血流如注的右肩，脚步不停继续朝着前方狂奔，转进了自己左手边的一条只能容人和单车通过的小路中。
<br/>　　在入住酒店的时候，布谷就看过了酒店周围的地图，熟悉周围的道路的情况。这是夜鹰对布谷的要求，到了一个新的地方，首先要做的就是了解自己所在地方的情况，这样才能在发生紧急情况的时候，能在最短时间内想到应对措施，而不会在手忙脚乱中丢了自己的性命。
<br/>　　“老鼠能从猫的眼皮子底下逃跑吗？”布谷嘴角勾起一抹自信的微笑，说道：“答案显然是不可能的！”
<br/>　　布谷也是一个转身钻进了旁边的街道中，只不过，布谷可不会像黑桃9那样规规矩矩的跑正常的路线，他一个加速冲到路边的消防栓边，脚在地上用力一蹬，跳上消防栓，以消防栓作为受力点，又是用力的一踩，直接朝着路边的商铺跳去，在商铺中店员的惊呼声中，布谷已经上爬上了商铺的屋顶。
<br/>　　屋顶上的障碍虽然很多，但是对于布谷来说却是跟平地没有任何的区别。
<br/>　　鳞次栉比的房屋屋顶上，一个敏捷的身影在不断翻腾跳跃着，在快速奔跑中规避着阻挡在自己面前的障碍。
<br/>　　就算是世界顶级的跑酷高手，在看到布谷的动作后，都会为他的动作竖起大拇指。
<br/>　　黑桃9往自己身后看了眼，发现自己身后并没有人追来的时候，放慢了自己的脚步，重重的喘着粗气，嘴里骂咧道：“妈的，这个家伙还真是难缠，还好把他给甩开了。”
<br/>　　“你说得是我吗？”
<br/>　　布谷从旁边的屋顶上跳了下来，站在黑桃9面前。
<br/>　　“你这家伙……”
<br/>　　黑桃9左手取出一把战术折刀握在手中，摆出格斗的架势。
<br/>　　“你觉得以你现在状态能打赢我吗？”
<br/>　　布谷拔出别在自己腰后的HK45手枪，对准黑桃9，说道：“更何况，我的手里有枪。说吧，你们是什么人？为什么要对我们，不，对韦斯利下手？别说自己什么都不知道，我可以很有把握的告诉你，我有很多办法从你嘴里挖出我想要的东西，只是我不想浪费那个时间而已。”
<br/>　　黑桃9眼看自己逃跑无望，索性也就把事情全都交待了，这样自己至少还能保住一条命，大不了自己不再回黑桃小组，隐姓埋名，远走高飞。
<br/>　　“‘黑桃’暗杀小组？布克尔特？”
<br/>　　布谷轻轻皱眉，显然他对“黑桃”暗杀小组和布克尔特都没有任何的印象。
<br/>　　“我知道只有这么多。”黑桃9对布谷说道：“这次任务是布克尔特直接跟黑桃A交待的，我们只是按照黑桃A的安排做事。”
<br/>　　“黑桃A？”布谷对黑桃9问道：“就是出现在竞技场里的那三个人中领头的那个吗？”
<br/>　　“没错。”黑桃9点头回答道。
<br/>　　“你们在什么地方见面？”布谷问道。
