　　天尘宇没有带任何人，独自一人，离开了天王宫，向人王宫而去。
<br/>　　在天王宫，知道他和帝一魂关系的人都不多，而知道还有第三具分魂之人，除了帝一魂之外，就只有他一人了。
<br/>　　此事，暂时不宜公布与众，需要徐徐图之。
<br/>　　没过多久，天尘宇便来到了人王宫，经弟子通报之后，进入了人王宫之中。
<br/>　　“看，是无上神王天尘宇！”
<br/>　　“他怎么来我们人王宫了！”
<br/>　　“好出众的气质！”
<br/>　　天尘宇在一名人王宫弟子的带领之下，向人王宫的会客大殿走去，一路上受到了众多人王宫弟子的瞩目。
<br/>　　无上神王，这不仅是一种无上的荣耀，更是实力和地位的表现。
<br/>　　整个神域，只有十位无上神王，而轮回神朝，只有一人，那就是天尘宇。
<br/>　　每一位无上神王，曾经都是万古天骄，而且是万古天骄之中，最顶级的存在。
<br/>　　所以，天尘宇在轮回神朝的威望和名气，非常的大。
<br/>　　可以说，除了六宫的宫主，以及那两位从不露面的禁忌人物，没有人在声望方面能超过天尘宇。
<br/>　　天尘宇一身白袍，俊朗的面容，气质极为独特，他单手背负在身后，大步而行，不多时，便来到了人王宫的会客大殿。
<br/>　　会客大殿之中，人德宫主坐在高高的主座之上，面色威严而淡漠。
<br/>　　除此之外，会客大殿之中再无他人。
<br/>　　“晚辈见过人德宫主！”
<br/>　　天尘宇走进大殿之中，双手抱拳，向仁德宫主郑重一拜。
<br/>　　他虽然是无上神王，名震整个神域，但是毕竟是小辈，无论是实力，还是身份地位，都不能和人德宫主比拟。
<br/>　　所以，必要的礼节，还是不能少的。
<br/>　　“天尘宇，你来求见本宫主所谓何事？”
<br/>　　人德宫主面带威压的说道，虽然天尘宇是小辈，但他也不能置之不理。
<br/>　　“人德宫主，轮回古城之事，您老应该知晓吧？”天尘宇面色郑重的问道。
<br/>　　“轮回古城有何事？”人德神王闻言，眸中露出疑惑之色。
<br/>　　天尘宇见此，微微皱眉，对方的态度，让得心中微沉。
<br/>　　“轮回古城之中，贵宫弟子段惊天表现出众，盖压六宫妖孽，所向披靡！”天尘山沉声说道。
<br/>　　“是吗？”
<br/>　　人德宫主闻言微微惊讶，笑着说道：“人王宫居然出现了如此妖孽，老夫倒是没有注意！”
<br/>　　“人德宫主难道不知？”天尘山的脸色有些沉了下来，对方此举，看来肯定不会配合他了。
<br/>　　“老夫最近在闭关炼丹，并未关注宫中之事！”
<br/>　　人德宫主点了点头，随即疑惑的问道：“天尘宇，这段惊天难道是得罪了你？”
<br/>　　“这倒不是！”
<br/>　　天尘宇摇了摇头，他心中可不相信，人德会不知道段惊天的存在，对方显然是在糊弄他。
<br/>　　“既然他没有得罪你，那你来找老夫所谓何事？”人德宫主诧异的问道，面带不解之色。
<br/>　　“人德宫主，我既然来了，那就向您挑明了，这段惊天本名苏莫，乃是天尊大人的分魂！”天尘山郑重的说道，他也没有拐弯抹角。
<br/>　　毕竟，他相信苏莫的身份，人王宫肯定知晓。
<br/>　　“什么？”人德宫主闻言，顿时面色一惊，直接就站了起来。
<br/>　　“此言当真！”人德宫主目光如电，紧紧的盯着天尘山。
<br/>　　“自然当真，贵宫玄天神王亲自带队前去轮回古城，想必很熟悉苏莫。”天尘宇阴沉着脸说道。
<br/>　　他心中有些恼怒，这人德宫主也太能装了，这么大的事情，怎么可能一无所知。
<br/>　　“来人！”
<br/>　　人德宫主闻言，立刻向外朗喝一声，随即一名弟子走进了大殿。
<br/>　　“传玄天长老来此！”人德宫主道。
<br/>　　“是，宫主！”弟子躬身领命，随即便快速的离开。
<br/>　　而后，天尘宇和人德宫主，也没有再多言，静等玄天长老到来。
<br/>　　不多时，玄天长老便来到了大殿之中。
<br/>　　“父亲，又何要事？”
<br/>　　玄天神王来到大殿，先是大量了天尘宇一眼，随即向人德宫主行礼。
<br/>　　“玄天，段惊天你是否了解？”人德宫主问道。
<br/>　　“父亲，我正要向你禀报呢，这个段惊天天赋非常逆天，在轮回古城之中，超越了所有妖孽！”玄天长老沉声说道。
<br/>　　“嗯，他在何处？”人德宫主颌首，转而问道。
<br/>　　“他已经离开了人王宫！”玄天长老道。
<br/>　　“离开了？去了何地？”人德宫主皱眉问道。
<br/>　　“不清楚，从轮回古城回来之后，他说要外出历练，便离开了人王宫！”玄天长老道。
<br/>　　“查，立刻追查，务必找到他，将他带回宫中见老夫！”人德宫主大声吩咐道，显得凝重无比。
<br/>　　“父亲，这是为何？段惊天难道违反了宫规？”玄天长老面带不解之色。
<br/>　　“先找到他再说，不能让他离开轮回神朝！”人德宫主大声道。
<br/>　　“是！父亲！”玄天长老应声，转身就要离去。
<br/>　　天尘宇一直在看着人德宫主和玄天长老，心中暗暗腹诽。
<br/>　　这两人的表演，还真像是那么回事，但是，他不可能相信。
<br/>　　“且慢！”眼见玄天长老将要离开，天尘宇立刻开口了。
<br/>　　“尘宇贤侄，不知有何指教？”玄天长老转头，看向天尘宇。
<br/>　　因为天尘宇的父亲，乃是天王宫大长老，和他算是比较熟悉，所以，出于表明上的关系，他称对方一句贤侄。
<br/>　　“玄天长老，段惊天此人天赋逆天，你怎会让他离开人王宫？”天尘宇沉声问道，他心中清楚，这都是假的。
<br/>　　毕竟，他可是安排了人手，段惊天根本没有离开人王宫。
<br/>　　可是，他又不能当面说出来，毕竟，对方不承认，他说出来也无用，反而直接撕破了脸皮。
<br/>　　“的确是逆天，但是，一个妖孽人物，必须经历风雨才能成长，总不能让他一直待在人王宫吧！”
<br/>　　玄天长老淡淡的道，言罢，他又道：“难道是尘宇贤侄，想要找段惊天？”
<br/>　　“人德宫主，玄天长老，既然段惊天不在，那我也不多留，这就告辞了！”
<br/>　　天尘宇没有回答玄天长老的话，抱拳告辞，果断的便大步离去。
<br/>　　他没有必要在此多留了，因为人德宫主和玄天长老，明显是要帮助苏莫，一直都在故作茫然。
<br/>　　再谈下去，也没有什么必要了。
<br/>　　他此次来此，也就是要试探一下人王宫的态度而已，现在已经知晓了。
<br/>　　人德宫主和玄天长老，望着天尘宇离去的背影，互相对视了一眼，心中均是明了。
