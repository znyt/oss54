
晚间，渭河旁的一间民宿里，墨凉绣敷着面膜，往身上擦着润肤露，旁边桌子上的电话开启了免提。<br/>　　<br/>　　“几个月前，茅小草和王惊蛰，都同时有一段时间是失踪了的，不知跑到哪里去了，这个空档期还没有查出来？随后再出现，就剩下了王惊蛰自己，茅小草人不见了”<br/>　　<br/>　　“二小姐，就查到他们最后是消失在了黔西南一带的深山附近，再往后是什么状况，我们就摸不到了，您也知道那里除了山还是山，一片连着一片的，隔着几十里上百里都不一定有人烟，他们钻进山林中，就彻底和外界失去了联系，我们……”<br/>　　<br/>　　凉绣不耐烦的打断对方的话说道：“经过我不想知道，前因和结果，你们能查出来就行了，再给你们一个星期的时间，我这段时间都不会离开陇西，尽快查然后通知我，否则一个判断失误都会让我全盘皆输的”<br/>　　<br/>　　“知道了，二小姐！”<br/>　　<br/>　　凉绣想了想，接着问道：“就你们来看，这个小草和王惊蛰之间的感情，能达到什么地步？是大难临头各自飞呢，还是山无棱天地合才敢于君绝啊？”<br/>　　<br/>　　“茅小草和王惊蛰之间是媒妁之言一纸婚约，感情应该谈不上有多深，最多是互有好感，不过当初王惊蛰为了这个婚约都打到了陇西李氏的墙头上，为此还和南昆仑结下了仇怨，所以估计是婚约不能毁的，感情就另说了”<br/>　　<br/>　　凉绣“哦”了一声，摘掉脸上的面膜扔在了桶里，冲着镜子拍了拍嫩嫩的脸蛋，轻声说道：“一个呆呆萌萌的，一个傻愣傻愣的，这两人倒也般配，只是可惜了感情要是不太深的话，可就有点难办了，不过好在他们有婚约在身……呀，你们说，我要是去勾引王惊蛰的话，他会不会把那个茅小草给甩了，然后对我投怀送抱什么的？”<br/>　　<br/>　　凉绣话锋转的太快，这个弯至少九十度以上。<br/>　　<br/>　　电话中的人听闻，顿时不吭声了。<br/>　　<br/>　　擦完脸，凉绣照着镜子，摆弄着自己的腰肢和曲线有致的身材，撇着小嘴说道：“资本很好么，茅小草可要什么没什么，比我差远了，不瞎眼的男人都该知道怎么选的，哼哼，我这一双腿扔给王惊蛰，就够他瞪着眼珠子欣赏半夜的了，茅小草那里顶多算是交通工具罢了，还有上面这……”<br/>　　<br/>　　电话里的人，很不矜持的瞬间呼吸浓重起来。<br/>　　<br/>　　“想什么呢，给我滚去面壁”墨凉绣忽然冲着话筒骂了一声，然后“啪”的一声按断了电话。<br/>　　<br/>　　“噗通”凉绣轻轻一跳，趴在床上，甩着两条长腿蹬踹着说道：“哎呀呀，我和茅小草就是买家秀和买家秀的区别！”<br/>　　<br/>　　另外一头，王惊蛰察觉到自己给许秀琴的一道护身符裂了，就招呼林汶骐急匆匆的出了陈田村，出来就拦了一辆车，说了声去天河的方向。<br/>　　<br/>　　“怎么回事？”林汶骐问道。<br/>　　<br/>　　王惊蛰阴着脸说道：“有人不知好歹，做了初一还想做十五，这是要和我斗个法了，本来这个过节到此为止掀过去了，那就是过去了，可人呢总有不甘心的地方，以为自己行，就是真的行了，都说岭南的水深，我可得试试到底能有多深了，谁能把谁给淹了”<br/>　　<br/>　　林汶骐皱眉看着他，小声说道：“你不能闹得太过了，别忘了，韩观山那边还盯着你呢，你闹的鸡犬不宁就有可能把自己的消息给漏了，韩观山完全有可能顺藤摸瓜的找过来，还有那个墨家的凉绣姑娘”<br/>　　<br/>　　“我他么还成一块蛋糕了？”王惊蛰愤愤的骂了一句，然后摇头说道：“不用这么束手束脚的，真要是见一次躲一次，我后半辈子不活了啊？对于韩观山和凉绣，就是见招拆招，你来我往呗，他们真要是把老子给逼急了，我豁出去两败俱伤，也得坑死了他们……”<br/>　　<br/>　　林汶骐笑了：“干就完了？”<br/>　　<br/>　　王惊蛰傲然说道：“以前是我朝歌王氏单打独斗，唱独角戏，现在好了，我认识了个牛逼的大爷，不行我找他照着我呗，我这大爷别的不说，绝对护犊子，关键是还不讲道理，只认亲，不分青红皂白，他要是看不顺眼了，谁来怼谁”<br/>　　<br/>　　“抱大腿了呗？”<br/>　　<br/>　　王惊蛰呲牙一笑：“还是条大象腿”<br/>　　<br/>　　大厦里，许秀琴和吴国栋被困在了电梯内，两人惊恐的缩在角落里，睁睁的望着眼前的人，这人他们曾经见过，就是在卫生间里掉了一串手链的女人，但他俩完全不知道对方是怎么冒出来的，之前进电梯里的时候，明明是没有人的。<br/>　　<br/>　　只不过这女人和之前稍微有些不太一样了，长发披肩，脸色煞白，一双干枯的手爪子泛着青紫色，就这幅形象首先给人的感觉就是，碰见鬼了。<br/>　　<br/>　　许秀琴和吴国栋不知道的是，要不是王惊蛰给的那道护身符，两人此时早就已经被缠上了。<br/>　　<br/>　　大厦顶层，三十三楼，落地窗前站着刘运辉和张大师，两人低头看着楼下。<br/>　　<br/>　　“大师，人要是来了，你打算……”<br/>　　<br/>　　张大师说道：“你不了解风水术法中的斗法，论凶险不比刀光剑影差到哪里去，一个不小心就能让自己坠入万劫不复的境地了，这个斗法比的就是自身的经验和积累，看谁家世源远流长，底蕴深厚，我家祖传三代都是风水世家，祖父曾经跟随闽南大师寇德清从艺十八年，那位老先生的祖上据说是明万历年间的御用风水师，这年轻人我就算他从娘胎里的时候开始修炼好了，满打满算也不过撑死三十年而已，他拿什么来和我斗？不知天高地厚的后生晚辈，太不懂规矩了，我得替他家长辈好好教育他一通，事后他要是能提出师长是个有名望的人，可能这件事到此为止，要是提不出来的话，呵呵……”<br/>　　<br/>　　(=)
