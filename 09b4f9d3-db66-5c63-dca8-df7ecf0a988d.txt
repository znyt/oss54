什么嘛，夏夕绾就想起以前叶翎跟她说的，男人所有的殷勤都在为晚上预热，那些情人节七夕节什么最畅销，畅销的是礼物么，no，是酒店，是避孕-套！
<br/>
<br/>        当时夏夕绾听得面红耳赤，直到这一刻她才真正佩服起叶翎的一针见血。
<br/>
<br/>        夏夕绾将他推了推，“陆先生，正经点，我生日！”
<br/>
<br/>        她不得不提醒道。
<br/>
<br/>        陆寒霆搂着她的腰肢将她推到了蛋糕前，然后用打火机点燃了蜡烛，“小公主，你有什么生日愿望，快许愿吧。”
<br/>
<br/>        夏夕绾坐在了羊毛地毯上，双手合十，闭上了眼。
<br/>
<br/>        她20岁的生日愿望就是愿明年的今日，她依然爱与被爱，愿多年以后，陆先生依然在她的身边。
<br/>
<br/>        “许好愿啦，我要吹蜡烛了。”夏夕绾睁开眼，将蜡烛全部给吹灭了，祝我生日快乐，开心开心！
<br/>
<br/>        这时陆寒霆覆在她雪白的耳畔低声问了一声，“陆太太，你许了什么愿？”
<br/>
<br/>        “不告诉你，生日愿望说出来就不灵了。”
<br/>
<br/>        “陆太太，你已经长大一岁了，应该学会现实了，对着我许愿，你的愿望才有可能会实现。”
<br/>
<br/>        这个…嚣张的男人！
<br/>
<br/>        夏夕绾看了他一眼，“就不告诉你！”
<br/>
<br/>        陆寒霆看着她黑漉漉的眸子和粉润润的红唇，然后将切刀递给了她，“小公主，那切蛋糕吧。”
<br/>
<br/>        夏夕绾接过了切刀，切下了蛋糕，但是很快她就发现蛋糕里藏了什么东西，被她给切到了。
<br/>
<br/>        她扒开蛋糕一看，是…一枚钻戒。
<br/>
<br/>        设计大方灵动的钻戒，上面的钻石璀璨耀眼，摇曳生辉。
<br/>
<br/>        好漂亮的钻戒啊。
<br/>
<br/>        夏夕绾扭头看着身边的男人，纤长的羽捷眨巴眨，“陆先生，这是你送给我的？”
<br/>
<br/>        陆寒霆看着她，“喜欢么？”
<br/>
<br/>        夏夕绾没有想到陆先生挺会的，还会将钻戒藏在蛋糕里，如果不捉弄他一下，怎么让他知道这江湖的险恶？
<br/>
<br/>        夏夕绾抬起了小巧的下颌，倨傲道，“陆先生，你可以向我表白了，说说看你有多喜欢我恩？”
<br/>
<br/>        “胆子肥了是吧，我看你又欠了！”陆寒霆夺过了钻戒，一把戴进了她右手的无名指里，大手扣住了她的后脑勺，他低眸就吻上了她的红唇。
<br/>
<br/>        夏夕绾被吻的晕乎乎的，他这个吻格外的热情炙烈，不但夺走了她的呼吸，好像还要将她给吃下去一样。
<br/>
<br/>        夏夕绾扭头躲避，纤白的手指挑起一块蛋糕，直接擦在了他的俊颊上，“陆先生，你太坏了，让你欺负我！”
<br/>
<br/>        亲吻被迫中断，陆寒霆看着她被吻过的模样，那双翦瞳水灵灵的，娇嫩的肌肤染着樱花一样的粉红，简直比甜品还要诱人。
<br/>
<br/>        陆寒霆再次扣上她的后脑勺，将她摁到自己的俊颊边，“给我舔掉，不然…在这里要你！”
<br/>
<br/>        他…说什么？
<br/>
<br/>        夏夕绾一颗心迅速怦怦乱动，他可是那种说得出就能做到的男人，她吓得迅速伸出小舌去舔他脸上的蛋糕。
<br/>
<br/>        陆寒霆上下滚动着喉结。
<br/>
<br/>        这时夏夕绾踢了他一脚，贝齿咬着红唇，“陆先生，见好就收了，今天究竟是你生日还是我生日？”
