　　第49章
<br/>　　二人目光相对，各自发愣，程大雷脸上发烧，想要找个地缝钻进去。
<br/>　　“……我……”
<br/>　　“公子，我错了，我不该来这里。”
<br/>　　程大雷愣了愣，忽然表情一变，厉色道：“枉我这么信任你，你竟然拿着山寨的钱，来这种地方！”
<br/>　　“公子，我只是路过，顺便进来看看。”
<br/>　　“路过，我一直跟在你后面，会相信你！”
<br/>　　“公子，我错了，你千万别告诉灵儿，我现在就走。”
<br/>　　“好啦。”程大雷拍拍他的肩膀：“我也不是不近人情，既然来了，何妨就看看，当然是批判性的看，看看她们究竟多么丑陋。”
<br/>　　徐神机脸上一喜，忙不迭的点头。
<br/>　　如此也怪不得他，食色性也，人之常情，何况徐神机还不知素了多久。山上的生活，见到只母兔子都是好的，好不容易下山一趟，他肯定马不停蹄往这里赶，想要和白生生，水灵灵的小娘皮亲近一番。
<br/>　　老鸨安排二人坐在大厅，有两个姑娘陪着喝酒。不得不说，相貌超乎程大雷预期很多。一口一个欧巴欧公子叫着，脆生生的声音把程大雷叫得心都酥了。不过想起她们一双藕臂千人枕，半口朱唇万夫尝，程大雷心里还是有点忌讳的。
<br/>　　徐神机肯定不忌讳这些，他忌讳的是程大雷在身边，不敢太过放肆，只能暗自吞着口水。
<br/>　　无论如何，二人都有些拘谨，拘谨是因为不管是程大雷还是徐神机，都是第一次逛青楼。
<br/>　　是什么让他们一直保持如此高的道德水准？
<br/>　　当然是穷。
<br/>　　这里和程大雷想象中不同，要比程大雷想象中大，入门是院子，其后有一座二层木楼，各有雅间，里面隐隐有乐声传出。
<br/>　　大厅内有散座的客人，三五成群，偶尔有客人跳到中间台子，击剑而歌，脱帽而舞，很是放浪形骸，倒有些风流味道。
<br/>　　这里倒像个聚会场所，才子佳人聊聊诗与星辰，风花雪月。当然，聊得最多的，还是最近城里发生的新闻。
<br/>　　无非是谁谁发了财，谁谁升了官，谁谁写出一首好诗，谁谁点了某家花魁……
<br/>　　信息滞后的时代，谣言就是最重要的传播渠道。
<br/>　　“诸位……城门贴的榜文你们都看见了没有？”
<br/>　　“废话，现在谁还不知道蛤蟆大王的事，陆哼怎样的人物，在他面前一招都没有走过！”
<br/>　　“啧啧，一百五十两啊，谁如果能取到他的人头，这辈子就吃喝不愁了。”
<br/>　　“我可是听说那蛤蟆大王身高丈二，使一把三百斤重的大斧。”
<br/>　　“嘘，噤声，我听说那魔头被神仙传过法术，会五雷法，有千里眼，顺风耳，咱们在这里说话，他都听得见。”有人压低声音。
<br/>　　“呔，一派胡言，子不语怪力乱神，这世上哪有什么千里眼顺风耳。”
<br/>　　蛤蟆大王四个字在大厅内屡屡响起，程大雷和徐神机面面相觑，这扯得也太离谱了吧。
<br/>　　“诸位说得如此热闹，可曾见过那蛤蟆大王？”一锦衣打扮的男人忽然道。
<br/>　　“许兄的意思是，你曾见过那魔头？”
<br/>　　“何止是见过呐！”
<br/>　　“这话怎么说？”
<br/>　　“那一日我从蛤蟆岭下经过，忽听得一声高呼，有一人从天而降，此人自报姓名，便是蛤蟆大王。我与此獠缠斗，大战有三十回合。”
<br/>　　“咦，后来怎样？”
<br/>　　“堪堪惜败呐！”锦衣人长叹一声，分外惆怅。
<br/>　　众人一片惊呼，有人说：“你说这话可信么？陆哼在蛤蟆大王手上都走不过一招，难道你比陆哼还厉害？”
<br/>　　“许公子家传剑法了得，这话我是信的。”
<br/>　　陪酒的姑娘莺歌燕语，纷纷说着『许公子威武』『许公子了得』之类的话。
<br/>　　“呔，都住了吧。”锦衣人忽然历喝：“上次技差一招，堪堪在那贼人手中落败。若是再有下次，我定凭我手中剑，胸中一腔正义，手刃那魔头，为落叶城扫除一害。”
<br/>　　“呵呵。”
<br/>　　这番话说得字字看铿锵，落地之后还有余音。大家围绕着锦衣人，一共奉承之声，那许公子面露绯红，说着『今天的酒我请』『大家尽情喝』之类。
<br/>　　就在这个时候，一道不合时宜的冷笑声响起。锦衣人猛地回头，见对方是个留着山羊胡的猥琐男人。
<br/>　　“你笑什么，不信我的话么！”
<br/>　　徐神机在一旁听着，忍不住冷笑出声，可被对方这么一问，顿时有些怂，吓得垂下头。
<br/>　　程大雷拍拍他的肩膀，挑眉笑道：“不单他不信，我也是不信的。”程大雷目光一历：“你真的见过蛤蟆大王？”
<br/>　　这锦衣公子名叫许茂，他眼珠一转，随即大声道：“我当然见过，你是什么人，凭什么怀疑我！”
<br/>　　“在下姓欧名巴，江湖人称温柔小斧。”程大雷抱着拳，漫不经心的往上举了举。
<br/>　　“欧巴！”
<br/>　　“欧巴！”
<br/>　　许茂一伙面面相觑，口中不断重复着这个名字，却也谁都没有听说过。
<br/>　　程大雷暗自点头，纵然是一帮大老粗，但被他们齐声声称呼欧巴，还是有些愉悦感的。
<br/>　　“你们……也是为胭脂红来的？”有人问。
<br/>　　程大雷眨眨眼，没听过这个名字，他嘴里答道：“是呐，哪有怎样？”
<br/>　　“哈，看看他，竟也想打胭脂红的主意！”
<br/>　　“连绸缎都穿不起，还不知哪里来得乡巴佬。”
<br/>　　“癞蛤蟆想吃天鹅肉么！”
<br/>　　程大雷被嘲笑得有些莫名其妙，他的确穿着棉布衣服，但也不至于这么看不起人吧。程大雷隐隐有种前世安踏被耐克嘲笑的感觉。
<br/>　　而这胭脂红究竟又是何许人也？
<br/>　　从他们的嘲笑声中，程大雷渐渐明白了怎么会事。
<br/>　　这胭脂红姓柳，名叫柳芷，柳家原先在落叶城也是大户，其父在军中任职，负责保护港口不遭海盗侵扰。但一直剿匪不利，后又查出来暗中和海盗还有利益上的往来。
<br/>　　城主一怒之下，将柳父砍了脑袋，家中家丁奴役被贬做奴隶，柳芷这原先的柳家千金就被卖为官奴。
<br/>　　如今这柳芷就在回燕楼挂牌，据说此人生得花容月貌，倾国倾城，最近几日多有富家公子奔着她来，但直到今日，还没人能摘花抱月，获得美人芳心。
<br/>　　程大雷心中一动，奴隶岂不正是自己此来的目的之一。
<br/>　　
<br/>(本章完)
