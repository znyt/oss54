　　萧逸带着顾怜星二人疾速飞行。
<br/>　　不过半个时辰，已来到险地深处。
<br/>　　远方，一座散发着深异红芒的洞府，赫然伫立于深处中央。
<br/>　　“那便是血光洞府了吧。”萧逸双眼一眯。
<br/>　　整个洞府的外表，与他见过的天星洞府、金光洞府还有幻光洞府，并无什么差别。
<br/>　　同样巨大，同样散发着古老的气息。
<br/>　　唯一不同的，只是各洞府间的属性气息。
<br/>　　现今这座血光洞府，血芒涌动。
<br/>　　那大片的诡异血色光芒，映衬得整个天空宛若红云密布。
<br/>　　刺眼的光芒，连萧逸多看几眼都为之刺痛。
<br/>　　越发靠近洞府，扑面而来的气息，越发让萧逸心头惊骇。
<br/>　　那洋溢着的浓郁血色光芒，让得洞府看起来似极了一人畜勿近的血色炼狱。
<br/>　　嗖…
<br/>　　不过数各呼吸，萧逸便来到了洞府之前。
<br/>　　此时的他，脸色凛然。
<br/>　　因为，周遭一股股强猛气息，早已锁定了他。
<br/>　　“哪里来的贼人，敢在此时擅闯血光禁地？”
<br/>　　洞府之前，一个个老者盘膝而坐，此时，眼眸猛地睁开，精光暴现。
<br/>　　很显然，这些老者个个修为深厚，绝非泛泛之辈。
<br/>　　萧逸身后，公孙火舞低语道，“小贼，这是各府的长老，个个成名多年，最弱都在圣皇境后期，你小心些。”
<br/>　　印象中，这还是公孙火舞第一次提醒萧逸，甚至言语中带着关切。
<br/>　　不过萧逸并未注意她的脸色，只是看向顾怜星。
<br/>　　顾怜星，此刻脸色苍白，但显然也认出了一些长老人物，面露忧色。
<br/>　　虽然她见识过萧逸剑斩圣皇境巅峰妖兽的实力。
<br/>　　但，当她知晓血光禁地开启，而且十八府过半天骄都齐聚于此时，便已知道这些天骄在此做什么。
<br/>　　也已然猜到，守在洞府之外的这一个个长老，是何等实力。
<br/>　　这些长老，最弱都在圣皇境后期修为；甚至于有好些，乃是常年不出关的十八府传奇人物。
<br/>　　“萧逸公子，小心些。”顾怜星面露担忧，认真地看了萧逸一眼。
<br/>　　萧逸点了点头，脸色淡漠如斯，显然未将公孙火舞的提醒以及顾怜星的话放在心上。
<br/>　　“是你们自己让开，还是我打进去？”
<br/>　　萧逸只淡漠地说了一句。
<br/>　　“萧逸？”有些老者，已然认出了萧逸。
<br/>　　而有些老者，虽认不出萧逸，但却显然听过这个名字。
<br/>　　“便是那个凶名赫赫，臭名昭著的萧逸小贼？”
<br/>　　不知为何，这些老者中，不少的，忽然面露忌惮之色。
<br/>　　那些在遥远地域中发生的事迹，他们知之不详。
<br/>　　但在十八府地域内的事迹，他们却很清楚。
<br/>　　特别是十八府内的大事，那些对于普通人来说乃是机密或者难以知道的事，他们这一个个长老，却很轻易便能知道。
<br/>　　如，两月前，万金府大长老金雄，暗中追杀萧逸，却在之后神秘失踪。
<br/>　　如，半月多前，天冥府精锐尽出，甚至是天冥大长老这个传奇杀手亲自带队，伏杀萧逸，却在不久后音讯全无。
<br/>　　一众金牌杀手，无一人回府复命。
<br/>　　除非是傻瓜，否则，明眼人都能猜到，这些人，恐怕已成尸体，身死在外了。
<br/>　　而很显然，在场这一个个长老，一个个名动中域的强者，并非傻瓜。
<br/>　　哒…
<br/>　　此时，萧逸的脚步，已然迈动。
<br/>　　沉稳却极轻的脚步声，让得在场一众长老脸色一凛。
<br/>　　“萧逸。”一个血光府长老，当即沉声道，“我十八府没兴趣与你为敌，你别逼我们。”
<br/>　　“逼？”萧逸淡漠地吐出一个音符。
<br/>　　血光府长老沉声道，“你若擅闯血光洞府，便休怪我等不客气。”
<br/>　　“现在退去，我等既往不咎。”
<br/>　　“给我个理由。”萧逸淡漠道。
<br/>　　“你不知道？”血光府长老皱了皱眉。
<br/>　　血光府长老，看向顾怜星和公孙火舞，“别告诉老夫，怜星小姐和火舞小姐也不知道。”
<br/>　　“知道。”顾怜星点了点头，随后看向萧逸，道，“萧逸公子，我们先出血光险地，稍后再与你解释，三天罢了，我可以等。”
<br/>　　萧逸摇了摇头，“能不能等，我很清楚。”
<br/>　　“另外，我比较赶时间。”
<br/>　　“还有。”萧逸陡然看向血光府长老，“该给我理由的，是你们。”
<br/>　　“你…”血光府长老语气一滞，显然没想到萧逸连半分面子都不给他。
<br/>　　萧逸的强硬、淡漠，乃至霸道，让得在场长老，陡然脸色难看。
<br/>　　“好，很好。”一个长老，语气冰冷。
<br/>　　“不愧是萧逸，果真如传言那般，行事霸道，乖张蛮横。”
<br/>　　“是又如何。”萧逸手中冷焱剑已然星光涌动。
<br/>　　很显然，萧逸已准备出手。
<br/>　　在场一众长老，脸色一凛，身上气势，亦悄然涌动。
<br/>　　“哼，不如何。”刚才说话的那长老，冷哼一声。
<br/>　　“你要理由，老夫便告诉你。”
<br/>　　“如今，七星府，撼星府，药星府，箭星府；还有雷光府，剑光府，幻光府，血光府，八府少府主尽皆在洞府之内凝聚血脉之力，准备第二次的武魂觉醒。”
<br/>　　“任何人等，不得打扰。”
<br/>　　“谁敢擅闯，等同与我八府武者宣战。”
<br/>　　“若出差池，便等着我八府武者无穷无尽的追杀吧。”
<br/>　　这位长老的话音刚落下，周遭八府长老，已悉数气势爆发，冷眼凝视着萧逸。
<br/>　　“原来是要准备武魂的第二次觉醒。”萧逸心头恍然。
<br/>　　武魂的第二次觉醒，所需耗费的力量，极其庞大。
<br/>　　而武者本身，亦为此准备了多年。
<br/>　　武魂的力量本身方面，自不必多说。
<br/>　　而血脉之力，十八府天骄，自然个个身怀血脉。
<br/>　　武魂方面，是无法提升的。
<br/>　　但血脉力量方面，却可以。
<br/>　　血光洞府之内，若有提升血脉力量的方法的话。
<br/>　　那么，八府天骄，齐聚洞府之内，便可借此将自身血脉之力发挥到极致。
<br/>　　配合自己多年的准备，一举冲击武魂的第二次觉醒。
<br/>　　难怪洞府之外，会有如此多的强者守护。
<br/>　　不过，萧逸虽明白，却不代表他认可。
<br/>　　“擅闯等同宣战？呵，我正有此意。”
<br/>　　萧逸冷笑一声，瞬间出手。
