　　第424章
<br/>　　听完了徐松的话，程大雷也明白了事情大概，总的来说高飞虎现在的情况不是很好。
<br/>　　他坐在帅案后，目光环视一圈，道：“诸位怎么看？”
<br/>　　程大雷发现，自己的话说完，如和珅、阿喜这几人还好说，刘关张秦蛮等人的眼神都已经炙热起来。
<br/>　　“高兄弟千里辛苦来投奔我们，秦蛮怎能袖手旁观。只等大当家一声令下，秦蛮便将高兄弟救回来。”
<br/>　　“哈，我曾与高兄弟并肩作战，这件事更不可能不管，我想秦大哥还是歇歇吧，让我鱼字军去就行。”关鱼刚被秦蛮打败，现在看秦蛮还有些不顺眼。
<br/>　　“大当家，令我去吧，一天之内，我就能救大哥出来。”高飞虎。
<br/>　　果然呐，靠一场比武大会，根本不可能浇灭这些人心头的热血。诸人听到高飞虎陷入重围，第一感觉怕不是愤怒，而是激动。
<br/>　　激动的原因是终于有机会打仗了。
<br/>　　即便在闲时，程大雷也没有放弃训练。早晚训练，各军之间还经常进行演习，养兵千日，为的什么，为的不就是用在一时么。
<br/>　　可程大雷不让大家出兵，大家心里都有些郁闷。像怀揣一柄绝世宝刀，但在诸人面前炫耀的机会都没有。
<br/>　　现在一听有机会打仗，大家红着眼睛，就像色鬼见到美女一样。
<br/>　　战意昂扬，终究是好事。可看着这一幕，程大雷却有些为难，为难是因为不知该派谁去。
<br/>　　奔袭自然是赵子龙的轻骑兵方便，但正面冲突却是秦蛮的步兵占优。不过，秦蛮已经足够强了，这样训练的机会，要交给其他人。高飞豹的豹字军在这些人中，算是稍弱的。可高飞虎毕竟是他大哥，却也不能不让他去……
<br/>　　在程大雷沉默的这段时间，大厅内已吵个不停。
<br/>　　“鱼兄弟，兹事体大，我看安稳起见，还是让我蛮字军出动吧。”
<br/>　　“哈，秦大哥过虑了，你袖手旁观，我也能将人救出来。正好，秦大哥也学学如何打仗……”
<br/>　　你一言我一语，乱作一团，程大雷耳边嗡嗡作响。
<br/>　　“够了！”
<br/>　　程大雷的手重重拍在帅案上，所有人同时安静下来，目光看着程大雷。
<br/>　　“都去，都去。”程大雷挥了挥手。
<br/>　　“是！”
<br/>　　秦蛮、关鱼、赵子龙、高飞豹、张肥同时起身，程大雷还没作战前动员呢，人已经撒丫子跑没影了。
<br/>　　程大雷准备向让阿喜汇报一下关于凉州兵的情况。宋伯康提了十万人去长安，但凉州城还有些人留守，目前凉州城的兵力，指挥官程大雷都让阿喜调查过。
<br/>　　毕竟是打仗，不是儿戏，多了解一些总是没错的。
<br/>　　不过看几人的意思，是没心情听程大雷说这些。程大雷想了想，也没有把他们叫回来。如今这五人手上，人数最少的赵子龙也有三千人。他们手上也配了军师，打仗的事，总也不是一股脑冲锋。却通过这件事，看看这些人训练的成果如何。
<br/>　　“大当家，我们……”阿喜。
<br/>　　“好了，下去吧，无非就是一场小仗，不要太当回事。”
<br/>　　程大雷踱着方步，一步一步走到后院。在避开其他人视线后，程大雷立刻快步飞奔起来。
<br/>　　“银眸，银眸！”
<br/>　　那银眸女孩出现在程大雷眼前，程大雷挥舞着大手：“我的斧头呢，我的斧头呢……”
<br/>　　鬼面斧出现在程大雷手中，身上披了黑色盔甲，那头黑牛被牵了出来。程大雷佩剑背斧，座下黑牛，身上黑甲宛若用黑夜锤成的。
<br/>　　他迫不及待冲出了城主府。
<br/>　　关鱼几人跃跃欲试，程大雷本人又何尝不是，事实上，程大雷有过之而无不及。
<br/>　　城门，乱作一团。
<br/>　　虽乱，却乱而有章。
<br/>　　秦蛮关鱼几个，迫不及待跑回本家军营，挥舞着大叫：“来活啦，都忙起来。”
<br/>　　手下人不明所以，诧异的问：“老大，什么事呐？”
<br/>　　“都给我快些，再晚就没机会了。”张肥破口大骂。
<br/>　　几座军营连着，互相说话声都听得见。如此，便也就忙碌起来，明里暗里还有竞争的意思，谁也不愿落后。
<br/>　　平常五军之间就存在竞争，从来文无第一，武无第二，大家彼此谁也不服气谁。
<br/>　　长久刻苦的训练发挥出作用，五支军队用最短的时间集结，然而最快的还是赵子龙，毕竟他的龙字军人数最少。
<br/>　　“妈的，这兔崽子。”张肥骂道，忽然想起一件事：“快，快把阿喜抢回来。”
<br/>　　张三爷猛虽然猛，但不是没有脑子。何况在程大雷的熏陶下，谁都知道情报的重要性。这场奔袭战，不用筹备粮草，自然是情报先行。
<br/>　　张肥用了抢这个字眼，用的极为准确，因为同时关鱼、秦蛮也派出了人。
<br/>　　三方在城主府门口不期而遇，互相眼里都冒着火。但却没有看见阿喜，只看到白元飞愣愣站在那里。
<br/>　　“你们干嘛呢？”白元飞还不知道怎么会事。
<br/>　　“喜队长呢？”三方人马围着白元飞问。
<br/>　　“那个，被赵队长拉上马带走了，你们……是怎么会事？”
<br/>　　“嘿！”
<br/>　　三方同时叹气，心里对赵子龙咬牙切齿。
<br/>　　此时，程大雷骑着黑牛立在城门口，他眼睛看着出击的军队。
<br/>　　他虽然也有些思战，但却不只是提着斧头杀人。也看看，这支部队的行军如何。
<br/>　　目前为止，程大雷还是很满意的。
<br/>　　集结够快，出击够迅速，关键是又快又稳。并没有因为快，而乱了章法。此时的五支军队，就像在山路上极速奔驰的赛车。他们以极快的速度奔驰，但却又平稳的转向过弯。
<br/>　　此时，程大雷也准备出城，座下的黑牛奔驰起来，四蹄触地，发出砰砰的响声。自从得到这支黑牛后，程大雷还没有骑着它上阵打仗的机会。今日才发现，这黑牛的速度极快，而且耐力很好，就连赵子龙的骑兵队伍也快被他追上了。
<br/>　　“喂，你等等我！”
<br/>　　身后突然传来喊声，程大雷回过头，发现樊梨花快马奔来。
<br/>　　“你怎么不等我！”樊梨花骑着胭脂马，手中绣绒刀。
<br/>　　“这个……很危险的。”
<br/>　　“呵，本姑娘怕危险么？”
